## 표기 규칙 (전 권 공통)

- 일본어 원문의 한자는 `漢字(かんじ)` 방식으로 후리가나 병기.
- 동사 분류는 **5단동사 / 1단동사 / 불규칙동사(する·くる)** 방식(그룹 1/2/3 표기 안 씀).
- 해석은 직역이 아니라 대응하는 자연스러운 한국어로.
