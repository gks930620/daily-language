# 버전 5

## 대화

전부 후리가나로. `/` 는 한 박자 쉬는 곳. 소리 내어 읽는다.

> A: はじめまして。/ 私(わたし)は キム・ミナです。/ 韓国(かんこく)から 来(き)ました。/ どうぞ よろしく お願(ねが)いします。
> B: はじめまして。/ 田中(たなか)です。/ こちらこそ、よろしく お願いします。
> A: では、/ さようなら。
> B: さようなら。

**낭독 발음 포인트**
- 조사 は는 「わ(와)」로 읽는다: 私(わたし)は → 「와타시 와」
- こんにちは의 끝 は도, では의 は도 「わ」: では → 「데와」
- 長音(장음)은 한 박자 길게: どうぞ=「도오조」 / おはよう=「오하요오」 / さようなら=「사요오나라」 / ありがとう=「아리가토오」
- お願(ねが)いします = 「오네가이시마스」, 끝 す는 약하게

**낭독 연습** — 위 대화를 3번. ① 천천히 ② `/` 에서만 쉬며 ③ 안 보고.

## 문법

명사문을 소리로 익힌다.

- 私(わたし)は 学生(がくせい)です。 「와타시와 가쿠세에데스」 (저는 학생입니다.)
- 学生(がくせい)ですか。↗ 「가쿠세에데스카」 끝을 올린다 — 질문.
- 学生(がくせい)じゃありません。 「가쿠세에 자아리마센」 (학생이 아닙니다.)
- これは 本(ほん)です。 / それは 何(なん)ですか。 / あれは かばんです。

**낭독 발음 포인트**
- 의문 「〜か」는 문장 끝을 올려 읽는다(↗).
- 学生 = がく-せい, せい는 「세에」로 길게.
- じゃありません = 「자아리마센」, じゃ 는 한 박자로.

## 표현

한 호흡에 또박또박 읽는다.

| 표현 | 낭독 | 뜻 |
|---|---|---|
| おはようございます | 오하요오고자이마스 | (아침) 안녕하세요 |
| ありがとうございます | 아리가토오고자이마스 | 감사합니다 |
| すみません | 스미마센 | 죄송·실례합니다 |
| いただきます | 이타다키마스 | 잘 먹겠습니다 |
| お願(ねが)いします | 오네가이시마스 | 부탁합니다 |
| はい / いいえ | 하이 / 이이에 | 네 / 아니요 |

**낭독 발음 포인트**
- ございます = 「고자이마스」, 끝 す는 약하게 내린다.
- いいえ = 「이-에」, 앞 い를 길게.

**낭독 연습** — 아침부터 저녁까지 순서대로 이어 읽는다.
> おはようございます。→ ありがとうございます。→ すみません。→ お願(ねが)いします。→ いただきます。→ さようなら。

## 한자

숫자를 리듬으로 소리 내어 센다.

> いち・に・さん・し・ご・ろく・しち・はち・きゅう・じゅう
> (일 · 이 · 삼 · 사 · 오 · 육 · 칠 · 팔 · 구 · 십)

**낭독 발음 포인트**
- 四(し)와 七(しち)를 헷갈리지 않기: し 한 박, しち 두 박.
- 九(きゅう) = 「큐우」, 十(じゅう) = 「쥬우」 — 둘 다 길게.
- 一(いち) = 「이치」.

**낭독 연습** — 앞으로 한 번, 거꾸로 한 번.
> 앞으로: いち・に・さん・し・ご・ろく・しち・はち・きゅう・じゅう
> 거꾸로: じゅう・きゅう・はち・しち・ろく・ご・し・さん・に・いち
> 전화번호처럼: 「090の いち に さん よん の ご ろく なな はち」

**기본자 낭독**
- 日本(にほん) 「니혼」 — 일본
- 日本人(にほんじん) 「니혼진」 · 韓国人(かんこくじん) 「캉코쿠진」 — 人=じん「진」
- 人(ひと) 「히토」 — 사람
- 日(ひ) 「히」 — 날
