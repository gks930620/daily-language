# 버전 6

## 대화

한국어를 보고 일본어로 소리 내어 말한 뒤 정답을 연다.

**Q1.** 처음 뵙겠습니다.
<details><summary>정답</summary>はじめまして。</details>

**Q2.** 안녕하세요. (낮 인사)
<details><summary>정답</summary>こんにちは。</details>

**Q3.** 안녕히 가세요. / 안녕히 계세요.
<details><summary>정답</summary>さようなら。</details>

**Q4.** 저는 미나입니다.
<details><summary>정답</summary>私(わたし)はミナです。</details>

**Q5.** 한국에서 왔습니다.
<details><summary>정답</summary>韓国(かんこく)から来(き)ました。</details>

**Q6.** 잘 부탁드립니다.
<details><summary>정답</summary>どうぞよろしくお願(ねが)いします。</details>

**Q7. 전체 복원** — 한국어만 보고 대화 전체를 일본어로.
> A: 안녕하세요. 처음 뵙겠습니다.
> A: 저는 미나입니다. 한국에서 왔습니다.
> B: 처음 뵙겠습니다. 다나카입니다.
> A: 잘 부탁드립니다.
> B: 저야말로, 잘 부탁드립니다.
> A: 그럼, 안녕히 가세요.

<details><summary>정답</summary>

> A: こんにちは。はじめまして。
> A: 私(わたし)はミナです。韓国(かんこく)から来(き)ました。
> B: はじめまして。田中(たなか)です。
> A: どうぞよろしくお願(ねが)いします。
> B: こちらこそ、よろしくお願(ねが)いします。
> A: では、さようなら。

</details>

## 문법

빈칸을 채우거나 한국어를 일본어로 바꾼 뒤 정답을 연다.

**Q1.** 「AはBです」로: 저는 학생입니다.
<details><summary>정답</summary>私(わたし)は 学生(がくせい)です。</details>

**Q2.** 의문문으로: 당신은 학생입니까?
<details><summary>정답</summary>あなたは 学生(がくせい)ですか。 (끝에 か만 붙인다)</details>

**Q3.** 부정문으로: 저는 학생이 아닙니다.
<details><summary>정답</summary>私(わたし)は 学生(がくせい)じゃありません。 (=では ありません)</details>

**Q4.** 주제 조사 「は」는 어떻게 읽나?
<details><summary>정답</summary>[わ] 로 읽는다. 私(わたし)は → 「わたしわ」</details>

**Q5.** 이것 / 그것 / 저것 을 일본어로.
<details><summary>정답</summary>これ(이것·내 쪽) / それ(그것·상대 쪽) / あれ(저것·둘 다 먼 것)</details>

**Q6.** 「これは 本(ほん)です」를 부정문·의문문으로 각각.
<details><summary>정답</summary>부정: これは 本(ほん)じゃありません。 / 의문: これは 本(ほん)ですか。</details>

## 한자

한자를 보고 일본어 읽기를 말한 뒤 정답을 연다. (한국 한자음이 힌트)

**Q1.** 一 二 三 四 五 의 읽기는? (일·이·삼·사·오)
<details><summary>정답</summary>いち / に / さん / し(よん) / ご — 三은 「さん」, 한국음 '삼'과 소리가 닮았다.</details>

**Q2.** 六 七 八 九 十 의 읽기는? (육·칠·팔·구·십)
<details><summary>정답</summary>ろく / しち(なな) / はち / きゅう(く) / じゅう</details>

**Q3.** 「日本」의 읽기와 뜻은?
<details><summary>정답</summary>にほん — 일본. 日(に)+本(ほん). 한국음 '일본'의 '본'→ほん.</details>

**Q4.** 人 은 언제 「ひと」, 언제 「じん・にん」?
<details><summary>정답</summary>혼자 쓰면 훈독 ひと(사람). 단어에 붙으면 음독 じん・にん — 韓国人(かんこくじん) 한국인, 三人(さんにん) 세 명. 한국음 '인'→じん.</details>

**Q5.** 日 은 언제 「ひ」, 언제 「にち」?
<details><summary>정답</summary>혼자 쓰면 훈독 ひ(날·해). 단어에 붙으면 음독 にち — 日曜日(にちようび) 일요일, 毎日(まいにち) 매일.</details>

**Q6. 종합** — 三人 / 日本人 / 二日 를 읽어보라.
<details><summary>정답</summary>三人(さんにん) 세 명 / 日本人(にほんじん) 일본인 / 二日(ふつか) 이틀·2일</details>
