# 버전 9

## 대화

| 한국어 | 일본어 | 대조 메모 |
|---|---|---|
| 처음 뵙겠습니다 | はじめまして | 첫 만남 첫 마디 — 통째로 대응 |
| 안녕하세요 (낮) | こんにちは | は는 [わ]로 발음 |
| 안녕히 가세요 | さようなら | 헤어질 때 |
| 저**는** 미나**입니다** | 私(わたし)**は** ミナ**です** | 저=私, 는=は, 입니다=です — 어순·부품 1:1 |
| 한국**에서** 왔습니다 | 韓国(かんこく)**から** 来(き)ました | 에서=から, 왔습니다=来ました |
| 잘 부탁드립니다 | どうぞ よろしく お願(ねが)いします | 정형구 — 통째로 |
| 저야말로 | こちらこそ | 상대의 よろしく에 되받는 말 |

**어순 대조** — 한 부품도 자리를 안 옮긴다.
```
저    는   미나   입니다
私(わたし) は   ミナ   です
```

## 문법

**명사문 「AはBです」 — 한국어와 어순이 같다**

| 한국어 | 일본어 |
|---|---|
| 저 **는** 학생 **입니다** | 私(わたし) **は** 学生(がくせい) **です** |
| 저 = 私 / 는 = は / 학생 = 学生 / 입니다 = です | 순서 그대로 갈아 끼우기만 |

**조사 대조 — 이 과의 조사들**

| 한국어 조사 | 일본어 조사 | 예 |
|---|---|---|
| 은/는 | は (발음 [わ]) | 私(わたし)**は** 저**는** |
| 에서 (출발점) | から | 韓国(かんこく)**から** 한국**에서** |

**의문·부정 — 붙이는 방식도 한국어와 닮았다**

| 한국어 | 일본어 | 다른 점 |
|---|---|---|
| 학생입니**까?** | 学生(がくせい)です**か**。 | 한국어 '-까'처럼 끝에 か만 붙임 |
| 학생**이 아닙니다** | 学生(がくせい)**じゃありません**。 | です → じゃありません (=では ありません) |

**지시어 대조 — 한국어 이/그/저와 정확히 대응**

| 한국어 | 일본어 | 위치 |
|---|---|---|
| 이것 | これ | 내 쪽에 가까움 |
| 그것 | それ | 상대 쪽에 가까움 |
| 저것 | あれ | 둘 다에게서 먼 것 |

## 표현

| 한국어 | 일본어 | 대조 메모 |
|---|---|---|
| 안녕하세요 (아침) | おはようございます | 시간대별로 인사가 갈림 |
| 감사합니다 | ありがとうございます | ございます = 정중 어미 |
| 죄송합니다 / 실례합니다 | すみません | 사과·부름 겸용 |
| 잘 먹겠습니다 | いただきます | 한국어와 상황 1:1 |
| 부탁합니다 | お願(ねが)いします | 주문·요청 |
| 네 / 아니요 | はい / いいえ | — |

## 한자

**숫자 — 한국 한자음 ↔ 일본 음독**

| 한국음 | 한자 | 일본음 | 대조 메모 |
|---|---|---|---|
| 일 | 一 | いち | |
| 이 | 二 | に | |
| 삼 | 三 | さん | 소리가 거의 같다 |
| 사 | 四 | し·よん | |
| 오 | 五 | ご | |
| 육 | 六 | ろく | |
| 칠 | 七 | しち·なな | |
| 팔 | 八 | はち | |
| 구 | 九 | きゅう·く | |
| 십 | 十 | じゅう | |

**기본 한자 — 한국음이 힌트가 되는 것 / 안 되는 것**

| 한국음 | 한자 | 음독 | 훈독 | 대조 메모 |
|---|---|---|---|---|
| 일 | 日 | にち | ひ (날·해) | 음독 にち는 한국음과 다름, 훈독은 별도 암기 |
| 본 | 本 | ほん | もと | '본'→ほん 대응이 뚜렷 |
| 인 | 人 | じん·にん | ひと (사람) | '인'→じん. 단어 속=음독, 혼자=훈독 |

**단어 대조**

| 한국어 | 일본어 | 분해 |
|---|---|---|
| 일본 | 日本(にほん) | 日(に)+本(ほん) |
| 일본인 | 日本人(にほんじん) | 日本+人(じん) |
| 한국인 | 韓国人(かんこくじん) | 韓国+人(じん) |
| 세 명 | 三人(さんにん) | 三+人(にん) |

## 한국어와 다른 점 (주의)

같아 보여도 어긋나는 지점만 모았다.

| 항목 | 한국어 | 일본어 | 다른 점 |
|---|---|---|---|
| 조사 は 발음 | 는 (글자대로) | は → [わ] | 글자는 は, 소리는 わ |
| 부정문 조사 | 학생**이** 아닙니다 | 学生(がくせい)**じゃ**ありません | 한국어 '이'에 해당하는 자리가 じゃ(では) |
| 띄어쓰기 | 저는 학생입니다 | 私(わたし)は学生(がくせい)です | 일본어는 원래 띄어쓰지 않는다 (여기선 학습용으로 띄움) |
| 이름 뒤 | 미나 씨 | ミナさん | 씨=さん, 이름 뒤에 붙는 위치까지 같다 |
| 되받는 인사 | 저야말로 | こちらこそ | 통째로 대응, 분해 불가 |
