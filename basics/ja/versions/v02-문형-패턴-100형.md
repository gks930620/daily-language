# v02 — 문형 패턴 100형

> 일본어 기초 4권 세트(①대화 입문 ②기본 문법 ③필수 표현 ④필수 한자, 6개월)의 대안 구성 버전.
> 비교 기준선: 정통 구성(정중체 대화 → 체계 문법 → 상황별 표현 → 급수순 한자).

---

## 1. 콘셉트

- 기초 일본어 전체를 **핵심 문형 100개(P001~P100)**로 수량화하고, 4권을 전부 이 100개의 축에 **역구성**한다. "오늘 뭘 배웠나?"의 답이 항상 "P037까지 왔다"로 나온다 — 진도가 숫자로 보이는 것이 이 버전의 핵심 경험이다.
- 정통 구성은 상황(공항→식당→…)이 목차의 축이고 문형은 그 안에 흩어져 있다. 이 버전은 반대로 **문형이 목차의 축**이고, 상황은 문형을 보여 주기 위한 무대로만 쓰인다.
- 4권의 관계: 1권에서 문형을 **대화 속에서 만나고**(패턴당 미니 대화 1편, 그 대화 안에 해당 문형이 최소 3회 등장하는 것이 집필 규칙), 2권에서 **왜 그렇게 되는지 원리**를 배우고, 3권에서 **변주 드릴로 몸에 박고**, 4권에서 그 문형 예문에 나온 **한자를 줍는다**.
- 어휘는 최소화 전략: 새 단어를 늘리기보다 **아는 단어를 100가지 틀에 끼우는 능력**을 먼저 만든다.

## 2. 누구에게 맞나 / 안 맞나

**맞는 사람**
- "몇 % 왔는지"가 보여야 지속되는 사람. P001~P100 체크리스트가 그대로 진도표다.
- 회화 순발력을 원하는 사람 — 변주 드릴(3권) 비중이 4권 세트 중 가장 크다.

**안 맞는 사람**
- 이야기·맥락 없이 드릴이 이어지면 지루해지는 사람(그 경우 v01이 맞다).
- "이 상황에서 뭐라고 하지?"를 바로 찾고 싶은 사람 — 이 버전의 목차는 상황이 아니라 문형이라 역인덱스(부록)를 거쳐야 한다.

## 3. 4권 구성 개요

| 권 | 이 버전에서의 정체 | 정통 구성과 다른 점 |
|---|---|---|
| 1권 | **패턴 대화 100** — 문형당 미니 대화 1편(4~6줄) × 100 | 긴 상황 대화가 아니라 **문형이 주인공인 초단편 100편**. 한 편에 해당 문형 최소 3회 |
| 2권 | **기관실 10** — 100패턴을 떠받치는 원리 10장 | 문법 백과가 아니라 **패턴 10개 묶음당 원리 1장**. 목차가 P번호로 인덱싱됨 |
| 3권 | **변주 매트릭스** — 패턴 × 6변주(부정·과거·과거부정·의문·반말·응용) | 상황별 표현집이 아니라 **활용 드릴북**. 600개 변주 문장 생성 |
| 4권 | **패턴 코퍼스 한자 300** — 1권 대화 600문장에서 빈도순 추출 | 급수순이 아니라 **이 책 예문에서 많이 나온 순서**. 모든 한자에 "출신 패턴" 표기 |

### 100 패턴 지도 (10블록 × 10패턴)

| 블록 | 번호 | 주제 | 대표 패턴 |
|---|---|---|---|
| A | P001~010 | 명사문 | 〜は〜です / 〜ですか / 〜じゃないです / AのB / これ・それ・あれ |
| B | P011~020 | 존재·위치 | 〜が あります・います / 〜は どこですか / 〜の 上(うえ)に |
| C | P021~030 | 형용사문 | い형용사 / な형용사 / 〜くて / 〜が 好(す)きです |
| D | P031~040 | 동사 기본 | 〜ます / 〜ません / 〜ました / 〜に 行(い)きます |
| E | P041~050 | 요청·권유 | 〜を ください / 〜てください / 〜ませんか / 〜ましょう |
| F | P051~060 | 희망·의지 | 〜たいです / 〜が ほしいです / 〜つもりです |
| G | P061~070 | 이유·연결 | 〜から / 〜ので / て형 연결 / 〜が(역접) |
| H | P071~080 | 가능·경험·의무 | 〜ことが できます / 〜たことが あります / 〜なければ なりません |
| I | P081~090 | 수수·부탁 | あげます・もらいます・くれます / 〜てもらえますか |
| J | P091~100 | 추측·인용·정리 | 〜でしょう / 〜と 思(おも)います / 〜そうです / 총정리 리믹스 |

- 진도 규칙: **주 4~5패턴** × 24주 ≒ 100패턴 + 리믹스 주간.
- 부록: 상황→패턴 역인덱스("식당에서 주문" → P041, P045, P052).

**블록 A 상세 — 패턴 10개의 실제 모습** (모든 블록이 각 장 첫머리에 이런 상세표를 갖는다)

| 번호 | 패턴 | 한 줄 설명 |
|---|---|---|
| P001 | 〜は 〜です | 명사문의 긍정 — 모든 것의 시작 |
| P002 | 〜ですか | 어순 그대로 + か = 의문문 |
| P003 | 〜じゃないです | 명사문의 부정 |
| P004 | AのB | 소속·소유의 の |
| P005 | これ・それ・あれ | 지시어 3분법(한국어 이·그·저와 평행) |
| P006 | 〜も 〜です | "~도" — は 자리에 も를 끼운다 |
| P007 | AとB | "와/과" 병렬 |
| P008 | 〜ですね / 〜ですよ | 종조사 — 동의 구하기(ね)와 알려 주기(よ) |
| P009 | 何(なん)ですか / どこですか / だれですか | 의문사 의문문 3종 |
| P010 | 〜でした / 〜じゃなかったです | 명사문의 과거·과거부정 |

**역인덱스(부록) 발췌** — 상황으로 찾고 싶을 때는 뒤에서 앞으로 온다.

| 상황 | 필요한 패턴 |
|---|---|
| 편의점 계산 | P041(ください), P009(いくら계), P008(よ·ね) |
| 첫 자기소개 | P001~P004, P093(と思います는 아직 불요) |
| 길 묻기 | P012(どこですか), P011(あります), P044(てください) |

### 각 권의 패턴당 내부 구조

- **1권**: 무대 한 줄 → 미니 대화(패턴 3회 이상 등장, 굵게 표시) → 그 문형만 뽑은 확인 1줄
- **2권**: 기관실 1장 = 패턴 10개의 공통 원리 → 부품 교환대(단어 갈아 끼우기 표) → 한국어 대응 한 줄
- **3권**: 패턴 1개 = 변주표 1장(6변주) + 30초 입 드릴 스크립트
- **4권**: 주당 한자 12~13자, 각 한자에 "출신 패턴 예문" 인용 → 24주 300자

## 4. 1단원 샘플 — P001 「〜は 〜です」

### 1권 패턴 대화 100 — 대화 001

**무대**: 회사 첫날, 옆자리 사람과.

| 대사 | 해석 |
|---|---|
| ミナ: はじめまして。キム・ミナ**です**。 | 처음 뵙겠습니다. 김민아입니다. |
| 田中: 田中(たなか)**です**。ミナさん**は** 学生(がくせい)**ですか**。 | 다나카입니다. 민아 씨는 학생입니까? |
| ミナ: いいえ、会社員(かいしゃいん)**です**。田中さん**は**？ | 아니요, 회사원입니다. 다나카 씨는요? |
| 田中: 私(わたし)**は** エンジニア**です**。 | 저는 엔지니어입니다. |
| ミナ: そう**です**か。よろしく お願(ねが)いします。 | 그렇군요. 잘 부탁드립니다. |

- **패턴 등장 횟수**: です 6회, は 3회 — 굵은 글씨만 따라 읽어도 오늘의 문형이 몸에 들어온다.
- 확인 1줄: 「私は キム・ミナです。」를 자기 이름으로 바꿔 소리 내어 3번.

**패턴 발견 연습** — 위 대화에서 P001이 쓰인 문장에 전부 밑줄을 긋는다(답: 5문장 전부). 이 "발견" 행위가 이 책의 학습 단위다: 대화를 읽는 게 아니라 **패턴을 사냥**한다.

**나의 대화 001 만들기** — 이름·직업 부품만 바꿔 자기 버전을 만든다.

| 부품 | 원본 | 내 버전 |
|---|---|---|
| 이름 | キム・ミナ | (자기 이름) |
| 직업 | 会社員(かいしゃいん) | 学生(がくせい)/主婦(しゅふ)/公務員(こうむいん)… |

완성한 자기 버전을 소리 내어 2회. **다음 대화 예고**: 대화 002는 P002(〜ですか)가 주인공 — 같은 두 사람이 서로의 출신을 묻는다.

### 2권 기관실 10 — 기관실 A 「명사문 엔진」 (P001~P010 담당)

**엔진 구조**

```
[주제]は  [명사]です
 은/는     -입니다
```

- **です의 정체**: 동사가 아니라 "이 문장을 정중하게 닫는 마침 장치". 한국어 "-입니다"와 기능·위치가 1:1로 겹친다.
- **は의 두 얼굴**: 글자는 ha지만 조사일 때 발음은 **[わ]**. 뜻은 "은/는" — 화제를 얹는 받침대라는 것까지 한국어와 같다.
- **의문문 공식**: 어순 그대로 + **か**. 「学生です」→「学生ですか」. 한국어 "-입니다→-입니까"와 평행이라 배우는 게 아니라 확인만 하면 된다.
- **부품 교환대** — 아래 표에서 좌우 아무거나 골라 끼우면 문장 9개가 즉시 생성된다.

| [주제]は | [명사]です |
|---|---|
| 私(わたし)は | 学生(がくせい)です |
| 田中(たなか)さんは | 会社員(かいしゃいん)です |
| これは | 水(みず)です |

- 이 기관실이 떠받치는 패턴: P001(〜は〜です), P002(〜ですか), P003(〜じゃないです), P004(AのB), P005(これ・それ・あれ) … P010.

### 3권 변주 매트릭스 — P001 변주표

**기본형**: 私(わたし)は 学生(がくせい)です。(저는 학생입니다.)

| 변주 | 일본어 | 해석 |
|---|---|---|
| ① 부정 | 私は 学生**じゃないです**。 | 저는 학생이 아닙니다. |
| ② 과거 | 私は 学生**でした**。 | 저는 학생이었습니다. |
| ③ 과거부정 | 私は 学生**じゃなかったです**。 | 저는 학생이 아니었습니다. |
| ④ 의문 | ミナさんは 学生**ですか**。 | 민아 씨는 학생입니까? |
| ⑤ 반말 | 私、学生**だよ**。 | 나, 학생이야. |
| ⑥ 응용 | 私は 学生で、弟(おとうと)は 会社員(かいしゃいん)です。 | 저는 학생이고, 남동생은 회사원입니다. |

**30초 입 드릴** — 재료를 바꿔 가며 ①~④를 논스톱으로:
- 재료 1: 韓国人(かんこくじん) → 韓国人です / 韓国人じゃないです / 韓国人でした / 韓国人ですか
- 재료 2: 休(やす)み(휴일) → 休みです / 休みじゃないです / 休みでした / 休みですか
- 막히지 않고 20초 안에 돌면 P001 합격. 체크박스에 날짜를 적는다: ☐ __/__

### 4권 패턴 코퍼스 한자 300 — 1주차 (빈도 순위 1~8)

이 책의 한자 순서는 급수가 아니라 **1권 대화 600문장에서의 등장 빈도**다. 1주차 8자는 전부 P001~P005 예문 출신.

| 순위 | 한자 | 음독 | 훈독 | 출신 패턴 예문 |
|---|---|---|---|---|
| 1 | 私 | し | わたし | 私(わたし)は 学生です。(P001) |
| 2 | 学 | がく | まな(ぶ) | 学生(がくせい)ですか。(P002) |
| 3 | 生 | せい | い(きる) | 学生(がくせい) (P002) |
| 4 | 人 | じん・にん | ひと | 韓国人(かんこくじん)です。(P001) |
| 5 | 会 | かい | あ(う) | 会社員(かいしゃいん)です。(P001) |
| 6 | 社 | しゃ | — | 会社員(かいしゃいん) (P001) |
| 7 | 日 | にち | ひ | 日本(にほん)は はじめてです。(P003) |
| 8 | 本 | ほん | もと | 日本(にほん) / 本(ほん)です。(P005) |

- **조립 확인**: 学+生=学生(がくせい), 会+社+員=会社員(かいしゃいん) — 한자는 낱자가 아니라 **패턴 예문의 단어로 재조립**해야 끝난 것이다.
- 쓰기 과제: 위 8자로 「私は会社員です」를 한자 섞어 손으로 2번.

## 5. 6개월 주행 계획

**주간 리듬(고정)**: 새 패턴 4~5개 / 주. 월·화(1권 대화로 만남) → 수(2권 원리) → 목·금(3권 변주 드릴) → 매일 10분(4권 한자 2~3자) → 토(이번 주 패턴 셔플 자작 대화) → 일(휴식).

| 월 | 패턴 진도 | 블록 | 4권(한자) | 마일스톤 |
|---|---|---|---|---|
| 1개월 | P001~020 | A 명사문 + B 존재·위치 | 순위 1~50 | 자기소개·사물 묘사를 문형 20개로 |
| 2개월 | P021~040 | C 형용사 + D 동사 기본 | 51~100 | "어제 뭐 했다"까지 말문 |
| 3개월 | P041~060 | E 요청·권유 + F 희망·의지 | 101~150 | 중간 점검: P001~060 랜덤 카드 드릴 |
| 4개월 | P061~080 | G 이유·연결 + H 가능·경험·의무 | 151~200 | 두 문장을 한 문장으로 잇는 단계 |
| 5개월 | P081~100 | I 수수·부탁 + J 추측·인용 | 201~250 | 100패턴 체크리스트 완주 |
| 6개월 | 리믹스 | 100패턴 셔플 통합 | 251~300 + 총복습 | 부록 역인덱스로 상황 시뮬레이션 20장면 |

> **완주 후 상태**: 문형 100 × 변주 6 = 이론상 600문장 생성 능력 + 한자 300자. "아는 단어는 적어도 문장은 만든다"는 상태가 이 버전의 졸업 정의다.
