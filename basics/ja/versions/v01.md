# 버전 1

## 대화

**인사말**

| 일본어 | 한국어 | 쓰는 때 |
|---|---|---|
| はじめまして | 처음 뵙겠습니다 | 처음 만난 사람에게 |
| こんにちは | 안녕하세요 | 낮에 만날 때 |
| さようなら | 안녕히 가세요 | 헤어질 때 |
| どうぞ よろしく お願(ねが)いします | 아무쪼록 잘 부탁드립니다 | 자기소개 끝맺음 |

**자기소개 부품**

| 일본어 | 한국어 | 메모 |
|---|---|---|
| 私(わたし)は ○○です | 저는 ○○입니다 | ○○에 이름 |
| ○○から 来(き)ました | ○○에서 왔습니다 | ○○에 나라·도시 |

**자기소개 대화**

| 화자 | 대사 | 해석 |
|---|---|---|
| A | はじめまして。私(わたし)は キム・ミナです。 | 처음 뵙겠습니다. 저는 김민아입니다. |
| A | 韓国(かんこく)から 来(き)ました。 | 한국에서 왔습니다. |
| B | はじめまして。田中(たなか)です。 | 처음 뵙겠습니다. 다나카입니다. |
| A | どうぞ よろしく お願(ねが)いします。 | 아무쪼록 잘 부탁드립니다. |
| B | こちらこそ、よろしく お願いします。 | 저야말로 잘 부탁드립니다. |

## 문법

**명사문 「AはBです」3형**

| 형태 | 문형 | 예문 | 해석 |
|---|---|---|---|
| 긍정 | AはBです | 私(わたし)は 学生(がくせい)です。 | 저는 학생입니다. |
| 의문 | AはBですか | 田中(たなか)さんは 学生ですか。 | 다나카 씨는 학생입니까? |
| 부정 | AはBじゃありません | 私は 学生じゃありません。 | 저는 학생이 아닙니다. |

**주제 조사 は**

| 항목 | 내용 |
|---|---|
| 뜻 | 은/는 (주제를 얹는다) |
| 표기 | は |
| 발음 | わ [와] — 조사로 쓰일 때만 바뀐다 |
| 예 | 私(わたし)は → 「와타시 와」 |

**지시어 これ・それ・あれ**

| 일본어 | 한국어 | 가리키는 위치 |
|---|---|---|
| これ | 이것 | 나에게 가까움 |
| それ | 그것 | 상대에게 가까움 |
| あれ | 저것 | 둘 다에게서 멂 |

| 예문 | 해석 |
|---|---|
| これは 本(ほん)です。 | 이것은 책입니다. |
| それは 何(なん)ですか。 | 그것은 무엇입니까? |
| あれは 私(わたし)の かばんです。 | 저것은 제 가방입니다. |

**생존 인사**

| ありがとうございます | 감사합니다 | 고마울 때 |
| すみません | 죄송합니다 / 실례합니다 | 사과하거나 부를 때 |

| はい / いいえ | 네 / 아니요 | 대답 |

## 한자

**숫자 一〜十**

| 한자 | 읽기 | 한국음 | 힌트 |
|---|---|---|---|
| 一 | いち | 일 | ㄹ받침 → ち |
| 二 | に | 이 | 이 → に |
| 三 | さん | 삼 | ㅁ받침 → ん |
| 四 | し / よん | 사 | 사 → し |
| 五 | ご | 오 | 오 → ご |
| 六 | ろく | 육·륙 | ㄱ받침 → く |
| 七 | しち / なな | 칠 | 칠 → しち |
| 八 | はち | 팔 | ㄹ받침 → ち |
| 九 | きゅう / く | 구 | 구 → きゅう |
| 十 | じゅう | 십 | ㅂ받침 → う |
