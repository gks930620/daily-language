# v06 — 미디어 몰입형

> 일본어 기초 4권 세트(①대화 입문 ②기본 문법 ③필수 표현 ④필수 한자, 6개월)의 대안 구성 버전.
> 비교 기준선: 정통 구성(정중체 대화 → 체계 문법 → 상황별 표현 → 급수순 한자).

---

## 1. 콘셉트

- 교과서 문장("これはペンです")이 아니라 **애니·드라마·J-POP·버라이어티에서 실제로 들리는 말**을 학습의 축으로 삼는다. 모든 대사는 저작권 문제가 없도록 실존 작품의 문체를 흉내 낸 **유사 창작 대본**이다.
- 정통 구성이 "정중체 → 반말" 순서인 것과 정반대로, **반말(タメ口)과 구어 축약이 1주차부터 등장**한다. 미디어에서 들리는 말이 대부분 반말이기 때문이다.
- 대신 리스크를 방치하지 않는다: **모든 반말 대사에 정중체 변환을 병기**하고, 3권의 모든 표현에 "누구에게 써도 되는가" 친밀도 미터를 단다.
- 학습 동기의 원천을 "시험·여행"이 아니라 **"자막 없이 좋아하는 것을 즐기고 싶다"**에 둔다. 매 단원의 도달 목표가 "이 장면이 들린다"로 정의된다.

## 2. 누구에게 맞나 / 안 맞나

**맞는 사람**
- 애니·드라마·J-POP·버라이어티를 이미 자막으로 즐기고 있고, "이 대사를 알아듣고 싶다"가 학습 동기의 전부인 사람.
- 정통 교재를 2주 만에 덮어 본 경험이 있는 사람 — 이 버전은 흥미 지속력이 가장 강하다.

**안 맞는 사람**
- 출장·업무 등 **정중체가 당장 필요한** 사람. 이 버전에서 정중체가 본격화되는 것은 4개월차(드라마 직장 편)다.
- "틀린 말버릇이 먼저 굳으면 어쩌지"가 크게 불안한 사람. 반말 선행은 이 버전의 특징이자 **실재하는 리스크**다(아래 각 권에서 관리 장치를 명시).

## 3. 4권 구성 개요

정통 구성과의 역할 차이: 4권이 "입문 → 문법 → 표현 → 한자"의 단계가 아니라, **같은 미디어 코퍼스를 네 방향에서 자르는 도구 세트**다.

| 권 | 이 버전에서의 정체 | 정통 구성과 다른 점 |
|---|---|---|
| 1권 | **장면 대본집** — 장르별 유사 창작 장면 24편 | 상황(공항·식당)이 아니라 **장르(학원물→일상물→드라마→버라이어티)** 순서 |
| 2권 | **자막 해부** — 귀에 들린 구어를 표준형으로 역추적 | 문법을 원형→활용이 아니라 **축약형→원형 복원** 방향으로 배움 |
| 3권 | **장르별 표현 사전** — 리액션·감정·팬덤 표현 + 친밀도 미터 | 상황별 정중 표현이 아니라 **감정·리액션** 중심, 전 표현에 사용 가능 범위 태그 |
| 4권 | **자막 한자 300** — 화면(자막·텔롭·가사)에 뜨는 빈도순 | 급수·획수 순이 아니라 **"화면에서 본 순서"** |

## 4. 1단원 샘플

### 1권 샘플 — 장면 01 「転校生(てんこうせい)」

> **도달 목표**: 학원물 1화의 "전학생 등장" 장면이 자막 없이 들린다.
> 場面(ばめん): 朝(あさ)の教室(きょうしつ)。 전학생 미나가 들어온다.

```
ユイ  ： ねえねえ、見(み)て！ 転校生(てんこうせい)だって！
유이   : 야야, 봐 봐! 전학생이래!

アオイ： まじ？ どこどこ？
아오이 : 진짜? 어디 어디?

ユイ  ： ほら、あの子(こ)。かわいくない？
유이   : 저기, 쟤. 귀엽지 않아?

ミナ  ： あの…はじめまして。ミナです。韓国(かんこく)から来(き)ました。
미나   : 저기… 처음 뵙겠습니다. 미나예요. 한국에서 왔어요.

アオイ： えっ、韓国(かんこく)！？ すごーい！
아오이 : 앗, 한국?! 대단하다ー!

ユイ  ： ね、隣(となり)の席(せき)、空(あ)いてるよ。ここ座(すわ)りなよ。
유이   : 있잖아, 옆자리 비어 있어. 여기 앉아.

ミナ  ： あ、ありがとう…ございます？
미나   : 아, 고마워…요?

ユイ  ： あはは、敬語(けいご)いらないって！ 同(おな)い年(どし)でしょ？
유이   : 아하하, 존댓말 필요 없다니까! 동갑이잖아?
```

**리얼 포인트** (이 장면에서만 3개)
1. **〜だって** 「転校生(てんこうせい)だって」 = "~래(전언)". 표준형은 「〜だそうです」. 애니 1화에서 반드시 들리는 형태.
2. **〜てる** 「空(あ)いてるよ」 = 「空(あ)いていますよ」의 축약. 2권 해부 01에서 본격 해부.
3. **〜なよ** 「座(すわ)りなよ」 = 부드러운 명령(친구 사이 전용). 초면·연상에게 쓰면 실례 — **위험 표시 ⚠**.

**정중체 변환 코너** (이 버전의 안전장치 — 매 장면 필수)

| 장면의 반말 | 실전에서 초면에게라면 |
|---|---|
| 転校生(てんこうせい)だって！ | 転校生(てんこうせい)だそうです。 |
| ここ座(すわ)りなよ。 | ここ、どうぞ。 |
| 敬語(けいご)いらないって！ | 敬語(けいご)は使(つか)わなくていいですよ。 |

**새 단어** — 転校生(てんこうせい) 전학생 / 教室(きょうしつ) 교실 / 隣(となり) 옆 / 席(せき) 자리 / 敬語(けいご) 존댓말 / 同(おな)い年(どし) 동갑

**섀도잉 과제**: 유이 역을 맡아 감정 그대로 3회. "ねえねえ"의 들뜬 톤까지 복사할 것.

### 2권 샘플 — 해부 01 「てる」

> **원리**: 귀에 들린 형태에서 출발해 표준형을 복원한다. 원형→활용이 아니라 **활용→원형** 방향.

**Step 1 — 들린다**: 드라마·애니에서 하루에도 수십 번 나오는 소리.

```
「今(いま)、何(なに)してる？」        지금 뭐 해?
「ドラマ見(み)てる。」               드라마 보고 있어.
「雨(あめ)、降(ふ)ってるね。」        비 오고 있네.
```

**Step 2 — 복원한다**: てる = **ている**의 い 탈락.

| 들린 형태 | 복원(표준형) | 정중체 |
|---|---|---|
| してる | している | しています |
| 見(み)てる | 見(み)ている | 見(み)ています |
| 降(ふ)ってる | 降(ふ)っている | 降(ふ)っています |

**Step 3 — 규칙을 얻는다**: 「〜ている」 = ①진행(하는 중) ②상태(~해 있다). 구어에서는 거의 항상 い가 탈락한다. **자막에는 てる로 적히고, 교과서에는 ている로 적힌다** — 둘이 같은 것임을 아는 순간 자막 문법의 절반이 풀린다.

**Step 4 — 역방향 훈련**: 이번엔 표준형을 구어로.

```
何(なに)を見(み)ていますか。 → 何(なに)見(み)てるの？
音楽(おんがく)を聞(き)いています。 → 音楽(おんがく)聞(き)いてる。
```

(조사 を의 탈락도 함께 일어난다 — 해부 03에서 정식으로 다룸)

**주의 ⚠**: 지원 면접·점원 응대 등 격식 장면에서 てる를 쓰면 어리게/무례하게 들린다. 격식에서는 반드시 ています로 복원할 것.

### 3권 샘플 — 표현 사전 01 「리액션 기본 3종」

> 친밀도 미터: ◎ 친구 / △ 직장(캐주얼한 팀이면 가능) / ✕ 초면·손윗사람

**やばい** 〔◎ / △ / ✕〕
- 원뜻은 "위험하다"인데, 현대 구어에서는 좋을 때도 나쁠 때도 쓰는 만능 감탄사.
- 「このラーメン、やばい。」 이 라멘, 미쳤다(맛있다). / 「時間(じかん)やばい！」 시간 큰일 났다!
- 격식 변환: (좋음) すごくおいしいです / (나쁨) まずいことになりました

**まじで？** 〔◎ / △ / ✕〕
- "진짜로?" 놀람·되물음. まじ？만으로도 성립. 버라이어티 리액션의 기본값.
- 「え、まじで？ 明日(あした)テスト？」 어, 진짜? 내일 시험이야?
- 격식 변환: 本当(ほんとう)ですか。

**うそ！** 〔◎ / △ / ✕〕
- 직역은 "거짓말!"이지만 실제 의미는 "말도 안 돼!". 상대를 거짓말쟁이라고 하는 게 아니다.
- 「うそ！ チケット当(あ)たったの！？」 대박! 티켓 당첨된 거야?!
- 격식 변환: 信(しん)じられません。

**미니 대본으로 확인** — 버라이어티 스튜디오 톤:

```
Ａ： このケーキ、３時間(さんじかん)並(なら)んで買(か)いました。
A  : 이 케이크, 3시간 줄 서서 샀습니다.
Ｂ： うそ！？ ３時間(さんじかん)！？
B  : 말도 안 돼!? 3시간?!
Ｃ： （一口(ひとくち)食(た)べて）……やばい。
C  : (한 입 먹고) ……미쳤다.
Ｂ： まじで？ 俺(おれ)も一口(ひとくち)！
B  : 진짜? 나도 한 입!
```

### 4권 샘플 — 자막 한자 01 「텔롭 초빈출 10자」

> 선정 기준: 버라이어티 텔롭·애니 차회 예고에서 한 편에 몇 번씩 뜨는 자.

| 자 | 훈독 | 음독 | 화면 용례 (실제로 뜨는 형태) |
|---|---|---|---|
| 今 | いま | コン | 今日(きょう)はここまで！ 오늘은 여기까지! |
| 何 | なに/なん | カ | 今(いま)、何(なに)してる？ 지금 뭐 해? |
| 見 | み(る) | ケン | お見逃(みのが)しなく！ 놓치지 마세요! |
| 行 | い(く) | コウ | 行(い)ってきます！ 다녀오겠습니다! |
| 好 | す(き) | コウ | 大好(だいす)き 정말 좋아해 |
| 気 | — | キ/ケ | 続(つづ)きが気(き)になる… 다음 내용이 궁금하다… |
| 人 | ひと | ジン/ニン | あの人(ひと)、誰(だれ)？ 저 사람 누구? |
| 大 | おお(きい) | ダイ/タイ | 大丈夫(だいじょうぶ)？ 괜찮아? |
| 神 | かみ | シン | 神回(かみかい) 레전드 회차 |
| 回 | まわ(る) | カイ | 次回(じかい)、最終回(さいしゅうかい)。 다음 화, 최종화. |

**읽는 훈련** (쓰기보다 읽기 우선 — 자막은 읽는 것이므로):

```
次回(じかい)予告(よこく)：「転校生(てんこうせい)の秘密(ひみつ)」。お見逃(みのが)しなく！
차회 예고: "전학생의 비밀". 놓치지 마세요!

テロップ：スタジオ大爆笑(だいばくしょう)ｗｗｗ
텔롭: 스튜디오 대폭소 ㅋㅋㅋ
```

- 쓰기 과제는 주당 3자만(今·見·人). 나머지는 **읽기 인식**까지만 — 이 버전은 "화면이 읽히는 속도"를 최우선한다.

|---|---|---|---|---|---|
| 1 | 학원 편 ①~④ + 가나 병행 완성 | 해부 01~02 | 1장 리액션 | 텔롭 초빈출 50자(읽기) | 학원물 인사·리액션 장면이 부분적으로 들림 |
| 2 | 학원 편 ⑤~⑥ + 일상물 ⑦~⑧ | 해부 03~06 | 2장 감정 | 텔롭 100자 누적 | 짧은 반말 대화를 통째로 섀도잉 가능 |
| 3 | 일상물 ⑨~⑫ | 해부 07~10 | 3장 팬덤·라이브 | 가사 빈출 시작(150자 누적) | 좋아하는 곡 1곡의 가사를 후리가나로 통독 |
| 4 | **드라마 편 ⑬~⑯ (정중체 본격)** | 해부 11~14 + 격식 복원 특훈 | 6장 격식 변환 선행 학습 | 200자 누적 | 반말↔정중체 전환이 됨 — 실전 사용 허가선 |
| 5 | 드라마 편 ⑰~⑱ + 버라이어티 ⑲~⑳ | 해부 15~18 | 4장 먹방·의성어 | 드라마 화면 빈출(250자 누적) | 버라이어티 텔롭을 실시간으로 따라 읽음 |
| 6 | 버라이어티 ㉑~㉔ 총정리 | 해부 19~20 + 총복습 | 5장 명대사 + 6장 재정리 | 300자 완성 + 쓰기 보충 | 자막 없이 1편 시청 도전(이해율 50% 목표) |

**정직한 리스크 요약**: ① 반말이 먼저 굳는다 — 4개월차 정중체 블록과 매 장면의 변환 코너가 방어선이지만, 당장 일본인 직장 동료가 있는 사람에겐 권하지 않는다. ② 시험(JLPT) 대비 효율은 정통 구성보다 낮다 — 어휘 선정이 급수와 무관하기 때문. ③ "듣기 위주라 쓰기가 약하다"는 6개월 후에도 남는 약점이다.

## 부록 — 정통 구성과의 차이 한눈에

| 비교 축 | 정통 구성 | v06 미디어 몰입형 |
|---|---|---|
| 첫 말투 | 정중체(です/ます), 반말은 2권 후반 | **반말·구어 축약이 1주차부터** (정중체는 4개월차 드라마 편) |
| 대화 소재 | 상황별(공항·식당·길 묻기) | 장르별(학원물→일상물→드라마→버라이어티) |
| 문법 진행 방향 | 원형 → 활용 (연역) | **들린 축약형 → 표준형 복원** (역추적) |
| 한자 순서 | 급수·획수순, 읽기+쓰기 병행 | 화면 노출 빈도순, **읽기 인식 우선**(쓰기는 주 3자만) |
| 표현 선정 | 여행·생활 정중 표현 | 리액션·감정·팬덤 표현 + 친밀도 미터(◎/△/✕) |
| 동기 설계 | "6개월 뒤 여행에서 쓴다" | "이번 주에 이 장면이 들린다" — 보상 주기가 짧다 |
| 최적 병행 활동 | 회화 스터디 | 좋아하는 작품 1편을 "교재 밖 실전"으로 지정해 주 1회 시청 |

**갈아타기 지점**: 이 버전으로 3개월을 달린 뒤 "업무에 일본어가 필요해졌다"면, 4개월차 드라마 편을 앞당기고 3권 6장(격식 변환)을 먼저 읽는 것으로 정통 트랙에 합류할 수 있다 — 구어를 이미 알기 때문에 정중체 습득은 오히려 빠르다.
