# 버전 7

## 대화

**［카드 1］ はじめまして**
- 뜻: 처음 뵙겠습니다
- 쓰임: 첫 만남의 첫 마디. 자기소개를 열 때.

**［카드 2］ こんにちは**
- 뜻: 안녕하세요 (낮 인사)
- 쓰임: 낮에 만났을 때. 는 발음은 [わ].

**［카드 3］ さようなら**
- 뜻: 안녕히 가세요 / 안녕히 계세요
- 쓰임: 헤어질 때. 한동안 못 볼 때의 인사.

**［카드 4］ 私(わたし)は○○です**
- 뜻: 저는 ○○입니다
- 예: 私(わたし)はミナです。 저는 미나입니다.

**［카드 5］ ○○から来(き)ました**
- 뜻: ○○에서 왔습니다
- 예: 韓国(かんこく)から来(き)ました。 한국에서 왔습니다.

**［카드 6］ どうぞよろしくお願(ねが)いします**
- 뜻: 잘 부탁드립니다
- 답: こちらこそ、よろしくお願(ねが)いします。 저야말로 잘 부탁드립니다.

**［카드 7］ 이어 붙인 자기소개 대화**
> A: こんにちは。はじめまして。私(わたし)はミナです。
> 　 안녕하세요. 처음 뵙겠습니다. 저는 미나입니다.
> A: 韓国(かんこく)から来(き)ました。どうぞよろしくお願(ねが)いします。
> 　 한국에서 왔습니다. 잘 부탁드립니다.
> B: はじめまして。田中(たなか)です。こちらこそ、よろしくお願(ねが)いします。
> 　 처음 뵙겠습니다. 다나카입니다. 저야말로 잘 부탁드립니다.

## 문법

**［카드 8］ AはBです — 명사문 기본형**
- 골격: A **は** B **です** (A는 B입니다)
- 예: 私(わたし)は 学生(がくせい)です。 저는 학생입니다.

**［카드 9］ 주제 조사 は**
- 역할: 「〜은/는」 주제 표시
- 주의: 조사로 쓰일 때 발음은 [わ]. 私(わたし)は → 「わたしわ」

**［카드 10］ 〜ですか — 의문문**
- 만드는 법: 문장 끝에 **か**만 붙인다 (어순 그대로)
- 예: あなたは 学生(がくせい)ですか。 당신은 학생입니까?

**［카드 11］ 〜じゃありません — 부정문**
- 만드는 법: です → **じゃありません** (=では ありません)
- 예: 私(わたし)は 学生(がくせい)じゃありません。 저는 학생이 아닙니다.

**［카드 12］ これ / それ / あれ — 지시어**
- これ: 이것 (내 쪽에 가까움)
- それ: 그것 (상대 쪽에 가까움)
- あれ: 저것 (둘 다에게서 먼 것)
- 예: これは 本(ほん)です。 이것은 책입니다.

## 표현

**［카드 13］ おはようございます**
- 뜻: 안녕하세요 (아침 인사)

**［카드 14］ ありがとうございます**
- 뜻: 감사합니다

**［카드 15］ すみません**
- 뜻: 죄송합니다 / 실례합니다
- 쓰임: 사과할 때도, 사람을 부르거나 말 걸 때도.

**［카드 16］ いただきます**
- 뜻: 잘 먹겠습니다
- 짝: ごちそうさまでした。 잘 먹었습니다.

**［카드 17］ お願(ねが)いします**
- 뜻: 부탁합니다
- 쓰임: 주문·요청할 때. これ、お願(ねが)いします。 이거 부탁합니다.

**［카드 18］ はい / いいえ**
- 뜻: 네 / 아니요

## 한자

**［카드 19］ 숫자 一〜五**
- 一 いち (일) / 二 に (이) / 三 さん (삼) / 四 し·よん (사) / 五 ご (오)
- 힌트: 三=さん 은 한국음 '삼'과 소리가 닮았다.

**［카드 20］ 숫자 六〜十**
- 六 ろく (육) / 七 しち·なな (칠) / 八 はち (팔) / 九 きゅう·く (구) / 十 じゅう (십)

**［카드 21］ 日本(にほん)**
- 뜻: 일본
- 분해: 日(に) + 本(ほん). 한국음 '일본'의 '본'→ほん.

**［카드 22］ 人 — ひと / じん・にん**
- 훈독 ひと: 사람 (혼자 쓸 때)
- 음독 じん・にん: 〜인 (단어 속). 韓国人(かんこくじん) 한국인, 三人(さんにん) 세 명
- 힌트: 한국음 '인'→じん

**［카드 23］ 日 — ひ / にち**
- 훈독 ひ: 날·해 (혼자 쓸 때)
- 음독 にち: 日曜日(にちようび) 일요일, 毎日(まいにち) 매일

**［카드 24］ 종합 읽기**
- 三人(さんにん) 세 명 / 日本人(にほんじん) 일본인 / 二日(ふつか) 이틀·2일
