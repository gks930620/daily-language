# 제1장 인사와 자기소개

> **이 챕터에서 살아남기**: 만나서 인사하고, 이름을 말하고, 헤어질 때까지. 일본어 대화의 처음과 끝은 전부 여기 있는 표현으로 돌아갑니다. 아침에 일어나서 잘 때까지의 인사를 시간 순서대로 배치했으니, 위에서 아래로 소리 내어 읽으면 하루가 한 바퀴 돕니다.

---

## 1. 시간대별 기본 인사

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| おはようございます | 안녕하세요 (아침) | 아침에 만난 누구에게나. 직장에선 오후 출근이어도 첫인사로 씀 |
| こんにちは | 안녕하세요 (낮) | 낮에 만났을 때. 가게·길·이웃 누구에게나 |
| こんばんは | 안녕하세요 (저녁) | 해 진 뒤 만났을 때 |
| おやすみなさい | 안녕히 주무세요 / 잘 자요 | 자기 전, 밤에 헤어질 때 |
| いってきます | 다녀오겠습니다 | 집·사무실에서 나갈 때 |
| いってらっしゃい | 다녀오세요 / 잘 다녀와요 | 나가는 사람을 배웅할 때 |
| ただいま | 다녀왔습니다 | 집·사무실에 돌아왔을 때 |
| おかえりなさい | 어서 와요 / 왔어요? | 돌아온 사람을 맞을 때 |

💡 **한 줄 해설**
- **おはようございます** — 직역은 "이르시네요"지만 실제로는 "오늘 처음 뵙네요"에 가깝습니다. 그래서 일본 직장·아르바이트에서는 저녁에 출근해도 첫인사는 おはようございます. 친한 사이에선 おはよう로 줄입니다〔반말〕.

## 2. 처음 만났을 때

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| はじめまして | 처음 뵙겠습니다 | 첫 만남의 첫마디. 이 말로 시작하면 절반은 성공 |
| ○○と申（もう）します | ○○라고 합니다 | 정중한 자기소개. 면접·비즈니스에서도 OK |
| ○○です | ○○입니다 / ○○예요 | 가벼운 자기소개. 대부분의 상황은 이걸로 충분 |
| よろしくお願（ねが）いします | 잘 부탁드립니다 | 자기소개의 마무리 고정 멘트 |
| こちらこそ | 저야말로요 | 상대가 よろしく라고 했을 때 받아치는 말 |
| お名前（なまえ）は？ | 성함이 어떻게 되세요? | 이름을 물을 때. 뒤를 흐리는 게 오히려 자연스러움 |
| 韓国（かんこく）から来（き）ました | 한국에서 왔어요 | 여행·유학 자기소개의 단골 |
| 韓国人（かんこくじん）です | 한국 사람이에요 | 국적을 말할 때 |
| 会社員（かいしゃいん）です | 회사원이에요 | 직업 소개 |
| 学生（がくせい）です | 학생이에요 | 직업 소개 |
| 日本語（にほんご）は少（すこ）しだけです | 일본어는 조금밖에 못 해요 | 미리 말해 두면 상대가 천천히 말해 줌 |

💡 **한 줄 해설**
- **よろしくお願いします** — 딱 맞는 한국어가 없는 만능 표현. 첫인사 마무리, 일을 맡길 때("부탁드립니다"), 메일 끝인사까지 전부 커버합니다. 일단 첫 만남 끝에 무조건 붙인다고 외우세요.
- **○○と申します vs ○○です** — 申します는 겸양어라 한 단계 더 정중합니다. 여행 회화라면 です로 충분하고, 격식 자리(비즈니스·면접)에선 申します.

## 3. 안부와 재회

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| お元気（げんき）ですか | 잘 지내세요? | 오랜만에 만난 사람에게. 매일 보는 사이엔 잘 안 씀 |
| おかげさまで | 덕분에 잘 지내요 | お元気ですか에 대한 모범 답안 |
| お久（ひさ）しぶりです | 오랜만이에요 | 오랜만의 재회. 반말은 久しぶり！ |
| 元気（げんき）？〔반말〕 | 잘 지냈어? | 친구에게 |
| 最近（さいきん）どうですか | 요즘 어때요? | 가벼운 안부 |
| 相変（あいか）わらずです | 늘 똑같죠 뭐 | "요즘 어때요?"에 대한 무난한 답 |

💡 **한 줄 해설**
- **おかげさまで** — 직역 "덕분에"인데, 상대가 뭘 해 준 게 없어도 씁니다. "잘 지내요"를 겸손하게 말하는 관용구로, 이 한마디면 인품이 좋아 보입니다.

## 4. 헤어질 때

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| じゃ、また | 그럼 또 봐요 | 가장 흔한 작별 인사. 친구·동료에게 |
| また明日（あした） | 내일 봐요 | 내일 또 만날 사이 |
| さようなら | 안녕히 가세요 | 실제로는 긴 이별·학교에서 정도. 일상에선 드묾 |
| お先（さき）に失礼（しつれい）します | 먼저 들어가 보겠습니다 | 직장에서 먼저 퇴근할 때의 고정 멘트 |
| お疲（つか）れさまです | 수고하십니다 / 수고하셨어요 | 직장 만능 인사. 만날 때도 헤어질 때도 |
| 失礼（しつれい）します | 실례하겠습니다 | 방에 들어갈 때, 나올 때, 전화 끊을 때 |
| 気（き）をつけて | 조심히 가요 | 배웅하며 |
| お世話（せわ）になりました | 그동안 감사했습니다 | 신세 진 사람과 헤어질 때 (호텔 체크아웃, 퇴사 등) |

💡 **한 줄 해설**
- **さようなら** — 교과서 1페이지에 나오지만 실생활에선 "다시 안 볼 사람" 느낌이 나서 거의 안 씁니다. 일상 작별은 じゃ、また / お疲れさまです / 失礼します 셋으로 해결하세요.
- **お疲れさまです** — 직역 "피곤하시겠네요"가 아니라 직장의 "안녕하세요+수고하세요+안녕히 가세요"를 전부 겸하는 만능 인사. 복도에서 스쳐도, 퇴근할 때도, 메신저 첫줄에도 이것.

## 5. 감사와 사과 (기본형)

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| ありがとうございます | 감사합니다 | 만능 감사. 과거 일에는 ありがとうございました |
| どういたしまして | 별말씀을요 | 감사 인사를 받았을 때 (실제로는 いえいえ가 더 흔함) |
| すみません | 죄송해요 / 저기요 / 고마워요 | 사과·부르기·가벼운 감사, 3역을 다 하는 최중요 단어 |
| ごめんなさい | 미안해요 | 개인적인 잘못을 사과할 때. 친근한 느낌 |
| 大丈夫（だいじょうぶ）です | 괜찮아요 | 사과를 받아 줄 때, 걱정을 덜어 줄 때 |
| いえいえ | 아니에요~ | 감사·사과 양쪽에 가볍게 손사래 치는 말 |

💡 **한 줄 해설**
- **すみません의 3용법** — ① 사과 "죄송합니다" ② 호출 "저기요!"(점원 부를 때) ③ 감사 "(수고를 끼쳐) 고마워요"(길을 비켜 준 사람에게). 일본에서 하루에 가장 많이 듣는 단어이니 세 얼굴을 다 익혀 두세요.

## 미니 대화 — 교류 모임에서 처음 만난 두 사람

**김**: はじめまして。キムと申（もう）します。
→ 처음 뵙겠습니다. 김이라고 합니다.

**다나카**: はじめまして。田中（たなか）です。よろしくお願（ねが）いします。
→ 처음 뵙겠습니다. 다나카예요. 잘 부탁드립니다.

**김**: こちらこそ。韓国（かんこく）から来（き）ました。日本語（にほんご）は少（すこ）しだけです。
→ 저야말로요. 한국에서 왔어요. 일본어는 조금밖에 못 해요.

**다나카**: いえいえ、お上手（じょうず）ですよ。お仕事（しごと）は？
→ 아니에요, 잘하시는데요. 일은 뭐 하세요?

**김**: 会社員（かいしゃいん）です。田中（たなか）さんは？
→ 회사원이에요. 다나카 씨는요?

**다나카**: 私（わたし）も会社員（かいしゃいん）です。あ、もう時間（じかん）ですね。じゃ、また。
→ 저도 회사원이에요. 아, 벌써 시간이 됐네요. 그럼 또 봬요.

**김**: はい、また。気（き）をつけて。
→ 네, 또 봬요. 조심히 가세요.

**여기서 굴러간 표현**: はじめまして / 〜と申します / よろしくお願いします / こちらこそ / 韓国から来ました / じゃ、また / 気をつけて
