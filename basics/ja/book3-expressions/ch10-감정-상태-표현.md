# 제10장 감정·상태 표현

> **이 챕터에서 살아남기**: "배고파요, 피곤해요, 기뻐요" — 내 상태를 말할 수 있으면 대화가 살아납니다. 일본어 감정 표현의 특징은 형용사 하나로 문장이 끝난다는 것(うれしい！만으로 "기뻐!"). 몸 상태 표현은 11장의 병원 표현과 이어지니 여기서 기초를 다져 두세요.

---

## 1. 기쁨과 즐거움

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| うれしいです | 기뻐요 | 선물·칭찬·좋은 소식을 받았을 때 |
| 楽（たの）しいです | 즐거워요 / 재밌어요 | 지금 이 시간이 즐거울 때 |
| 楽（たの）しかったです | 즐거웠어요 | 헤어질 때의 최고 인사. 今日は楽しかったです |
| 面白（おもしろ）いですね | 재밌네요 / 흥미롭네요 | 콘텐츠·이야기가 재미있을 때 |
| 幸（しあわ）せです | 행복해요 | 맛있는 걸 먹을 때도 과장 없이 씀 |
| 最高（さいこう）です | 최고예요 | 강한 만족. 반말은 最高！ |
| 楽（たの）しみです | 기대돼요 | 앞으로의 일이 기다려질 때 |
| よかった！ | 다행이다! / 잘됐다! | 안도의 탄성 |

💡 **한 줄 해설**
- **楽しい vs 面白い** — 楽しい는 몸으로 겪는 즐거움(여행·파티가 즐겁다), 面白い는 머리로 느끼는 재미(영화·이야기가 재밌다/흥미롭다). 여행은 楽しかった, 영화는 面白かった가 자연스럽습니다.
- **楽しみです** — 直역 "즐거움입니다"가 아니라 "기대돼요". 약속을 잡고 헤어질 때 楽しみにしています(기대하고 있을게요)라고 하면 다음 만남이 기분 좋게 예약됩니다.

## 3. 몸의 상태 — 배고픔부터 컨디션까지

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| お腹（なか）がすきました | 배고파요 | 친구끼리는 お腹すいた〔반말〕 |
| お腹（なか）がいっぱいです | 배불러요 | 잘 먹었다는 신호로도 |
| のどが渇（かわ）きました | 목말라요 | |
| 疲（つか）れました | 피곤해요 / 지쳤어요 | 오늘 하루에 지쳤을 때 |
| 眠（ねむ）いです | 졸려요 | |
| 昨日（きのう）、眠（ねむ）れませんでした | 어제 잠을 못 잤어요 | 컨디션 난조의 이유 설명 |
| 暑（あつ）いです／寒（さむ）いです | 더워요 / 추워요 | 날씨·실내 온도 |
| 気持（きも）ちいいです | 상쾌해요 / 시원해요 | 바람·온천·날씨가 좋을 때 |
| 気持（きも）ち悪（わる）いです | 속이 안 좋아요 / 징그러워요 | 멀미·과음, 또는 혐오감 |
| 気分（きぶん）が悪（わる）いです | 몸이 안 좋아요 / 기분이 상해요 | 주로 몸 상태. 문맥에 따라 기분 |
| 具合（ぐあい）が悪（わる）いです | 몸 상태가 안 좋아요 | 아픈 쪽에 확실히 기운 표현 |
| 頭（あたま）が痛（いた）いです | 머리가 아파요 | お腹が痛い = 배가 아파요 |
| 風邪（かぜ）をひきました | 감기 걸렸어요 | |

💡 **한 줄 해설**
- **気持ち悪い 주의보** — "기분이 나쁘다"로 직역하면 오해합니다. 실제 뜻은 ① 속이 메스껍다(멀미·숙취) ② 징그럽다·소름 끼친다. 감정이 상했다는 "기분 나쁘다"와는 다르니, 몸이 안 좋으면 具合が悪いです가 오해 없는 선택입니다.
- **お腹がすきました** — 형태는 과거형이지만 뜻은 현재 "배고파요"입니다. 배고픔·목마름·피곤함은 "그 상태가 되었다"는 과거형으로 현재 상태를 말하는 것이 일본어의 습관(すいた・渇いた・疲れた).

## 4. 상태와 근황

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| 忙（いそが）しいです | 바빠요 | 最近忙しいです = 요즘 바빠요 |
| 暇（ひま）です | 한가해요 | 暇なとき = 한가할 때 |
| 大丈夫（だいじょうぶ）？〔반말〕 | 괜찮아? | 상대가 안 좋아 보일 때 |
| 元気（げんき）がないですね | 기운이 없어 보여요 | 걱정의 말 걸기 |
| 気（き）になります | 신경 쓰여요 / 궁금해요 | 마음에 걸리는 것, 궁금한 것 |
| ドキドキします | (심장이) 두근거려요 | 긴장·설렘 양쪽 |
| ワクワクします | (기대로) 설레요 | 두근두근 중 좋은 쪽만 |
| ほっとしました | 한시름 놨어요 / 안심했어요 | 걱정이 풀렸을 때 |
| 落（お）ち込（こ）んでいます | 우울해요 / 축 처져 있어요 | 실패 후의 침울함 |

💡 **한 줄 해설**
- **ドキドキ vs ワクワク** — 일본어 의태어의 대표 쌍. ドキドキ는 긴장이든 설렘이든 심장이 뛰는 것 전부, ワクワク는 소풍 전날 같은 신나는 기대만. 데이트 전엔 둘 다 가능하고, 면접 전엔 ドキドキ만 됩니다.

## 미니 대화 — 컨디션 안 좋은 친구 챙기기 〔반말 회화〕

**유이**: あれ、どうしたの？元気（げんき）がないね。
→ 어라, 왜 그래? 기운이 없네.

**나**: うん、ちょっと疲（つか）れた。昨日（きのう）、眠（ねむ）れなかったんだ。
→ 응, 좀 지쳤어. 어제 잠을 못 잤거든.

**유이**: 大丈夫（だいじょうぶ）？具合（ぐあい）が悪（わる）いの？
→ 괜찮아? 몸이 안 좋은 거야?

**나**: ううん、明日（あした）の面接（めんせつ）が心配（しんぱい）で。緊張（きんちょう）してる。
→ 아니, 내일 면접이 걱정돼서. 긴장돼.

**유이**: そっか。でもキムなら大丈夫（だいじょうぶ）だよ。
→ 그렇구나. 근데 너라면 괜찮을 거야.

**나**: ありがとう。そう言（い）ってもらえて、ちょっとほっとした。
→ 고마워. 그렇게 말해 주니까 좀 마음이 놓인다.

**유이**: 終（お）わったら焼肉（やきにく）行（い）こう！
→ 끝나면 고기 먹으러 가자!

**나**: いいね！楽（たの）しみ！
→ 좋아! 기대된다!

**여기서 굴러간 표현**: 元気がない / 疲れた / 眠れなかった / 大丈夫？ / 具合が悪い / 心配 / 緊張してる / ほっとした / 楽しみ
