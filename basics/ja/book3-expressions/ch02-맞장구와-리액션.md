# 제2장 맞장구와 리액션

> **이 챕터에서 살아남기**: 일본어 대화는 맞장구(あいづち)가 절반입니다. 일본인은 상대가 말하는 동안 끊임없이 고개를 끄덕이며 반응하는데, 이게 없으면 "듣고 있나?" 하고 불안해합니다. 문장을 못 만들어도 이 챕터의 리액션만 제때 던지면 "대화가 되는 사람"이 됩니다.

---

## 1. 기본 맞장구 — 일단 이 여섯 개

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| そうですね | 그러네요 / 글쎄요 | 동의할 때. 뜸 들일 때는 "글쎄요"가 됨 |
| そうですか | 그래요? / 그렇군요 | 처음 듣는 얘기에. 억양 내리면 수긍, 올리면 놀람 |
| そうなんです | 그렇다니까요 / 맞아요 | 상대가 정곡을 찔렀을 때 "바로 그거예요" |
| そうそう〔반말〕 | 맞아 맞아 | 친구가 내 말을 대신 해 줬을 때 |
| なるほど | 아하, 그렇구나 | 설명을 듣고 이해했을 때 |
| 確（たし）かに | 하긴 / 듣고 보니 그러네요 | 반박 못 할 지적을 받았을 때 |

💡 **한 줄 해설**
- **そうですね vs そうですか** — ね는 "나도 그렇게 생각해요"(공감), か는 "그건 몰랐네요"(새 정보 접수). 상대 말에 동의하면 ね, 처음 알았으면 か. 이 한 글자가 대화의 방향을 정합니다.
- **なるほど** — 편리하지만 "당신 말을 평가해 줌" 뉘앙스가 살짝 있어 손윗사람·고객에게 연발하면 건방져 보일 수 있습니다. 정중하게는 そうなんですね가 안전.

---

## 2. 놀람과 감탄

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| 本当（ほんとう）ですか | 정말요? | 놀라운 얘기를 들었을 때의 표준 리액션 |
| ほんと？〔반말〕 | 진짜? | 친구 사이 |
| うそ！〔반말〕 | 헐, 말도 안 돼 | "거짓말"이라는 비난이 아니라 그냥 놀람 |
| まじで？〔반말〕 | 진심? 레알? | 젊은 층 구어 |
| えー！ | 에엥?! | 놀람·가벼운 불만. 억양으로 감정이 다 실림 |
| へえ | 오~ 호오 | 흥미로운 얘기에. 감탄의 길이만큼 관심의 크기 |
| すごいですね | 대단하네요 | 상대 자랑에 대한 표준 반응. 친구에겐すごい！ |
| いいですね | 좋네요 / 좋겠어요 | 제안에 찬성할 때, 부러울 때 둘 다 |
| うらやましいです | 부럽네요 | 상대의 좋은 소식에 |
| まさか | 설마 | 믿기 힘든 전개에 |

💡 **한 줄 해설**
- **うそ！** — 직역 "거짓말!"이지만 상대를 의심하는 게 아니라 한국어 "헐/대박"에 해당하는 놀람 감탄사입니다. 단, 반말이므로 손윗사람에게는 本当ですか로.

---

## 3. 공감과 위로

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| よかったですね | 잘됐네요 / 다행이네요 | 좋은 소식에 |
| 大変（たいへん）でしたね | 고생하셨네요 / 힘드셨겠어요 | 고생담을 들었을 때 |
| 残念（ざんねん）ですね | 아쉽네요 / 안타깝네요 | 아쉬운 결과에 |
| わかります | 그 마음 알아요 | 공감을 표할 때 |
| ですよね | 그렇죠? 그니까요 | "나도 그렇게 생각했어요" 공감 확인 |
| でしょう？ | 그렇죠? 내 말이 맞죠? | 내가 한 말·추천이 적중했을 때 |
| 大丈夫（だいじょうぶ）ですか | 괜찮으세요? | 상대가 아프거나 곤란해 보일 때 |
| 気（き）にしないでください | 신경 쓰지 마세요 | 사과해 오는 상대를 안심시킬 때 |

---

## 4. 대답과 얼버무림

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| ええ | 네 (맞장구용) | はい보다 부드러운 "네에" |
| うん／ううん〔반말〕 | 응 / 아니 | 발음 주의: 올리면 긍정, 첫음 길면 부정 |
| もちろんです | 물론이죠 | 흔쾌한 승낙 |
| たぶん | 아마도 | 확신이 없을 때 문장 앞에 |
| どうでしょうね | 글쎄요, 어떨까요 | 즉답을 피할 때 |
| それはちょっと… | 그건 좀… | 부드러운 난색·거절 신호 (9장에서 자세히) |
| とんでもないです | 별말씀을요 / 당치도 않아요 | 과한 칭찬·감사를 겸손하게 받을 때 |
| こちらこそ | 저야말로요 | 감사·사과를 맞받을 때 |

💡 **한 줄 해설**
- **はい、はい (두 번)** — "네네"를 두 번 연달아 하면 한국어와 똑같이 건성으로 들려 실례입니다. 대답은 늘 한 번, はい.
- **ですよね vs でしょう？** — ですよね는 상대 말에 올라타는 공감("그니까요"), でしょう？는 내 말이 맞았음을 확인받는 것("거 봐요, 맞죠?"). 주어가 상대냐 나냐의 차이.

## 미니 대화 — 동료의 여행 자랑 들어 주기

**사토**: 先週（せんしゅう）、韓国（かんこく）に行（い）ってきました。
→ 지난주에 한국에 다녀왔어요.

**나**: えー！本当（ほんとう）ですか。いいですね。
→ 엥! 정말요? 좋았겠다.

**사토**: 食（た）べ物（もの）が全部（ぜんぶ）おいしくて。三日（みっか）で三（さん）キロ太（ふと）りました。
→ 음식이 전부 맛있어서요. 사흘 만에 3킬로 쪘어요.

**나**: へえ、すごいですね。でも、わかります。韓国（かんこく）の食（た）べ物（もの）、おいしいですから。
→ 오~ 대단하네요. 근데 그 마음 알아요. 한국 음식 맛있으니까요.

**사토**: ですよね。特（とく）に焼肉（やきにく）が最高（さいこう）でした。
→ 그니까요. 특히 고기구이가 최고였어요.

**나**: 確（たし）かに。あ、写真（しゃしん）ありますか。
→ 하긴요. 아, 사진 있어요?

**사토**: もちろんです。これです。
→ 물론이죠. 이거예요.

**나**: うわ、おいしそう！うらやましいです。
→ 우와, 맛있겠다! 부럽네요.

**여기서 굴러간 표현**: 本当ですか / いいですね / へえ / すごいですね / わかります / ですよね / 確かに / もちろんです / うらやましいです
