# 제3장 숫자·시간·돈

> **이 챕터에서 살아남기**: 여행에서 실제로 오가는 말의 절반은 숫자입니다. "얼마예요?"를 말하는 건 쉽지만 대답을 못 알아들으면 소용없으니, 이 챕터는 묻는 표현과 함께 **듣게 될 대답의 형태**까지 세트로 익힙니다.

---

## 1. 개수 세기 — 고유수사와 사람 수

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| 一（ひと）つ／二（ふた）つ／三（みっ）つ | 하나 / 둘 / 셋 | 물건 개수의 만능 단위. 열(とお)까지만 외우면 됨 |
| これ、二（ふた）つください | 이거 두 개 주세요 | 가게에서 개수 말하기의 기본형 |
| 一人（ひとり）です | 한 명이에요 | 식당 입구에서 |
| 二人（ふたり）です | 두 명이에요 | 식당 입구에서. 셋부터는 三人（さんにん） |
| 一枚（いちまい）ください | 한 장 주세요 | 표·종이 등 얇은 것 |
| 一個（いっこ）ください | 한 개 주세요 | 작은 물건 |
| 一本（いっぽん） | 한 병 / 한 자루 | 병·우산 등 긴 것. 二本（にほん）·三本（さんぼん）으로 소리가 변함 |

💡 **한 줄 해설**
- **ひとつ・ふたつ 만능론** — 일본어의 단위(助数詞)는 수십 가지지만, 입문 단계에선 물건은 전부 ひとつ・ふたつ로 세도 통합니다. 단 사람만은 반드시 ひとり・ふたり・〜にん. "맥주 둘, 사람 둘"은 ビールふたつ、ふたりです.

---

## 2. 시간 묻고 답하기

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| 今（いま）、何時（なんじ）ですか | 지금 몇 시예요? | 시간 묻기의 기본 |
| 三時（さんじ）です | 3시예요 | 대답. 4시=よじ, 7시=しちじ, 9시=くじ (예외 주의) |
| 三時半（さんじはん）です | 3시 반이에요 | 30분은 半 하나로 해결 |
| ちょうど | 정각 / 딱 | 三時ちょうど = 3시 정각 |
| 何時（なんじ）からですか | 몇 시부터예요? | 개점·개장 시간 묻기 |
| 何時（なんじ）までですか | 몇 시까지예요? | 폐점·마감 시간 묻기 |
| 午前（ごぜん）／午後（ごご） | 오전 / 오후 | 午後三時 = 오후 3시 |
| どのくらいかかりますか | 얼마나 걸려요? | 소요 시간 묻기 (돈에도 씀: 얼마나 들어요?) |
| 三十分（さんじゅっぷん）ぐらいです | 30분 정도예요 | 대답. ぐらい = ~정도 |
| もうすぐです | 곧이에요 / 다 왔어요 | 기다리는 사람에게 |

💡 **한 줄 해설**
- **4시·7시·9시** — よじ・しちじ・くじ. 「よんじ」「ななじ」「きゅうじ」라고 하면 바로 티가 납니다. 이 세 개만 예외로 따로 외우세요.
- **どのくらいかかりますか** — かかる는 시간에도 돈에도 쓰는 동사라, 이 한 문장으로 "얼마나 걸려요?"와 "비용이 얼마나 들어요?"를 다 물을 수 있습니다. 문맥이 정해 줍니다.

## 3. 날짜와 요일

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| 今日（きょう）／明日（あした）／昨日（きのう） | 오늘 / 내일 / 어제 | 훈독 3형제. 표기와 소리가 안 맞으니 통째로 암기 |
| 今週（こんしゅう）／来週（らいしゅう） | 이번 주 / 다음 주 | 先週（せんしゅう）= 지난주 |
| 何曜日（なんようび）ですか | 무슨 요일이에요? | 月・火・水・木・金・土・日+曜日 |
| 何日（なんにち）ですか | 며칠이에요? | 1일=ついたち, 2일=ふつか… 10일까지는 고유어 |
| 週末（しゅうまつ） | 주말 | 週末は何をしますか = 주말에 뭐 하세요? |
| 平日（へいじつ） | 평일 | 平日だけ = 평일만 |

## 4. 돈 — 가격 묻기와 계산

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| いくらですか | 얼마예요? | 가격 묻기의 기본. 손가락으로 가리키며 これ、いくらですか |
| 全部（ぜんぶ）でいくらですか | 전부 얼마예요? | 여러 개 살 때 합계 묻기 |
| 千円（せんえん）です | 천 엔입니다 | 듣게 될 대답. 백=ひゃく, 천=せん, 만=まん |
| 一万円（いちまんえん） | 만 엔 | "만"은 반드시 一万. 그냥 まん이라고 안 함 |
| 高（たか）いですね | 비싸네요 | 혼잣말처럼. 흥정 문화는 거의 없음 |
| 安（やす）いですね | 싸네요 | 득템했을 때 |
| お金（かね）が足（た）りません | 돈이 모자라요 | 계산대에서 위기 상황 |
| お釣（つ）り | 거스름돈 | お釣りです = 거스름돈입니다 (듣는 말) |
| 細（こま）かいお金（かね）がありません | 잔돈이 없어요 | 만 엔짜리밖에 없을 때 미리 한마디 |
| 両替（りょうがえ）できますか | 환전 되나요? / 잔돈으로 바꿔 주실 수 있나요? | 환전소·가게에서 |
| 無料（むりょう）／有料（ゆうりょう） | 무료 / 유료 | 표지판·안내에서 자주 보는 말 |
| 一人（ひとり）いくらですか | 1인당 얼마예요? | 인원제 요금 묻기 |

💡 **한 줄 해설**
- **엔화 숫자의 소리 변화** — 300=さんびゃく, 600=ろっぴゃく, 800=はっぴゃく, 3000=さんぜん, 8000=はっせん. 자주 틀리는 다섯 개만 예외로 외우면 나머지는 규칙대로입니다.
- **高いですね** — 일본은 정찰제 문화라 흥정용이 아니라 감상용입니다. 깎아 달라는 신호로 오해받지 않으니 편하게 중얼거려도 됩니다.

## 5. 번호와 순서

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| 何番（なんばん）ですか | 몇 번이에요? | 대기 번호·승강장 번호 묻기 |
| 番号（ばんごう） | 번호 | 電話番号 = 전화번호 |
| 一番（いちばん） | 1번 / 제일 | 一番おいしい = 제일 맛있는 (최상급도 이 단어) |
| 次（つぎ）の方（かた）どうぞ | 다음 분 오세요 | 창구·계산대에서 듣는 말 |
| 順番（じゅんばん）にお並（なら）びください | 순서대로 줄 서 주세요 | 안내 방송·표지판 |

## 미니 대화 — 빵집에서 계산까지

**나**: すみません、これはいくらですか。
→ 저기요, 이건 얼마예요?

**점원**: そちらは二百八十円（にひゃくはちじゅうえん）です。
→ 그건 280엔입니다.

**나**: じゃ、これを二（ふた）つと、あれを一（ひと）つください。
→ 그럼 이거 두 개랑 저거 하나 주세요.

**점원**: はい、全部（ぜんぶ）で八百四十円（はっぴゃくよんじゅうえん）です。
→ 네, 전부 840엔입니다.

**나**: すみません、細（こま）かいお金（かね）がありません。一万円（いちまんえん）でもいいですか。
→ 죄송한데 잔돈이 없어서요. 만 엔짜리도 괜찮을까요?

**점원**: 大丈夫（だいじょうぶ）ですよ。九千百六十円（きゅうせんひゃくろくじゅうえん）のお釣（つ）りです。
→ 괜찮습니다. 거스름돈 9,160엔입니다.

**나**: ありがとうございます。あ、何時（なんじ）までですか。
→ 감사합니다. 아, 몇 시까지 하세요?

**점원**: 午後（ごご）七時（しちじ）までです。
→ 오후 7시까지입니다.

**여기서 굴러간 표현**: いくらですか / ふたつ・ひとつください / 全部でいくら(です) / 細かいお金がありません / お釣り / 何時までですか
