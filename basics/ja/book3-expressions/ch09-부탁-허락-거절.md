# 제9장 부탁·허락·거절 — 일본어다운 애매함

> **이 챕터에서 살아남기**: 일본어에서 "No"는 거의 발음되지 않습니다. 거절은 ちょっと…(좀…)처럼 **문장을 끝까지 말하지 않는 것**으로 전달되고, 상대도 그걸로 알아듣습니다. 이 챕터는 부탁하는 법·승낙하는 법과 함께, 이 "애매함의 문법"을 해독하는 법을 다룹니다. 이걸 모르면 일본인의 거절을 승낙으로 오해하는 사고가 납니다.

---

## 1. 부탁하기 — 부드러움의 단계

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| お願（ねが）いがあるんですが | 부탁이 좀 있는데요 | 본격적인 부탁의 서두 |
| ちょっといいですか | 잠깐 괜찮으세요? | 말 걸기·시간 뺏기 전의 노크 |
| 〜てもらえますか | ~해 줄 수 있어요? | 기본 부탁형. 開けてもらえますか = 열어 줄래요? |
| 〜ていただけますか | ~해 주시겠어요? | 한 단계 더 정중. 점원·모르는 사람에게 |
| 写真（しゃしん）を撮（と）っていただけますか | 사진 좀 찍어 주시겠어요? | 여행 필수 부탁 |
| 手伝（てつだ）ってもらえますか | 도와줄 수 있어요? | 도움 요청 |
| ちょっと待（ま）ってください | 잠깐만 기다려 주세요 | 급할 땐 ちょっと待って！〔반말〕 |
| できれば今日中（きょうじゅう）に | 가능하면 오늘 안에 | 부탁에 조건 달기 |
| 助（たす）かります | (그렇게 해 주시면) 정말 감사하죠 | 부탁을 들어주려는 사람에게 |
| 無理（むり）なら大丈夫（だいじょうぶ）です | 어려우시면 괜찮아요 | 부담을 덜어 주는 쿠션 |

💡 **한 줄 해설**
- **〜てもらえますか / 〜ていただけますか** — 직역하면 "~해 받을 수 있을까요?"라는 요상한 구조지만, 일본어 부탁의 표준 문형입니다. もらえますか(친근·동등) < いただけますか(정중)의 2단계만 기억하면 모든 부탁이 됩니다.
- **助かります** — 직역 "살았습니다(구조됨)". 실제로는 "그래 주시면 큰 도움이 돼요"라는 감사 표현으로, 부탁을 들어준 사람에게 ありがとうございます、助かりました라고 하면 감사가 두 배로 전해집니다.

## 2. 허락 구하기와 승낙하기

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| 〜てもいいですか | ~해도 될까요? | 허락 구하기 만능형 |
| ここに座（すわ）ってもいいですか | 여기 앉아도 될까요? | 옆자리 확인 |
| これ、使（つか）ってもいいですか | 이거 써도 될까요? | 물건 빌릴 때 |
| どうぞ | 그러세요 / 쓰세요 / 먼저 하세요 | 허락·양보·건네기의 만능 응답 |
| いいですよ | 좋아요, 그러세요 | 가벼운 승낙 |
| もちろんです | 물론이죠 | 흔쾌한 승낙 |
| 全然（ぜんぜん）大丈夫（だいじょうぶ）です | 전혀 문제없어요 | 상대의 미안함을 덜어 주는 승낙 |
| お先（さき）にどうぞ | 먼저 하세요 / 먼저 가세요 | 문·엘리베이터·줄에서 양보 |
| 遠慮（えんりょ）しないでください | 사양하지 마세요 | 상대가 머뭇거릴 때 |
| お言葉（ことば）に甘（あま）えて | 그럼 염치 불고하고 | 호의를 받아들일 때의 세련된 한마디 |

💡 **한 줄 해설**
- **どうぞ** — 한 단어로 "앉으세요·드세요·먼저 가세요·여기요(건네며)·그러세요"를 전부 해결하는 만능 허락어. 상대가 〜てもいいですか라고 물으면 どうぞ 한마디가 가장 자연스러운 대답입니다.
- **お言葉に甘えて** — 직역 "말씀에 응석 부려서". 상대가 "사양 말고 그렇게 해요"라고 했을 때 "그럼 호의를 받아들일게요"라고 응하는 관용구입니다. 사양 문화의 일본에서 이 말을 쓰면 주고받기가 아름답게 끝납니다.

## 3. 거절하기 — 애매함의 문법

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| すみませんが、ちょっと… | 죄송하지만, 좀… | 일본어 거절의 표준형. 뒷말은 안 함 |
| 今日（きょう）はちょっと… | 오늘은 좀… | 약속 거절. "안 돼요"라고 말하지 않는 "안 돼요" |
| それはちょっと… | 그건 좀… | 제안에 대한 난색 |
| また今度（こんど）、ぜひ | 다음에 꼭이요 | 거절 후 관계를 지키는 말 (다음은 대체로 없음) |
| 考（かんが）えておきます | 생각해 볼게요 | 사실상 정중한 No인 경우가 많음 |
| 難（むずか）しいですね | 어렵겠네요 | 비즈니스에서의 사실상 거절 |
| 微妙（びみょう）ですね | 애매하네요 / 글쎄요 | 긍정도 부정도 아닌, 대체로 부정 |
| 結構（けっこう）です | 괜찮습니다 (사양) | 권유 거절. 어조가 차가우면 "됐거든요"가 되니 웃으며 |
| 遠慮（えんりょ）しておきます | 사양할게요 | 結構です보다 부드러운 거절 |
| 大丈夫（だいじょうぶ）です | 괜찮아요 (필요 없어요) | 젊은 층이 제일 많이 쓰는 부드러운 거절 |
| 申（もう）し訳（わけ）ないんですが | 정말 죄송하지만 | 거절·어려운 부탁의 서두 쿠션 |
| だめですか | 안 될까요? | 거절당할 것 같을 때 한 번 더 밀어 보기 |

💡 **한 줄 해설**
- **ちょっと…의 해독법** — 일본인이 ちょっと…라고 말끝을 흐리면 그건 거의 확실한 **거절**입니다. "조금…뭐요?"라고 되물으면 상대를 곤란하게 만드는 것. そうですか、わかりました(그렇군요, 알겠습니다)로 받아 주는 것이 정답입니다.
- **考えておきます** — 직역 "생각해 두겠습니다"이지만, 가게에서든 제안에서든 십중팔구 "(안 살 거지만/안 할 거지만) 정중하게 마무리할게요"입니다. 진짜로 검토할 때는 前向（まえむ）きに検討（けんとう）します 같은 더 적극적인 말이 붙습니다.
- **結構です** — "좋습니다(승낙)"와 "됐습니다(거절)" 양쪽 다 가능한 위험한 단어. 초급자는 거절이라면 大丈夫です, 승낙이라면 お願いします로 갈아타는 것이 안전합니다.

## 미니 대화 — 회식 권유를 부드럽게 거절하기

**선배**: キムさん、今晩（こんばん）みんなで飲（の）みに行（い）くんだけど、どう？
→ 김 씨, 오늘 밤 다 같이 한잔하러 가는데, 어때? 〔반말〕

**나**: あ、すみません。今日（きょう）はちょっと…
→ 아, 죄송해요. 오늘은 좀…

**선배**: そっか、残念（ざんねん）。用事（ようじ）？
→ 그렇구나, 아쉽네. 볼일 있어?

**나**: はい、明日（あした）朝（あさ）が早（はや）いんです。申（もう）し訳（わけ）ないんですが。
→ 네, 내일 아침이 일러서요. 정말 죄송해요.

**선배**: いやいや、全然（ぜんぜん）大丈夫（だいじょうぶ）。また今度（こんど）ね。
→ 아냐 아냐, 전혀 괜찮아. 다음에 가자.

**나**: はい、また今度（こんど）、ぜひ。お誘（さそ）いありがとうございます。
→ 네, 다음에 꼭이요. 불러 주셔서 감사해요.

**선배**: じゃ、お疲（つか）れさま！
→ 그럼 수고!

**나**: お疲（つか）れさまです。
→ 수고하셨습니다.

**여기서 굴러간 표현**: 今日はちょっと… / 申し訳ないんですが / 全然大丈夫です / また今度、ぜひ / お疲れさまです
