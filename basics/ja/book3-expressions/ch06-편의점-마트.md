# 제6장 편의점·마트

> **이 챕터에서 살아남기**: 편의점은 일본 여행에서 가장 자주 가는 곳인데, 특이하게도 **내가 말할 일보다 점원의 질문을 알아들을 일**이 많습니다. "데워 드릴까요?" "봉투 필요하세요?" "포인트 카드 있으세요?" — 이 3연속 질문에 막힘없이 답하는 것이 이 챕터의 목표입니다. 표에서 (듣) 표시는 주로 듣게 되는 말입니다.

---

## 1. 계산대 3연속 질문 — 듣고 답하기

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| 温（あたた）めますか (듣) | 데워 드릴까요? | 도시락·호빵 계산 시 반드시 나옴 |
| お願（ねが）いします | 네, 데워 주세요 | 승낙의 만능 대답 |
| そのままで大丈夫（だいじょうぶ）です | 그냥 주세요 | 안 데워도 될 때 |
| 袋（ふくろ）はご利用（りよう）ですか (듣) | 봉투 필요하세요? | 봉투는 유료 |
| 袋（ふくろ）お願（ねが）いします | 봉투 하나 주세요 | |
| 袋（ふくろ）は大丈夫（だいじょうぶ）です | 봉투는 괜찮아요(안 주셔도 돼요) | 거절의 大丈夫 |
| ポイントカードはお持（も）ちですか (듣) | 포인트 카드 있으세요? | 여행자는 없다고 하면 끝 |
| ないです | 없어요 | 짧고 충분한 대답 |
| お箸（はし）はおつけしますか (듣) | 젓가락 넣어 드릴까요? | 도시락 살 때 |
| お箸（はし）、二（ふた）つください | 젓가락 두 개 주세요 | 개수 지정 |
| スプーンください | 숟가락 주세요 | 아이스크림·푸딩 등 |

💡 **한 줄 해설**
- **大丈夫です의 두 얼굴** — "괜찮아요"는 한국어처럼 **거절**("필요 없어요")과 **승낙**("문제없어요") 둘 다 됩니다. 편의점에서 袋は大丈夫です라고 하면 100% "봉투 필요 없어요"로 통하니 안심하고 쓰세요. 헷갈리면 いりません(필요 없어요)이 확실.
- **温めますか** — あたためますか. 빨리 말하면 "아타타메마스카"가 뭉개져서 들립니다. 도시락을 올려놨을 때 뭔가 물어보면 십중팔구 이 질문이니, 내용보다 타이밍으로 알아채세요.

## 2. 결제 순간

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| カードで | 카드로요 | 결제 수단은 짧게 ~で로 끝남 |
| 現金（げんきん）で | 현금으로요 | |
| 交通（こうつう）系（けい）で | 교통카드(스이카 등)로요 | 편의점 결제 가능 |
| バーコードで | (페이 앱) 바코드로요 | 스마트폰 결제 |
| 年齢確認（ねんれいかくにん）お願（ねが）いします (듣) | 연령 확인 부탁드립니다 | 술·담배 구매 시 화면 터치 요청 |
| 以上（いじょう）でよろしいですか (듣) | 이게 다인가요? | 주문·구매 마무리 확인 |
| はい、以上（いじょう）です | 네, 그게 다예요 | 마무리 대답 |
| レシートはご利用（りよう）ですか (듣) | 영수증 드릴까요? | 필요 없으면 大丈夫です |

💡 **한 줄 해설**
- **以上です** — 주문이나 볼일을 "여기까지요"라고 닫는 말. 식당 주문 끝에도, 편의점에서도, 발표 끝에도 씁니다. 점원의 以上でよろしいですか에는 はい、以上です로 대답하면 리듬이 딱 맞습니다.

## 3. 물건 찾기와 서비스 이용

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| これ、どこにありますか | 이거 어디에 있어요? | 사진을 보여 주며 물으면 최강 |
| ○○はありますか | ○○ 있나요? | 상품 유무 확인 |
| 売（う）り切（き）れです (듣) | 품절이에요 | 아쉽지만 포기 |
| 温（あたた）かい飲（の）み物（もの）はどこですか | 따뜻한 음료는 어디 있어요? | 겨울 편의점의 온장고 찾기 |
| 一番（いちばん）人気（にんき）なのはどれですか | 제일 잘 나가는 게 어느 거예요? | 신상 과자 고를 때 |
| トイレ借（か）りてもいいですか | 화장실 좀 써도 될까요? | 일본 편의점은 대부분 OK. 한마디 하고 쓰는 게 매너 |
| ATMはありますか | ATM 있나요? | 편의점 ATM은 해외 카드 출금 가능(세븐은행 등) |
| コピー機（き）はどこですか | 복사기 어디예요? | 티켓 발권·인쇄도 여기서 |
| チケットの発券（はっけん）はここでできますか | 티켓 발권 여기서 되나요? | 예매 번호 들고 갔을 때 |
| 宅配便（たくはいびん）を送（おく）りたいんですが | 택배를 보내고 싶은데요 | 편의점 택배 접수 |
| 賞味期限（しょうみきげん）はいつまでですか | 유통기한 언제까지예요? | 할인 상품 확인 |

## 4. 마트에서 추가로

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| 試食（ししょく）できますか | 시식해 봐도 되나요? | 시식 코너에서 |
| 半額（はんがく） | 반값 | 저녁 마트의 반값 스티커. 여행자의 친구 |
| お一人様（ひとりさま）一点（いってん）まで (듣/표지) | 1인 1개 한정 | 특가 상품 안내문 |
| レジ袋（ぶくろ）は有料（ゆうりょう）です (듣) | 비닐봉투는 유료입니다 | 계산대 안내 |
| 少々（しょうしょう）お待（ま）ちください (듣) | 잠시만 기다려 주세요 | 점원이 자리를 비울 때 |
| お待（ま）たせしました (듣) | 오래 기다리셨습니다 | 돌아온 점원의 첫마디 |
| 温（あたた）かいものと冷（つめ）たいもの、袋（ふくろ）を分（わ）けますか (듣) | 따뜻한 것과 차가운 것, 봉투 나눠 드릴까요? | 도시락+아이스크림 조합일 때 |
| 分（わ）けてください | 나눠 주세요 | 위 질문에 대한 답 |

💡 **한 줄 해설**
- **少々お待ちください / お待たせしました** — 이 한 쌍은 일본 접객의 기본 리듬입니다. 어디서든 이 말이 들리면 "기다리라는 거구나 / 이제 됐구나"로 이해하면 됩니다. 내가 쓸 일은 적지만 하루에도 몇 번씩 듣게 됩니다.

## 미니 대화 — 편의점에서 도시락 사기

**점원**: いらっしゃいませ。お弁当（べんとう）、温（あたた）めますか。
→ 어서 오세요. 도시락 데워 드릴까요?

**나**: お願（ねが）いします。
→ 네, 부탁해요.

**점원**: お箸（はし）はおつけしますか。
→ 젓가락 넣어 드릴까요?

**나**: はい、一（ひと）つください。
→ 네, 하나 주세요.

**점원**: 袋（ふくろ）はご利用（りよう）ですか。
→ 봉투 필요하세요?

**나**: 大丈夫（だいじょうぶ）です。あ、このお茶（ちゃ）も一緒（いっしょ）に。
→ 괜찮아요. 아, 이 차도 같이요.

**점원**: 以上（いじょう）でよろしいですか。六百二十円（ろっぴゃくにじゅうえん）です。
→ 이게 다인가요? 620엔입니다.

**나**: 交通系（こうつうけい）で。
→ 교통카드로 할게요.

**점원**: お待（ま）たせしました。ありがとうございました。
→ 오래 기다리셨습니다. 감사합니다.

**여기서 굴러간 표현**: 温めますか→お願いします / お箸おつけしますか→一つください / 袋はご利用ですか→大丈夫です / 以上でよろしいですか / 交通系で
