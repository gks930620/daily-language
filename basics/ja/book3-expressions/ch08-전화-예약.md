# 제8장 전화·예약

> **이 챕터에서 살아남기**: 얼굴이 안 보이는 전화는 초급자에게 최종 보스지만, 예약 전화의 진행 순서는 정해져 있습니다 — 용건 → 날짜·시간 → 인원 → 이름·연락처 → 확인. 이 순서대로 표현을 배치했고, "못 알아들었을 때 다시 묻는 기술"도 함께 담았습니다. 요즘은 식당 예약 앱이 많지만, 변경·취소는 여전히 전화가 빠릅니다.

---

## 1. 전화 걸고 받기

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| もしもし | 여보세요 | 전화 전용 인사. 대면에선 절대 안 씀 |
| ○○と申（もう）しますが | ○○라고 하는데요 | 이름을 밝히며 용건 시작 |
| 夜分（やぶん）にすみません | 밤늦게 죄송합니다 | 밤에 전화할 때의 매너 멘트 |
| 今（いま）、お時間（じかん）大丈夫（だいじょうぶ）ですか | 지금 통화 괜찮으세요? | 지인에게 걸 때 |
| 間違（まちが）い電話（でんわ）です | 잘못 거셨어요 | 모르는 전화가 왔을 때 |
| 後（あと）でかけ直（なお）します | 나중에 다시 걸게요 | 상대가 바쁠 때 |
| 電波（でんぱ）が悪（わる）いです | 신호가 안 좋아요 | 목소리가 끊길 때 |
| 聞（き）こえますか | 들리세요? | 통화 상태 확인 |
| では、失礼（しつれい）します | 그럼 끊겠습니다 | 전화 끊을 때의 표준 마무리 |

💡 **한 줄 해설**
- **もしもし** — 어원은 申（もう）します를 겹친 말. 전화에서만 쓰는 "여보세요"이고, 길에서 사람을 부를 때 쓰면 이상합니다(그건 すみません). 비즈니스 전화에서는 もしもし 대신 회사명을 대는 것이 정식이지만, 개인 통화는 もしもし면 충분.
- **では、失礼します (전화 끊기)** — "실례하겠습니다"가 전화에서는 "이만 끊겠습니다"가 됩니다. 뚝 끊지 말고 이 한마디 후에 상대가 끊기를 반 박자 기다리는 게 일본식 매너.

## 2. 예약하기 — 진행 순서대로

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| 予約（よやく）をしたいんですが | 예약을 하고 싶은데요 | 용건 첫마디. 이 문장이 예약 전화의 시동 |
| 今晩（こんばん）七時（しちじ）、二人（ふたり）でお願（ねが）いします | 오늘 저녁 7시, 두 명으로 부탁해요 | 날짜·시간·인원을 한 번에 |
| 明日（あした）の夜（よる）は空（あ）いていますか | 내일 저녁은 자리 있나요? | 빈자리 확인 |
| 満席（まんせき）です (듣) | 만석입니다 | 아쉬운 대답 |
| 何時（なんじ）なら空（あ）いていますか | 몇 시면 비어 있나요? | 대안 시간 묻기 |
| その時間（じかん）でお願（ねが）いします | 그 시간으로 할게요 | 제안 수락 |
| お名前（なまえ）をお願（ねが）いします (듣) | 성함 부탁드립니다 | 이름 등록 단계 |
| キムです。K・I・Mです | 김입니다. K, I, M이요 | 외국 이름은 철자를 불러 주면 확실 |
| お電話番号（でんわばんごう）をお願（ねが）いします (듣) | 전화번호 부탁드립니다 | 숫자를 하나씩 또박또박 |
| 当日（とうじつ）でも大丈夫（だいじょうぶ）ですか | 당일 예약도 되나요? | 급할 때 |
| 何時（なんじ）まで営業（えいぎょう）していますか | 몇 시까지 영업해요? | 영업시간 확인 |
| 定休日（ていきゅうび）はいつですか | 정기 휴일이 언제예요? | 허탕 방지 |

💡 **한 줄 해설**
- **予約をしたいんですが** — 8장의 주인공. 식당·미용실·병원·호텔 어디든 예약 전화는 이 한 문장으로 시작합니다. 그다음은 상대가 いつですか(언제요?)・何名様ですか(몇 분요?)를 순서대로 물어 주니, 날짜·시간·인원 숫자만 준비하면 됩니다.

---

## 3. 변경·취소·확인

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| 予約（よやく）を変更（へんこう）したいんですが | 예약을 변경하고 싶은데요 | 날짜·시간 바꾸기 |
| 予約（よやく）をキャンセルしたいんですが | 예약을 취소하고 싶은데요 | 취소는 빠를수록 매너 |
| キャンセル料（りょう）はかかりますか | 취소 수수료 있나요? | 당일 취소 전 확인 |
| 七時（しちじ）を八時（はちじ）に変（か）えられますか | 7시를 8시로 바꿀 수 있나요? | 구체적 변경 요청 |
| 予約（よやく）の確認（かくにん）をしたいんですが | 예약 확인을 하고 싶은데요 | 예약이 잘 됐는지 재확인 |
| ○○の名前（なまえ）で予約（よやく）しています | ○○ 이름으로 예약돼 있어요 | 가게 도착 후 첫마디 |
| 人数（にんずう）が変（か）わりました | 인원이 바뀌었어요 | 2명→4명 등 |
| 少（すこ）し遅（おく）れます | 조금 늦을 것 같아요 | 예약 시간에 못 맞출 때의 매너 전화 |

---

## 4. 못 알아들었을 때 — 초급자의 생명줄

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| もう一度（いちど）お願（ねが）いします | 다시 한번 말씀해 주세요 | 만능 되묻기 |
| ゆっくり話（はな）していただけますか | 천천히 말씀해 주시겠어요? | 속도 조절 요청 |
| 日本語（にほんご）があまりできません | 일본어를 잘 못해요 | 미리 말하면 상대도 쉽게 말해 줌 |
| 英語（えいご）できる方（かた）はいますか | 영어 하시는 분 계세요? | 최후의 수단 |
| ○○ということですか | ○○라는 말씀이세요? | 알아들은 내용 재확인 |
| はい、わかりました | 네, 알겠습니다 | 확인 완료 |
| 少々（しょうしょう）お待（ま）ちください (듣) | 잠시만 기다려 주세요 | 담당자 바꿔 줄 때 등 |

💡 **한 줄 해설**
- **もう一度お願いします** — 전화에서 못 알아들었을 때 침묵이 제일 위험합니다. 이 한마디를 부끄러워하지 말고 몇 번이든 쓰세요. 더 좋은 건 숫자를 되풀이해서 확인하는 것: 七時（しちじ）、二人（ふたり）ですね？ = "7시, 2명 맞죠?"

## 미니 대화 — 이자카야에 예약 전화

**가게**: はい、居酒屋（いざかや）やまだです。
→ 네, 이자카야 야마다입니다.

**나**: もしもし、予約（よやく）をしたいんですが。
→ 여보세요, 예약을 하고 싶은데요.

**가게**: ありがとうございます。いつがよろしいですか。
→ 감사합니다. 언제가 좋으세요?

**나**: 明日（あした）の七時（しちじ）、二人（ふたり）でお願（ねが）いします。
→ 내일 7시, 두 명으로 부탁해요.

**가게**: 申（もう）し訳（わけ）ありません、七時（しちじ）は満席（まんせき）でして……
→ 죄송합니다, 7시는 만석이라서요……

**나**: そうですか。何時（なんじ）なら空（あ）いていますか。
→ 그래요? 몇 시면 자리가 있나요?

**가게**: 八時（はちじ）なら大丈夫（だいじょうぶ）です。
→ 8시라면 괜찮습니다.

**나**: じゃ、その時間（じかん）でお願（ねが）いします。キムと申（もう）します。
→ 그럼 그 시간으로 할게요. 김이라고 합니다.

**가게**: キム様（さま）ですね。明日（あした）八時（はちじ）、二名様（にめいさま）でお待（ま）ちしております。
→ 김 님이시죠. 내일 8시, 두 분으로 기다리고 있겠습니다.

**나**: はい、よろしくお願（ねが）いします。では、失礼（しつれい）します。
→ 네, 잘 부탁드려요. 그럼 끊겠습니다.

**여기서 굴러간 표현**: もしもし / 予約をしたいんですが / 〜時、二人でお願いします / 何時なら空いていますか / その時間でお願いします / 〜と申します / では、失礼します
