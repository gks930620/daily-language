# 제11장 여행·호텔·긴급상황

> **이 챕터에서 살아남기**: 호텔 체크인부터 물건 분실, 병원까지 — 여행에서 "이 말을 몰라서 진짜 곤란해지는" 상황을 모았습니다. 긴급 표현은 쓸 일이 없기를 바라며 외우는 보험입니다. 특히 분실 관련 표현은 일본 여행에서 실제 사용률이 높습니다(일본은 유실물이 정말 잘 돌아오는 나라이니 포기하지 말 것).

---

## 1. 호텔 체크인·체크아웃

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| チェックインお願（ねが）いします | 체크인 부탁해요 | 프런트 첫마디 |
| 予約（よやく）したキムです | 예약한 김입니다 | 이름 대기. 여권을 함께 내밀면 순조 |
| チェックアウトは何時（なんじ）ですか | 체크아웃 몇 시예요? | 보통 10~11시 |
| 荷物（にもつ）を預（あず）かってもらえますか | 짐 좀 맡아 주실 수 있나요? | 체크인 전·체크아웃 후 짐 보관 |
| Wi-Fi（ワイファイ）のパスワードは何（なん）ですか | 와이파이 비밀번호가 뭐예요? | |
| 朝食（ちょうしょく）は何時（なんじ）からですか | 조식은 몇 시부터예요? | 장소도 함께: どこですか |
| 一泊（いっぱく）いくらですか | 1박에 얼마예요? | 二泊（にはく）= 2박 |
| もう一泊（いっぱく）できますか | 하루 더 묵을 수 있나요? | 연박 요청 |
| タクシーを呼（よ）んでもらえますか | 택시 좀 불러 주시겠어요? | 프런트 활용 |
| おすすめの観光地（かんこうち）はありますか | 추천 관광지 있나요? | 현지인 정보가 최고 |

## 2. 방에서 문제 발생

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| 部屋（へや）に鍵（かぎ）を忘（わす）れました | 방에 열쇠를 두고 나왔어요 | 오토락 호텔의 단골 사고 |
| お湯（ゆ）が出（で）ません | 뜨거운 물이 안 나와요 | 水（みず）= 찬물과 구분 |
| エアコンが動（うご）きません | 에어컨이 작동을 안 해요 | 動かない = 기계가 안 움직임 |
| 電気（でんき）がつきません | 불이 안 켜져요 | 카드키 삽입식인지 먼저 확인 |
| タオルをもう一枚（いちまい）お願（ねが）いします | 수건 한 장 더 주세요 | 어메니티 추가 요청 |
| 部屋（へや）を変（か）えてもらえますか | 방을 바꿔 주실 수 있나요? | 소음·냄새 등 문제가 있을 때 |
| 隣（となり）の部屋（へや）がうるさいです | 옆방이 시끄러워요 | 방 변경 요청의 사유 |
| 掃除（そうじ）はいりません | 청소는 안 해 주셔도 돼요 | 연박 중 프라이버시 |

💡 **한 줄 해설**
- **〜が出ません／〜が動きません／〜がつきません** — 고장 신고 3형제. 물·전파처럼 나오는 것은 出ない, 기계는 動かない, 불·전원은 つかない. 프런트에 이 세 패턴만 말해도 뭐가 문제인지 다 통합니다.

## 3. 분실 — 일본 여행 실전 최다 발생

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| 財布（さいふ）をなくしました | 지갑을 잃어버렸어요 | 분실 신고의 기본형 |
| パスポートをなくしました | 여권을 잃어버렸어요 | 영사관 연락이 필요한 비상사태 |
| 電車（でんしゃ）に忘（わす）れ物（もの）をしました | 전철에 물건을 두고 내렸어요 | 역무원에게. 유실물 센터로 연결됨 |
| スマホをタクシーに忘（わす）れました | 스마트폰을 택시에 두고 내렸어요 | 영수증이 있으면 추적 가능 |
| 交番（こうばん）はどこですか | 파출소가 어디예요? | 일본은 동네마다 파출소가 있음 |
| 遺失物（いしつぶつ）センターはどこですか | 유실물 센터 어디예요? | 큰 역에는 반드시 있음 |
| 見（み）つかったら連絡（れんらく）ください | 찾으면 연락 주세요 | 연락처를 남기며 |
| 盗（ぬす）まれました | 도둑맞았어요 | 분실이 아니라 도난일 때. 경찰 신고 사안 |
| カードを止（と）めたいです | 카드를 정지하고 싶어요 | 카드사 전화에서 |

💡 **한 줄 해설**
- **なくしました vs 忘れました** — なくす는 "어디서 잃었는지 모름"(분실), 忘れる는 "어디에 뒀는지 앎"(두고 옴). 전철에 두고 내렸으면 電車に忘れました라고 해야 역무원이 바로 그 열차의 유실물을 조회해 줍니다. 이 구분이 물건을 되찾는 속도를 좌우합니다.
- **交番** — 파출소. 길 잃음·분실·사고, 여행 중 모든 곤란의 1차 창구입니다. 지갑을 주운 사람도 여기로 가져오므로, 잃어버렸다면 근처 交番부터 들르세요.

## 4. 긴급·병원

| 표현 | 뜻 | 이럴 때 |
|---|---|---|
| 助（たす）けてください！ | 도와주세요! (구조 요청) | 진짜 위급할 때만. 일상 도움 요청은 すみません |
| 警察（けいさつ）を呼（よ）んでください | 경찰을 불러 주세요 | 사건·사고. 경찰 직통 110번 |
| 救急車（きゅうきゅうしゃ）を呼（よ）んでください | 구급차를 불러 주세요 | 응급 상황. 구급·화재 119번 |
| 病院（びょういん）に行（い）きたいです | 병원에 가야 해요 | 주변에 도움 청하기 |
| 具合（ぐあい）が悪（わる）いです | 몸이 안 좋아요 | 증상 설명의 시작 |
| ここが痛（いた）いです | 여기가 아파요 | 아픈 곳을 가리키며. 만능 진료 표현 |
| 熱（ねつ）があります | 열이 있어요 | |
| めまいがします | 어지러워요 | 吐（は）き気（け）がします = 토할 것 같아요 |
| 薬（くすり）アレルギーがあります | 약 알레르기가 있어요 | 진료·처방 전 필수 고지 |
| 薬局（やっきょく）はどこですか | 약국이 어디예요? | 드러그스토어=ドラッグストア도 통함 |
| 風邪薬（かぜぐすり）はありますか | 감기약 있나요? | 頭痛薬（ずつうやく）= 두통약 |
| 海外旅行保険（かいがいりょこうほけん）に入（はい）っています | 여행자 보험에 들어 있어요 | 병원 접수에서. 영수증 꼭 받기 |
| 韓国語（かんこくご）ができる人（ひと）はいますか | 한국어 할 수 있는 분 있나요? | 대형 병원·관광지엔 의외로 있음 |
| 大使館（たいしかん）に連絡（れんらく）したいです | 대사관에 연락하고 싶어요 | 여권 분실 등 최후의 카드 |

💡 **한 줄 해설**
- **助けてください는 아껴 두는 말** — 길을 물을 때 이 말을 쓰면 상대가 깜짝 놀랍니다. 일상적 도움은 すみません、ちょっといいですか, 생명·안전이 걸린 순간에만 助けて！. 한국어 "도와주세요"보다 훨씬 무거운 말입니다.
- **110과 119** — 일본은 경찰 110, 구급·화재 119로 한국과 구급 번호가 같습니다. 전화로 말할 자신이 없으면 주변 사람에게 위 표의 呼んでください 문장을 쓰는 것이 현실적입니다.

## 미니 대화 — 전철에 가방을 두고 내렸다

**나**: すみません、電車（でんしゃ）に忘（わす）れ物（もの）をしました。
→ 저기요, 전철에 물건을 두고 내렸어요.

**역무원**: 何（なに）を忘（わす）れましたか。
→ 무엇을 두고 내리셨나요?

**나**: 黒（くろ）いかばんです。中（なか）に財布（さいふ）とパスポートが入（はい）っています。
→ 검은색 가방이요. 안에 지갑이랑 여권이 들어 있어요.

**역무원**: 何時（なんじ）ごろの電車（でんしゃ）ですか。
→ 몇 시쯤 전철인가요?

**나**: 三十分（さんじゅっぷん）ぐらい前（まえ）です。渋谷（しぶや）行（ゆ）きでした。
→ 30분쯤 전이요. 시부야행이었어요.

**역무원**: 確認（かくにん）しますので、少々（しょうしょう）お待（ま）ちください。……ありました。次（つぎ）の駅（えき）で保管（ほかん）しています。
→ 확인해 볼 테니 잠시만 기다려 주세요. ……있네요. 다음 역에서 보관 중입니다.

**나**: 本当（ほんとう）ですか！ありがとうございます。助（たす）かりました！
→ 정말요?! 감사합니다. 살았어요!

**여기서 굴러간 표현**: 電車に忘れ物をしました / 〜が入っています / ○○行き / 少々お待ちください / 助かりました
