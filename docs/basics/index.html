대화 입문

  • 일본어 무작정 시작하기 — 1권: 대화로 시작하는 첫 두 달
  • 1주차 — 히라가나 완성 + 기본 인사
  • 2주차 — 가타카나 + 자기소개
  • 3주차 — 숫자·시간·날짜 + これ/それ/あれ
  • 4주차 — 실전 쇼핑과 주문: 편의점·옷가게·식당·카페
  • 5주차 — 길 묻기·위치·교통: 〜は どこですか / 〜が あります・います
  • 6주차 — 스몰토크의 기술: 날씨·계절·취미 + 형용사
  • 7주차 — 동사 ます형: 하루 일과, 약속, 전화
  • 8주차 — 실전 종합: 편의점 심화·병원·감정·여행 계획, 그리고 총정리
