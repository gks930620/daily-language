제4장 쇼핑·계산

이 챕터에서 살아남기: 일본 가게 점원은 말이 빠르고 경어가 화려하지만, 손님인 내가 할 말은 의외로 몇 가지 안 됩니다. "구경만 할게요"부터 "이걸로 할게요", 그리고 계산대의 단골 질문 대응까지 — 쇼핑 한 사이클을 순서대로 배치했습니다.

1. 가게에 들어가서 — 구경과 문의

표현뜻이럴 때
いらっしゃいませ어서 오세요듣는 말. 대답할 필요 없음. 가볍게 목례면 충분
見（み）ているだけです그냥 구경 중이에요점원이 다가와서 뭔가 물을 때의 방패
ちょっと見（み）せてください좀 보여 주세요진열장 안 물건을 볼 때
○○を探（さが）しているんですが○○을 찾고 있는데요용건 꺼내기. 뒤를 흐리면 점원이 알아서 이어받음
これはありますか이거 있나요?사진·메모를 보여 주면서
色違（いろちが）いはありますか다른 색 있나요?같은 디자인의 다른 색
もう少（すこ）し大（おお）きいのはありますか좀 더 큰 거 있나요?사이즈 업. 작은 건 小（ちい）さいの
試着（しちゃく）してもいいですか입어 봐도 되나요?옷 가게에서. 신발은 履（は）いてみてもいいですか
新（あたら）しいのはありますか새 상품(진열 안 된 것) 있나요?진열품 말고 새것을 원할 때
在庫（ざいこ）はありますか재고 있나요?품절처럼 보일 때
セール中（ちゅう）ですか세일 중인가요?가격표가 애매할 때

💡 한 줄 해설

  • 見ているだけです — 일본 점원은 손님에게 잘 달라붙는 편인데, 이 한마디면 정중하게 물러나 줍니다. 직역 "보고 있을 뿐입니다"지만 실제 뉘앙스는 "천천히 볼게요, 신경 안 쓰셔도 돼요".
  • 〜んですが (뒤 흐리기) — 探しているんですが…처럼 문장을 끝까지 말하지 않고 흐리는 것이 일본어에선 무례가 아니라 정중함입니다. "…인데요 (도와주실래요?)"의 후반부를 상대에게 맡기는 화법.

2. 결정하기 — 살까 말까

표현뜻이럴 때
これにします이걸로 할게요구매 결정의 한마디
これください이거 주세요결정 즉시 구매. にします보다 직접적
ちょっと考（かんが）えます좀 생각해 볼게요부드럽게 자리를 뜨는 말
また来（き）ます다음에 다시 올게요안 사고 나갈 때. (다시 안 와도 됩니다)
どちらが人気（にんき）ですか어느 쪽이 인기예요?둘 중 고민될 때 점원 찬스
おすすめはどれですか추천은 어느 거예요?아예 맡겨 버리기
プレゼント用（よう）です선물용이에요포장 여부를 묻기 전에 먼저
包（つつ）んでもらえますか포장해 주실 수 있나요?선물 포장 요청

💡 한 줄 해설

  • これにします — 「〜にする」는 "~로 정하다"라는 선택의 표현. 식당 주문에서도 쇼핑에서도 "(고민 끝) 이걸로 결정!"의 느낌입니다. これください가 "이거 주세요"라면 これにします는 "이걸로 할게요".

3. 계산대에서

표현뜻이럴 때
お会計（かいけい）お願（ねが）いします계산해 주세요계산대로 물건을 가져가며
カードで払（はら）えますか카드 되나요?소도시·개인 가게는 현금 전용이 아직 있음
カードでお願（ねが）いします카드로 할게요결제 수단 말하기
現金（げんきん）で払（はら）います현금으로 낼게요
一括（いっかつ）でお願（ねが）いします일시불로 해 주세요카드 결제 시 반드시 듣는 질문에 대한 답
袋（ふくろ）お願（ねが）いします봉투 주세요일본은 봉투가 유료(보통 3~5엔)
袋（ふくろ）はいりません봉투는 필요 없어요그대로 들고 갈 때
レシートください영수증 주세요안 주는 가게도 있으니 필요하면 요청
免税（めんぜい）できますか면세 되나요?여권 지참. 보통 5,000엔 이상 구매 시
ポイントカードはありません포인트 카드 없어요계산대 단골 질문에 대한 답

💡 한 줄 해설

  • 一括ですか、分割ですか — 일본에서 카드를 내면 거의 반드시 "일시불이요, 할부요?"라고 묻습니다. 해외 카드는 어차피 일시불만 되니 一括（いっかつ）で 한마디를 준비해 두면 계산대에서 얼어붙지 않습니다.

미니 대화 — 옷 가게에서 셔츠 하나 사기

점원: いらっしゃいませ。何（なに）かお探（さが）しですか。 → 어서 오세요. 찾으시는 거 있으세요?

나: シャツを探（さが）しているんですが、これ、試着（しちゃく）してもいいですか。 → 셔츠를 찾고 있는데요, 이거 입어 봐도 되나요?

점원: はい、どうぞ。試着室（しちゃくしつ）はこちらです。 → 네, 그럼요. 탈의실은 이쪽입니다.

나: （입어 본 후）ちょっと小（ちい）さいですね。もう少（すこ）し大（おお）きいのはありますか。 → 좀 작네요. 조금 더 큰 거 있나요?

점원: はい、Lサイズをお持（も）ちしますね。……こちらです。 → 네, L사이즈 가져다드릴게요. ……여기 있습니다.

나: あ、ちょうどいいです。これにします。カードで払（はら）えますか。 → 아, 딱 좋네요. 이걸로 할게요. 카드 되나요?

점원: はい。お支払（しはら）いは一括（いっかつ）でよろしいですか。 → 네. 결제는 일시불로 괜찮으세요?

나: 一括（いっかつ）でお願（ねが）いします。袋（ふくろ）はいりません。 → 일시불로 해 주세요. 봉투는 필요 없어요.

여기서 굴러간 표현: 〜を探しているんですが / 試着してもいいですか / もう少し大きいのはありますか / これにします / カードで払えますか / 一括でお願いします / 袋はいりません
