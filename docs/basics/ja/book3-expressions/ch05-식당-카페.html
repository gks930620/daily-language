제5장 식당·카페

이 챕터에서 살아남기: 들어가서 인원 말하기 → 주문 → 식사 중 요청 → 계산. 이 네 장면만 통과하면 일본의 어떤 식당에서도 굶지 않습니다. 카페 주문에서 반드시 듣는 질문(매장/포장, 핫/아이스)도 함께 담았습니다.

1. 입장 — 자리 안내받기

표현뜻이럴 때
何名様（なんめいさま）ですか몇 분이세요?듣는 말. 입구에서 반드시 물어봄
二人（ふたり）です두 명이요한 명=一人（ひとり）, 세 명=三人（さんにん）
予約（よやく）していません예약 안 했는데요예약 여부를 물었을 때
どのくらい待（ま）ちますか얼마나 기다려요?대기줄이 있을 때
席（せき）は空（あ）いていますか자리 있나요?만석처럼 보일 때
カウンターでもいいです카운터석도 괜찮아요혼밥·빨리 앉고 싶을 때
禁煙席（きんえんせき）でお願（ねが）いします금연석으로 부탁해요이자카야 등 흡연 가능 가게에서

2. 주문하기

표현뜻이럴 때
すみません！저기요!점원 부르기. 일본에선 크게 불러도 실례 아님
メニューください메뉴판 주세요
おすすめは何（なん）ですか추천 메뉴가 뭐예요?뭘 시킬지 모르겠을 때
まだ決（き）まっていません아직 못 골랐어요점원이 너무 빨리 왔을 때
注文（ちゅうもん）お願（ねが）いします주문할게요준비됐을 때 점원 호출
これとこれをください이거랑 이거 주세요메뉴판 손가락 주문. 최강의 생존 기술
とりあえずビールで일단 맥주부터요이자카야의 국민 첫 주문
私（わたし）も同（おな）じもので저도 같은 걸로요일행 따라가기
大盛（おおも）りでお願（ねが）いします곱빼기로 주세요밥·면 양 추가. 보통=並（な）み
わさび抜（ぬ）きでお願（ねが）いします와사비 빼 주세요○○抜き = ○○ 빼고
アレルギーがあります알레르기가 있어요卵（たまご)=계란, えび=새우 등을 앞에 붙여서
これはどんな料理（りょうり）ですか이건 어떤 음식이에요?메뉴 이름만 봐선 모를 때

💡 한 줄 해설

  • すみません! (호출) — 한국에서 "저기요~"를 크게 외치는 것과 똑같이, 일본 식당에서도 손을 들고 すみません!이라고 부르는 게 표준입니다. 조용히 눈빛만 보내면 평생 주문 못 합니다. 테이블 호출벨이 있으면 그걸로.
  • とりあえず — "일단, 우선"이라는 뜻으로, 메뉴 고민은 나중에 하고 마실 것부터 시키는 이자카야 문화의 핵심 단어. とりあえずビールで(일단 맥주로)는 거의 관용구입니다.

3. 식사 중 — 요청과 리액션

표현뜻이럴 때
いただきます잘 먹겠습니다식사 시작. 혼잣말이어도 습관처럼
おいしいです맛있어요최고 반응은 おいしい！한 마디
お水（みず）ください물 좀 주세요물은 기본 무료 (お냉수=お冷（ひや）)
お代（か）わりください리필해 주세요 / 한 그릇 더요밥·음료 리필
取（と）り皿（ざら）ください앞접시 주세요나눠 먹을 때
お手洗（てあら）いはどこですか화장실 어디예요?トイレ보다 부드러운 말
これ、頼（たの）んでいません이거 주문 안 했는데요잘못 나온 음식
注文（ちゅうもん）したものがまだ来（き）ていません주문한 게 아직 안 나왔어요오래 기다렸을 때
持（も）ち帰（かえ）りできますか포장 되나요? / 남은 거 싸 갈 수 있나요?남은 음식·테이크아웃

💡 한 줄 해설

  • いただきます / ごちそうさまでした — "받겠습니다/성찬이었습니다"라는 직역은 잊으세요. 실제로는 "잘 먹겠습니다/잘 먹었습니다"에 해당하는 식사 인사이고, 계산하고 나가면서 점원에게 ごちそうさまでした라고 하면 "맛있게 잘 먹고 갑니다"라는 최고의 인사가 됩니다.

4. 계산과 카페 주문

표현뜻이럴 때
お会計（かいけい）お願（ねが）いします계산할게요자리에서 또는 카운터에서
別々（べつべつ）でお願（ねが）いします따로따로 계산해 주세요일본은 더치페이가 기본이라 자연스러운 요청
一緒（いっしょ）でお願（ねが）いします같이 계산할게요한 사람이 몰아서 낼 때
ごちそうさまでした잘 먹었습니다나가면서 점원에게 한마디
店内（てんない）でお願（ねが）いします매장에서 먹고 갈게요카페·패스트푸드 단골 질문에 대한 답
持（も）ち帰（かえ）りで포장이요 / 가지고 갈게요테이크아웃
ホットで／アイスで따뜻한 걸로 / 차가운 걸로커피 주문 필수 선택지
サイズはどうしますか사이즈는 어떻게 하시겠어요?듣는 말. 톨=トール, 작은 것=ショート
ラストオーダーは何時（なんじ）ですか라스트 오더 몇 시예요?늦은 시간 입장 시

💡 한 줄 해설

  • 別々でお願いします — 일본은 각자 계산(割（わ）り勘（かん))이 기본값이라 "따로 계산해 주세요"가 전혀 실례가 아닙니다. 계산대에서 인원수만큼 나눠 찍어 주는 가게도 많습니다. 얻어먹었다면 다음에 ごちそうさまでした、今度（こんど）は私（わたし）が = "잘 먹었어요, 다음엔 제가".

미니 대화 — 라멘집에서 혼밥

점원: いらっしゃいませ！何名様（なんめいさま）ですか。 → 어서 오세요! 몇 분이세요?

나: 一人（ひとり）です。 → 한 명이요.

점원: カウンターへどうぞ。ご注文（ちゅうもん）が決（き）まったらお呼（よ）びください。 → 카운터석으로 앉으세요. 주문 정해지시면 불러 주세요.

나: すみません！おすすめは何（なん）ですか。 → 저기요! 추천 메뉴가 뭐예요?

점원: みそラーメンが人気（にんき）ですよ。 → 미소라멘이 인기예요.

나: じゃ、それをください。あ、大盛（おおも）りでお願（ねが）いします。 → 그럼 그걸로 주세요. 아, 곱빼기로 부탁해요.

나: （다 먹고）おいしかったです。お会計（かいけい）お願（ねが）いします。 → 맛있었어요. 계산할게요.

점원: ありがとうございます。九百円（きゅうひゃくえん）です。 → 감사합니다. 900엔입니다.

나: ごちそうさまでした。 → 잘 먹었습니다.

여기서 굴러간 표현: 一人です / すみません(호출) / おすすめは何ですか / 〜をください / 大盛りでお願いします / お会計お願いします / ごちそうさまでした
