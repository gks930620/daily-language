제12장 자주 쓰는 축약·구어 — 입문자용 맛보기

이 챕터에서 살아남기: 교과서 일본어로 공부하다 드라마·애니를 켜면 "이게 같은 언어인가" 싶어집니다. 범인은 축약과 구어체. 이 챕터는 그 간극을 메우는 맛보기입니다. 목표는 알아듣기 — 직접 쓰는 건 친구 사이에서만, 점원·손윗사람에게는 지금까지 배운 정중체를 쓰세요. 이 장의 표현은 별도 표시가 없어도 대부분 반말 상황용입니다.

1. 문장을 잇고 여는 말들

표현뜻이럴 때
じゃ／じゃあ그럼では의 구어. 화제 전환·작별 양쪽
じゃ、また그럼 또 봐작별 인사의 국민 축약형
っていうか그게 아니라 / 아니 근데화제를 비틀거나 정정할 때. 젊은 층은 てか
なんか뭔가 / 왠지문장 곳곳에 넣는 필러. 한국어 "뭔가"와 사용감 동일
とりあえず일단뭐부터 할지 정할 때
やっぱり／やっぱ역시예상대로일 때. やっぱ는 더 캐주얼
ちなみに참고로 / 그나저나정보를 하나 얹을 때
例（たと）えば예를 들면설명 보조
一応（いちおう）일단은 / 명색이"완벽하진 않지만 어쨌든" 뉘앙스
まあ뭐, 그럭저럭얼버무림·양보의 쿠션
別（べつ）に딱히 / 별로뒤에 부정이 옴. 퉁명스럽게 들릴 수 있음

💡 한 줄 해설

  • っていうか — というか(라고 할까)의 축약. 상대 말을 살짝 정정하거나("그게 아니라") 아예 화제를 갈아탈 때("아니 근데") 씁니다. 대화 첫머리에 てか로 시작하는 것이 요즘 구어의 전형.
  • 別に — "딱히 (상관없어/아무것도 아냐)". 짧게 別に만 던지면 한국어 "별로"처럼 시큰둥한 인상을 주므로, 정중한 자리에서는 特（とく）に(특별히)를 쓰는 것이 안전합니다.

2. 동사 축약 — 알아듣기의 핵심

표현뜻이럴 때
〜てる~하고 있어〜ている의 い 탈락. 食（た）べてる = 먹고 있어
〜ちゃう／〜ちゃった~해 버려 / ~해 버렸어〜てしまう의 축약. 아차·완료의 뉘앙스
食（た）べちゃった다 먹어 버렸어"남겨 두려 했는데!" 하는 아차 느낌
〜なきゃ~해야 해〜なければ(ならない)의 축약. 行（い）かなきゃ = 가야 해
〜んだ／〜んです~거든 / ~라서요이유·사정 설명의 ん. 구어의 최중요 장치
〜って~래 / ~라던데인용 축약. 明日（あした）休（やす）みだって = 내일 쉰대
わかんない몰라わからない의 축약. 정중형은 わかりません
〜とく~해 둘게〜ておく의 축약. 予約（よやく）しとく = 예약해 둘게
〜てもいい？~해도 돼?〜てもいいですか의 반말. 끝을 올려 발음
かも~일지도かもしれない의 꼬리 자르기. 行（い）くかも = 갈지도

💡 한 줄 해설

  • 〜ちゃう — てしまう(해 버리다)가 ちゃう로 줄어든 것. "의도치 않게·아쉽게도·몽땅"의 감정이 실립니다. 遅（おく）れちゃった(늦어 버렸어), 全部（ぜんぶ）使（つか）っちゃった(다 써 버렸어). 노래 가사와 일상 회화의 최다 출연 축약.
  • 〜んです — 정중체에도 쓰는 유일한 장치라 입문자에게 가성비 최고. 문장 끝에 んです를 붙이면 "~라서요, ~거든요"라는 사정 설명이 됩니다. 頭が痛いんです = "머리가 아파서요(그래서 이러는 거예요)".

3. 감탄·강조의 구어

표현뜻이럴 때
マジ／マジで진짜 / 진심으로本当に의 구어. マジで？= 진심?
めっちゃ엄청관서 출신이지만 전국구가 된 강조어
超（ちょう）완전超おいしい = 완전 맛있어
ヤバい대박 / 큰일났다좋을 때도 나쁠 때도. 문맥이 전부
すごっ헐, 대박すごい의 꼬리 자르기 감탄
高（たか）っ！비싸!형용사 끝을 잘라 놀람 표현. 安（やす）っ！= 싸다!
うける웃겨 ㅋㅋ재밌는 얘기에 대한 리액션
さすが역시~ (대단해)기대에 부응한 상대를 치켜세울 때
だよね그치? / 내 말이ですよね의 반말
ってこと？그런 뜻이야?ということですか의 축약
〜みたいな~같은 (느낌)?인용·비유를 흐리는 젊은 층 말버릇
どういうこと？무슨 소리야? / 어떻게 된 거야?설명을 요구할 때
お疲（つか）れ！수고!お疲れさまです의 반말. 친구·동기끼리

💡 한 줄 해설

  • ヤバい — 원래 "위험하다"였지만 지금은 "대박 맛있다"부터 "큰일 났다"까지 전부 커버하는 만능어. このラーメン、ヤバい는 칭찬이고 電車（でんしゃ）来（こ）ない、ヤバい는 위기입니다. 표정과 문맥으로 판별하세요.
  • さすが — "역시 너야!"라고 상대를 세워 주는 말. さすがキムさん！이라고 하면 상대 기분이 확실히 좋아집니다. 반어로 쓰면 비꼼이 되니 표정은 밝게.

4. 정중체로도 쓸 수 있는 부드러운 구어

표현뜻이럴 때
〜って言（い）ってました~라고 했어요전달의 정중 구어
そうなんですね그렇군요なるほど보다 안전한 정중 맞장구
ですかね~려나요 / ~일까요단정을 피하는 부드러운 의문
〜な感（かん）じです~한 느낌이에요두루뭉술하게 설명할 때
ぼちぼちです그럭저럭이요"요즘 어때요?"에 대한 겸손한 답

미니 대화 — 친구와 라멘집 앞에서 〔반말 회화〕

리쿠: あ、この店（みせ）、めっちゃ並（なら）んでる。 → 아, 이 가게 엄청 줄 서 있네.

나: マジで？……ほんとだ。ヤバいね。 → 진짜? ……진짜네. 장난 아니다.

리쿠: てか、もう二時（にじ）だよ。お腹（なか)すいたし、別（べつ）の店（みせ）にしない？ → 아니 근데 벌써 2시야. 배도 고픈데 다른 가게로 하지 않을래?

나: そうだね。あ、駅前（えきまえ）にいい店（みせ）あるって聞（き）いたかも。 → 그러네. 아, 역 앞에 괜찮은 가게 있다고 들었던 것 같아.

리쿠: じゃ、そこ行（い）こう。予約（よやく）とかいらないよね？ → 그럼 거기 가자. 예약 같은 건 필요 없지?

나: たぶん。でも一応（いちおう）、電話（でんわ）しとく？ → 아마도. 근데 일단 전화해 둘까?

리쿠: お、さすが。頼（たの）んだ！ → 오, 역시. 부탁해!

나: はいはい……あ、電話（でんわ）つながった。やっぱ人気店（にんきてん）かも。 → 네네…… 아, 전화 연결됐다. 역시 인기 가게인가 봐.

여기서 굴러간 표현: めっちゃ / マジで / ヤバい / てか(っていうか) / 〜って聞いた / かも / 一応 / 〜しとく / さすが / やっぱ
