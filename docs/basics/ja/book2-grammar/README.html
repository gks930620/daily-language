일본어 기초 교재 2권 — 기본 문법서

일본어 입문~초급(N5~N4 진입) 수준의 기본 문법책입니다. 책1(대화 입문)을 병행하거나 마친 한국어 화자를 대상으로 하며, "규칙 나열"이 아니라 한국어 화자가 실제로 헷갈리는 지점을 중심으로 설명합니다.

이 책의 동사 분류 방식 — 왜 "그룹1/2/3"이 아니라 5단·1단·불규칙인가

시중 교재 다수는 동사를 "그룹1 / 그룹2 / 그룹3"으로 나눕니다. 이 책은 일본 학교문법의 원래 이름인 5단동사 · 1단동사 · 불규칙동사를 씁니다. 이름 자체가 활용 원리를 담고 있어서, 외울 것이 오히려 줄기 때문입니다.

이 책의 이름흔한 다른 이름정체
5단동사그룹1, u동사어미가 오십음도의 5개 단(あ・い・う・え・お단)을 오가며 활용한다
1단동사그룹2, ru동사어미가 1개 단에 고정되고, る만 떼었다 붙였다 한다
불규칙동사그룹3する(하다)・くる(오다) 딱 2개. 그냥 외운다

예를 들어 5단동사 書(か)く는 이렇게 움직입니다.

단(段)형태뒤에 붙는 것결과
あ단書か＋ない書かない (쓰지 않다)
い단書き＋ます書きます (씁니다)
う단書く(사전형)書く (쓰다)
え단書け＋ば / 명령書けば (쓰면) / 書け (써라)
お단書こ＋う書こう (쓰자/써야지)

"あ단+ない, い단+ます, え단+ば, お단+う" — 이 한 줄이 5단동사 활용의 전부입니다. "그룹1은 ます형이 i로 끝나고, ない형은 a로 바꾸고…"처럼 형태마다 따로 외우는 대신, 오십음도 한 장으로 모든 활용을 꿰는 것이 이 책의 방식입니다. (자세한 것은 Part 3)

사용법

  • 순서대로 읽되, Part 3이 중심입니다. Part 1~2는 Part 3을 위한 준비 운동에 가깝습니다.
  • 각 단원은 같은 형식입니다:
    1. 핵심 규칙 — 표로 정리
    2. 예문 — 일본어(후리가나 병기) + 한국어 해석
    3. 한국어 화자 팁 — 한국어와 같은 점 / 다른 점
    4. ⚠️ 자주 하는 실수 — 한국어 화자가 실제로 자주 틀리는 것
    5. 미니 확인문제 — 3문제. 정답은 접혀 있으니 먼저 풀고 펼치세요.
  • 후리가나는 漢字(かんじ) 방식으로 병기합니다. 예: 私(わたし)は学生(がくせい)です。
  • 활용표는 소리 내어 읽으며 외우세요. 특히 Part 3의 て형 음편표는 노래처럼 입에 붙여야 합니다.

표기 약속

  • ○ = 자연스러운 문장 / ✕ = 틀린 문장 / △ = 문법상 가능하지만 어색함
  • 「정중체」= です・ます체, 「보통체」= だ・사전형체(반말)
  • 예문 해석의 괄호는 직역·뉘앙스 보충입니다.
