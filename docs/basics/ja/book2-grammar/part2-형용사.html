Part 2 — 형용사

일본어 형용사는 い형용사와 な형용사 두 종류뿐이고, 각각 활용이 완전히 규칙적입니다 (예외는 いい 단 하나). 이 Part에서 구분법 → 활용 → 부사화 → 비교 표현 → 함정 단어 순으로 끝냅니다.

1과. い형용사와 な형용사 — 구분법

핵심 규칙

い형용사な형용사
사전형~い로 끝남: 高(たか)い, 寒(さむ)い제한 없음: 静(しず)か, 便利(べんり), きれい
명사 수식그대로: 高い山(やま)な를 붙임: 静かな町(まち)
서술高い(です) — だ 안 붙음静かだ / 静かです
정체순수 일본어가 다수한자어·외래어가 다수 (동사 아닌 "명사스러운" 말)

구분 요령: 사전형이 い로 끝나면 대부분 い형용사. 단, い로 끝나는데 な형용사인 함정 단어가 소수 있습니다(반드시 암기):

함정 단어뜻증거
きれい(だ)예쁘다, 깨끗하다きれいな花(はな) ○ / きれい花 ✕
嫌(きら)い(だ)싫다嫌いな食(た)べ物(もの)
有名(ゆうめい)(だ)유명하다有名な人(ひと)
丁寧(ていねい)(だ)정중하다丁寧な説明(せつめい)

요령: きれい(綺麗)・きらい(嫌い)・ゆうめい(有名)・ていねい(丁寧)는 한자로 쓰면 い가 어미가 아니라 단어 몸통의 일부입니다. い형용사는 어미 い가 반드시 히라가나로 노출됩니다(高い, 安い).

예문

  • この店(みせ)は安(やす)いです。

이 가게는 쌉니다. (い형용사 + です)

  • 駅(えき)の近(ちか)くに大(おお)きいスーパーがあります。

역 근처에 큰 슈퍼가 있습니다. (い형용사는 그대로 명사 수식)

  • ここは静(しず)かな公園(こうえん)ですね。

여기는 조용한 공원이네요. (な형용사 + な + 명사)

  • 彼女(かのじょ)は有名(ゆうめい)な歌手(かしゅ)です。

그녀는 유명한 가수입니다. (い로 끝나 보여도 な형용사)

한국어 화자 팁

  • 한국어 형용사는 활용이 한 종류지만("예쁘다-예쁜-예뻤다"), 일본어는 두 갈래입니다. 명사를 수식해 보는 것이 가장 확실한 판별법입니다: な가 필요하면 な형용사.
  • な형용사는 사실상 "명사+だ"처럼 움직입니다(활용이 명사문과 동일). "な형용사 = です/だ 문법을 재활용"이라고 생각하면 외울 게 절반이 됩니다.

⚠️ 자주 하는 실수

  1. ✕ きれい海(うみ) / ✕ きれいの海 — ○ きれいな海。 きれい는 な형용사이므로 な로 수식. (の를 넣는 것은 중국어권 학습자, な를 빼는 것은 한국어권 학습자의 단골 실수)
  2. ✕ 高いな山 / ✕ 高いだ — い형용사에는 な도 だ도 붙지 않습니다. ○ 高い山 / 高い。 보통체 서술은 사전형 그대로 끝.

미니 확인문제

Q1. 다음 중 な형용사를 모두 고르세요: 新(あたら)しい / 元気(げんき) / 嫌(きら)い / 忙(いそが)しい / 便利(べんり)

Q2. "친절한 사람" — 親切(しんせつ)를 써서.

Q3. "이 방은 깨끗합니다" — きれい를 써서 정중체로.

정답 보기

A1. 元気, 嫌い, 便利. (新しい・忙しい는 い형용사. 嫌い는 い로 끝나는 함정 な형용사)

A2. 親切(しんせつ)な人(ひと)。

A3. この部屋(へや)はきれいです。(な형용사 서술은 명사문과 동일하게 です)

2과. い형용사의 활용

핵심 규칙

い형용사는 어미 い를 갈아 끼워서 활용합니다. 高(たか)い로 전체 표:

보통체정중체
현재 긍정高い高いです
현재 부정高くない高くないです (=高くありません)
과거 긍정高かった高かったです
과거 부정高くなかった高くなかったです (=高くありませんでした)
명사 수식高い+명사(동일)
연결(그리고)高くて(동일)
  • 활용 공식: い → くない / かった / くなかった / くて. 정중체는 각 형태 뒤에 です만 붙입니다.
  • 최대 함정: 과거형. 「✕ 高いでした」가 아니라 「○ 高かったです」. い형용사는 자기 몸으로 시제를 표현하고, です는 정중함만 담당합니다.
  • 유일한 예외 いい(좋다): 활용할 때는 옛형태 よい 쪽이 활용합니다.
いい의 활용형태
현재いい (よい)
부정よくない (✕ いくない)
과거よかった (✕ いかった)
과거 부정よくなかった
연결よくて
  • かっこいい(멋있다)도 같은 규칙: かっこよかった, かっこよくない.

예문

  • 今日(きょう)は寒(さむ)いですね。

오늘은 춥네요.

  • このラーメン、あまり辛(から)くないです。

이 라멘, 별로 맵지 않아요. (あまり+부정 = 별로 ~않다)

  • 旅行(りょこう)は楽(たの)しかったです。

여행은 즐거웠습니다.

  • 昨日(きのう)のテストは難(むずか)しくなかったです。

어제 시험은 어렵지 않았습니다.

  • 天気(てんき)がよくて、気持(きも)ちがいいです。

날씨가 좋아서 기분이 좋습니다. (いい→よくて)

한국어 화자 팁

  • 한국어는 "쌌습니다"처럼 어간이 활용하고 "-습니다"가 정중함을 담당합니다. 일본어 い형용사도 완전히 같은 구조입니다: 安かった(쌌)+です(습니다). 「安いでした」가 왜 틀린지 한국어로 이해 가능: "싸다였습니다"라고 말하는 셈입니다.
  • 부정의 くない는 "~지 않다"에 대응: 高くない=높지 않다/비싸지 않다.

⚠️ 자주 하는 실수

  1. ✕ おいしいでした — ○ おいしかったです。 い형용사 과거에 でした 금지. (명사·な형용사만 でした를 씁니다)
  2. ✕ いくない・いかった — いい는 활용형에서 よ로 변신: よくない・よかった。

미니 확인문제

Q1. 忙(いそが)しい를 과거 정중체로.

Q2. "이 영화는 재미없었어" (보통체) — 面白(おもしろ)い 사용.

Q3. "머리가 좋고 친절합니다" — 頭(あたま)がいい를 연결형으로.

정답 보기

A1. 忙(いそが)しかったです。(✕ 忙しいでした)

A2. この映画(えいが)は面白(おもしろ)くなかった。

A3. 頭(あたま)がよくて、親切(しんせつ)です。(いい→よくて)

3과. な형용사의 활용

핵심 규칙

な형용사의 활용은 명사문(です/だ)과 100% 동일합니다. 静(しず)か로 전체 표:

보통체정중체
현재 긍정静かだ静かです
현재 부정静かじゃない静かじゃありません (=じゃないです)
과거 긍정静かだった静かでした
과거 부정静かじゃなかった静かじゃありませんでした
명사 수식静かな+명사(동일)
연결(그리고)静かで(동일)
  • Part 1의 명사문 표와 비교해 보세요 — 완전히 같습니다. 다른 것은 명사 수식(な)과 연결형(で)뿐.
  • い형용사와 달리 な형용사는 でした를 씁니다: 静かでした ○.

예문

  • この辺(あた)りは夜(よる)、静(しず)かです。

이 근처는 밤에 조용합니다.

  • 昔(むかし)、ここは有名(ゆうめい)な観光地(かんこうち)でした。

옛날에 여기는 유명한 관광지였습니다.

  • その説明(せつめい)は簡単(かんたん)じゃなかった。

그 설명은 간단하지 않았다.

  • 彼(かれ)は真面目(まじめ)で、親切(しんせつ)です。

그는 성실하고 친절합니다. (연결형 で)

  • 交通(こうつう)が便利(べんり)なところに住(す)みたいです。

교통이 편리한 곳에 살고 싶습니다. (명사 수식 な)

한국어 화자 팁

  • い형용사 연결은 ~くて, な형용사 연결은 ~で: 安くて おいしい店 / きれいで 静かな町. 한국어 "-고"에 해당하는 이 연결형은 Part 3의 て형과 함께 회화 필수품입니다.
  • 이유를 나타낼 수도 있습니다: 静かで いいですね (조용해서 좋네요) — 한국어 "-아서/어서"와 겹치는 용법.

⚠️ 자주 하는 실수

  1. ✕ 静かかった / ✕ 静かくない — い형용사 활용을 な형용사에 적용한 실수. ○ 静かだった / 静かじゃない。 "な형용사는 명사처럼"만 기억하면 안 틀립니다.
  2. ✕ 便利の店 / ✕ 便利い店 — 명사 수식은 な: ○ 便利な店。

미니 확인문제

Q1. 元気(げんき)를 과거 정중체로.

Q2. "일본어는 능숙하지 않습니다(잘하지 못합니다)" — 上手(じょうず) 사용, 정중체.

Q3. "한가하고 조용한 주말이었습니다" — 暇(ひま)/静(しず)か/週末(しゅうまつ).

정답 보기

A1. 元気(げんき)でした。(な형용사는 でした OK — い형용사와 반대)

A2. 日本語(にほんご)は上手(じょうず)じゃありません。(=上手じゃないです)

A3. 暇(ひま)で、静(しず)かな週末(しゅうまつ)でした。(연결 で + 수식 な)

4과. 부사화 — く/に, 그리고 なる

핵심 규칙

형용사를 "~하게"로 바꾸는 규칙:

종류규칙예
い형용사い → く早(はや)い → 早く (일찍/빨리)
な형용사+ に静(しず)か → 静かに (조용히)

なる(되다)와의 결합 — "~해지다, ~이 되다":

종류공식예
い형용사~くなる暖(あたた)かくなる (따뜻해지다)
な형용사~になるきれいになる (예뻐지다)
명사~になる先生(せんせい)になる (선생님이 되다)

예문

  • もう少(すこ)しゆっくり話(はな)してください。

조금 더 천천히 말해 주세요. (ゆっくり는 원래 부사)

  • 朝(あさ)早(はや)く起(お)きて、夜(よる)遅(おそ)く寝(ね)ます。

아침 일찍 일어나고 밤늦게 잡니다.

  • 図書館(としょかん)では静(しず)かにしてください。

도서관에서는 조용히 해 주세요.

  • 最近(さいきん)、日本語(にほんご)が上手(じょうず)になりました。

요즘 일본어가 늘었습니다(능숙해졌습니다).

  • 部屋(へや)を掃除(そうじ)して、きれいになりました。

방을 청소해서 깨끗해졌습니다.

한국어 화자 팁

  • 한국어 "선생님이 되다"의 "이" 때문에 「✕ 先生がなる」라고 하기 쉽습니다. 일본어는 になる: ○ 先生になる. "~이 되다 = ~になる"를 통째로 외우세요.
  • 「暑くなりました(더워졌네요)」「安くなりました(싸졌어요)」처럼 くなる/になる는 변화를 말하는 만능 패턴입니다. 계절 인사, 쇼핑 어디서나 쓰입니다.

⚠️ 자주 하는 실수

  1. ✕ 先生がなりました — ○ 先生になりました。 (되다의 대상은 に)
  2. ✕ 静かくなる — 静か는 な형용사이므로 ○ 静かになる。 (く는 い형용사 전용)

미니 확인문제

Q1. "빨리 먹어요"의 "빨리" — 速(はや)い를 부사로.

Q2. "물가가 비싸졌습니다" — 物価(ぶっか)/高(たか)い/なる.

Q3. "아들은 의사가 되었습니다" — 息子(むすこ)/医者(いしゃ)/なる.

정답 보기

A1. 速(はや)く食(た)べます。

A2. 物価(ぶっか)が高(たか)くなりました。

A3. 息子(むすこ)は医者(いしゃ)になりました。(✕ 医者がなりました)

5과. 비교 표현

핵심 규칙

일본어 형용사에는 영어의 -er/-est 같은 비교급 변화가 없습니다. 조사와 부사로 해결합니다.

문형뜻예
AはBより ~A는 B보다 ~飛行機(ひこうき)は電車(でんしゃ)より速(はや)い
BよりAのほうが ~B보다 A 쪽이 ~電車より飛行機のほうが速い
AとBと、どちらが ~A와 B 중 어느 쪽이 ~?コーヒーと紅茶(こうちゃ)と、どちらが好(す)きですか
—のほうが ~(대답) ~ 쪽이コーヒーのほうが好きです
(범위)で(의문사)が一番(いちばん) ~~에서 무엇이 제일 ~?果物(くだもの)で何(なに)が一番好きですか
—が一番 ~(대답) ~이 제일いちごが一番好きです
  • 비교 의문은 대상이 둘이면 どちら, 셋 이상이거나 범위면 何/誰/どこ+一番.
  • どちら의 회화체는 どっち.

예문

  • 韓国(かんこく)は日本(にほん)より冬(ふゆ)が寒(さむ)いです。

한국은 일본보다 겨울이 춥습니다.

  • バスより地下鉄(ちかてつ)のほうが速(はや)いですよ。

버스보다 지하철 쪽이 빨라요.

  • 犬(いぬ)と猫(ねこ)と、どちらが好(す)きですか。— 猫のほうが好きです。

개와 고양이 중 어느 쪽을 좋아해요? — 고양이 쪽을 좋아해요.

  • クラスで誰(だれ)が一番(いちばん)背(せ)が高(たか)いですか。

반에서 누가 제일 키가 큽니까?

  • 一年(いちねん)で八月(はちがつ)が一番暑(あつ)いです。

일 년 중 8월이 제일 덥습니다.

한국어 화자 팁

  • より=보다, 一番=제일/가장, のほうが=~쪽이. 한국어와 어순까지 같아서 그대로 조립하면 됩니다.
  • 둘 중 하나를 묻는 질문에 대답할 때는 のほうが를 붙이는 것이 자연스럽습니다. 「猫が好きです」보다 「猫のほうが好きです」가 "(둘 중에선) 고양이 쪽"이라는 질문 의도에 맞는 대답입니다.

⚠️ 자주 하는 실수

  1. ✕ どちらが一番好きですか(대상이 둘일 때) — 둘일 때는 一番을 쓰지 않습니다. ○ どちらが好きですか。 一番은 셋 이상/범위 전용.
  2. ✕ もっと높다는 뜻으로 형용사를 변형하려는 시도 — 일본어 형용사는 비교급 변화가 없습니다. 비교 상대를 より로 밝히거나 부사 もっと(더)를 앞에 둡니다: もっと安いのはありますか (더 싼 것은 있나요?).

미니 확인문제

Q1. "여름과 겨울 중 어느 쪽을 좋아합니까?" — 夏(なつ)/冬(ふゆ).

Q2. "오늘은 어제보다 따뜻합니다" — 今日(きょう)/昨日(きのう)/暖(あたた)かい.

Q3. "일본 음식 중에서 뭐가 제일 맛있었어요?" — 日本(にほん)の食(た)べ物(もの)/おいしい.

정답 보기

A1. 夏(なつ)と冬(ふゆ)と、どちらが好(す)きですか。

A2. 今日(きょう)は昨日(きのう)より暖(あたた)かいです。

A3. 日本(にほん)の食(た)べ物(もの)で何(なに)が一番(いちばん)おいしかったですか。(범위의 で + 一番 + い형용사 과거)

6과. 자주 틀리는 것 모음 — 함정 단어 사전

핵심 규칙

한국어 화자가 형용사에서 틀리는 지점 총정리:

함정정체올바른 사용
きれいい로 끝나지만 な형용사きれいな花 / きれいでした / 부정 きれいじゃない (✕ きれくない)
嫌(きら)いい로 끝나지만 な형용사嫌いな人 / 嫌いじゃない (✕ 嫌くない)
好(す)き・嫌(きら)い한국어는 동사(좋아하다/싫어하다), 일본어는 형용사대상에 が: 犬が好きだ (✕ 犬を好きだ)
同(おな)じな형용사인데 명사 수식 시 な가 안 붙는 특수형同じ学校 (✕ 同じな学校). 단 서술은 同じだ/同じです
大(おお)きい/小(ちい)さいい형용사지만 大きな/小さな라는 연체사 형태도 존재大きい犬 ○ / 大きな犬 ○ (둘 다 가능. 大きな는 수식 전용)
いい활용은 よい가 담당よくない/よかった (✕ いくない/いかった)
多(おお)い/少(すく)ない명사를 직접 수식하기 어려움✕ 多い人が来た → ○ 人が多い / 多くの人(ひと)
ほしい형용사. 대상에 が車(くるま)がほしい (✕ 車をほしい)

예문

  • 私(わたし)は納豆(なっとう)が嫌(きら)いじゃないです。

저는 낫토를 싫어하지 않아요. (✕ 嫌くない)

  • 私(わたし)たちは同(おな)じクラスです。

우리는 같은 반입니다. (✕ 同じなクラス)

  • 昨日(きのう)より天気(てんき)がよくなりました。

어제보다 날씨가 좋아졌습니다. (いい→よく)

  • このカフェは人(ひと)が多(おお)いですね。

이 카페는 사람이 많네요. (✕ 多い人がいますね)

한국어 화자 팁

  • "좋아하다·싫어하다"를 동사로 번역하려는 습관이 好き・嫌い 오용의 뿌리입니다. 일본어에서는 "개가 좋다/개가 싫다"라고 말한다고 생각을 전환하면 が도 자연스럽게 따라옵니다.
  • 同じ는 일본어 문법에서도 특급 예외입니다. "同じ+명사에는 な 없음" 하나만 통째로 암기.

⚠️ 자주 하는 실수

  1. ✕ きれくない / ✕ きれいくない — きれい는 な형용사. ○ きれいじゃない。
  2. ✕ 同じな意味(いみ) — ○ 同じ意味。 (단, 서술로는 「意味が同じだ」 ○)

미니 확인문제

Q1. "그 가수를 좋아했었어(지금은 아님)" — 보통체로. (その歌手(かしゅ)/好き)

Q2. "어제는 날씨가 좋지 않았습니다" — 天気(てんき)/いい.

Q3. 다음 중 틀린 것을 모두 고르고 고치세요: ① 大きな犬 ② 同じな学校 ③ きれいな部屋 ④ いくなかった

정답 보기

A1. その歌手(かしゅ)が好(す)きだった。(な형용사 과거 보통체 = だった)

A2. 昨日(きのう)は天気(てんき)がよくなかったです。(=よくありませんでした)

A3. ②를 同じ学校로(な 제거), ④를 よくなかった로. ①③은 맞음.

Part 2 마무리 체크리스트

  • [ ] 명사를 수식해 보고 い형용사/な형용사를 판별할 수 있다
  • [ ] 高い의 4활용(くない/かった/くなかった)과 いい→よ~ 예외를 말할 수 있다
  • [ ] な형용사는 "명사문과 같다"는 것을 알고 でした/じゃない를 쓸 수 있다
  • [ ] くなる/になる로 변화를 표현할 수 있다
  • [ ] より/のほうが/一番으로 비교·최상 표현을 만들 수 있다
  • [ ] きれい・嫌い・同じ・好き의 함정을 피할 수 있다
