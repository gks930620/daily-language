2주차 — 가타카나 + 자기소개

이번 주 목표

  1. 가타카나 46자 + 외래어 표기 규칙(장음 ー 포함)을 읽을 수 있다.
  2. 이름·국적·직업이 들어간 자기소개를 30초 동안 막힘없이 할 수 있다.
  3. 뼈대 문형 「〜は 〜です(긍정) / 〜ですか(질문) / 〜じゃないです(부정)」를 부린다.

이번 주까지는 한글 발음 보조 [ ]가 붙습니다. 다음 주부터는 없습니다.

가타카나 오십음도 — 이번 주의 지도

발음은 히라가나와 완전히 같습니다. 모양만 새로 익히면 됩니다. 주 용도: 외래어(コーヒー), 외국 지명·인명(ソウル、キム).

6차시 — 가타카나 ア〜ト / 이름 말하기

① 오늘의 대화

어학원 교실, 처음 만난 짝과.

  • A: はじめまして。キム・ミンジュンです。 [하지메마시테. 키무 민준데스]
    • 처음 뵙겠습니다. 김민준입니다.
  • B: はじめまして。さとうです。おなまえは、ミンジュンさん? [하지메마시테. 사토-데스. 오나마에와, 민준상?]
    • 처음 뵙겠습니다. 사토입니다. 성함이, 민준 씨(라고요)?
  • A: はい、ミンジュンです。よろしく おねがいします。 [하이, 민준데스. 요로시쿠 오네가이시마스]
    • 네, 민준입니다. 잘 부탁드립니다.
  • B: よろしく おねがいします。 [요로시쿠 오네가이시마스]
    • 잘 부탁드립니다.

③ 오늘의 포인트

1. 가타카나 전반 20자 (ア〜ト)

히라가나와 짝지어 외우면 빠릅니다: あ→ア, か→カ, さ→サ… 헷갈림 3대장 예고: シ(시)와 ツ(츠), ソ(소)와 ン(ㄴ). 구분법 — シ·ン은 획이 가로로 눕고 아래에서 위로 삐침, ツ·ソ는 획이 세로로 서고 위에서 아래로 삐침. 오늘은 ア〜ト 20자를 히라가나 옆에 나란히 5번씩 쓰세요.

2. 한국 이름은 가타카나로

외국인의 이름은 가타카나로 씁니다. キム(김), イ(이), パク(박), チェ(최), チョン(정). 이름 사이 가운뎃점 「・」은 성과 이름의 경계 표시입니다(キム・ミンジュン). 내 이름의 가타카나 표기를 오늘 확정하세요 — 앞으로 8주 내내 씁니다.

④ 소리 내어 연습

  1. 자기 성+이름을 가타카나로 종이에 쓰고, 「はじめまして。○○です。よろしく おねがいします。」를 3번 말하세요.
  2. 상대 이름을 확인하는 B의 말 「おなまえは、○○さん?」을 친구 이름을 넣어 2번 말해 보세요.
  3. ア〜ト 20자를 무작위로 짚으며 읽으세요. 히라가나 짝(ア=あ…)도 같이 말하면 두 배 효과.

7차시 — 가타카나 ナ〜ホ / 국적: 〜じんです·〜から きました

① 오늘의 대화

게스트하우스 라운지에서.

  • A: ミンジュンさんは にほんじん ですか。 [민준상와 니혼진 데스카]
    • 민준 씨는 일본인이에요?
  • B: いいえ、かんこくじん です。ソウルから きました。 [이-에, 캉코쿠진 데스. 소우루카라 키마시타]
    • 아니요, 한국인이에요. 서울에서 왔어요.
  • A: そうですか。わたしは にほんじんです。おおさかから きました。 [소-데스카. 와타시와 니혼진데스. 오-사카카라 키마시타]
    • 그렇군요. 저는 일본인이에요. 오사카에서 왔어요.
  • B: おおさか! いいですね。 [오-사카! 이-데스네]
    • 오사카! 좋네요.

② 새 단어

③ 오늘의 포인트

1. 가타카나 ナ〜ホ (10자)

ナ ニ ヌ ネ ノ / ハ ヒ フ ヘ ホ. ヘ는 히라가나 へ와 거의 같은 모양이라 덤으로 얻는 글자입니다. ソウル(서울)의 ソ를 쓰며 ン과 비교해 보세요.

2. 나라 이름 + じん = 국적, 문장 끝 か = 물음표

かんこく+じん=한국인, にほん+じん=일본인, アメリカ+じん=미국인. 한국어 "~인"과 같은 원리라 공짜로 먹는 문형입니다. 질문은 문장 끝에 か만 붙이면 끝: にほんじんです(일본인입니다) → にほんじんですか(일본인입니까?). 어순 변화도, 조동사도 없습니다. 답은 はい(네) / いいえ(아니요)로 시작하면 됩니다. ※ 참고: 대답의 そうですか는 끝을 내리면 "그렇군요", 올리면 "그래요?"가 됩니다.

④ 소리 내어 연습

  1. 「わたしは かんこくじんです。○○から きました。」에 자기 도시(ソウル/プサン부산/インチョン인천 등)를 넣어 3번.
  2. 상대에게 「○○さんは にほんじんですか。」라고 묻고, 스스로 B가 되어 いいえ로 답하는 것까지 혼자 2역.
  3. 어제와 오늘 배운 가타카나 30자(ア〜ホ)를 통독. シ·ツ·ソ·ン 네 글자는 따로 5번씩.

8차시 — 가타카나 マ〜ン + 장음 ー / 직업 말하기

① 오늘의 대화

교류회에서, 명함을 주고받으며.

  • A: おしごとは なんですか。 [오시고토와 난데스카]
    • 하시는 일이 뭐예요?
  • B: かいしゃいんです。ITの かいしゃです。 [카이샤인데스. 아이티-노 카이샤데스]
    • 회사원이에요. IT 회사요.
  • A: そうですか。わたしは がくせいです。 [소-데스카. 와타시와 가쿠세-데스]
    • 그렇군요. 저는 학생이에요.
  • B: だいがくせい ですか。 [다이가쿠세- 데스카]
    • 대학생이에요?
  • A: はい、だいがくせいです。 [하이, 다이가쿠세-데스]
    • 네, 대학생이에요.

② 새 단어

③ 오늘의 포인트

1. 가타카나 마지막 그룹(マ〜ン) + 장음 기호 ー

マ ミ ム メ モ / ヤ ユ ヨ / ラ リ ル レ ロ / ワ ヲ ン — 46자 완성입니다. 가타카나의 장음은 히라가나와 달리 막대기 ー 하나로 씁니다: コーヒー(커피, 코-히-), スーパー(슈퍼), ソウル이 아닌 예로 ニュース(뉴스). 요음·촉음 규칙은 히라가나와 동일합니다(작은 ャュョ, 작은 ッ).

2. の로 명사를 잇는다 + 직업 묻기

「ITの かいしゃ」= IT(의) 회사. 일본어는 명사+명사를 이을 때 거의 항상 の를 넣습니다. 한국어에서 "의"를 생략하는 자리에도 の는 필요합니다: 한국 회사 → かんこくの かいしゃ. 직업 질문 「おしごとは なんですか」는 통째로 외우세요. 남의 것에는 お(오시고토), 내 것에는 안 붙입니다(내 직업을 말할 땐 그냥 しごと).

④ 소리 내어 연습

  1. 자기 직업을 넣어 「わたしは ○○です。」 3번. (회사원/학생/선생님이 아니면 지금은 かいしゃいん으로 대체 — 6차시 이름처럼 사전에서 자기 직업을 찾아 확정해 두면 최고)
  2. 「おしごとは なんですか。」→ 자기 답 → 「○○さんは?」(○○씨는요?)까지 이어 말해 보세요.
  3. 가타카나로 읽기: コーヒー / スーパー / ソウル / アメリカ / ニュース. 장음 ー에서 확실히 한 박자 끌기.

9차시 — です 3형제: です·ですか·じゃないです

① 오늘의 대화

교류회 뒷풀이에서, 사토 씨가 사람을 착각한 상황.

  • A: たなかさんは せんせい ですか。 [타나카상와 센세- 데스카]
    • 다나카 씨는 선생님이에요?
  • B: いいえ、せんせい じゃないです。かいしゃいんです。 [이-에, 센세- 쟈나이데스. 카이샤인데스]
    • 아니요, 선생님이 아니에요. 회사원이에요.
  • A: あ、すみません。じゃ、あのひとが せんせいですか。 [아, 스미마셍. 쟈, 아노히토가 센세-데스카]
    • 아, 죄송해요. 그럼 저 사람이 선생님이에요?
  • B: はい、そうです。やまだせんせいです。 [하이, 소-데스. 야마다센세-데스]
    • 네, 맞아요. 야마다 선생님이에요.

③ 오늘의 포인트

1. 이 책의 뼈대 — です 3형제

형태뜻예
〜です~입니다がくせいです
〜ですか~입니까?がくせいですか
〜じゃないです~이 아닙니다がくせい じゃないです

한국어와 1:1 대응입니다: "학생입니다/입니까/이 아닙니다". じゃないです는 회화체 표준이고, 더 격식 차린 형태 では ありません은 눈으로만 알아 두면 됩니다(문서·아나운서 말투).

2. 질문에 답하는 두 갈래

  • 맞으면: はい、そうです。(네, 맞아요)
  • 아니면: いいえ、〜じゃないです。○○です。(아니요, ~가 아니에요. ○○예요) — 부정만 하고 끝내지 말고 정답을 덧붙이는 것이 일본어 회화의 매너입니다.

④ 소리 내어 연습

  1. 다음 질문에 사실대로 답하세요(소리 내어): ①あなたは にほんじんですか。 ②あなたは せんせいですか。 ③あなたは だいがくせいですか。
  2. 틀린 정보 → 정정 놀이: 「キムさんは いしゃ(의사)ですか」라고 스스로 묻고 「いいえ、いしゃじゃないです。○○です」로 답하기.
  3. 「あのひとが やまだせんせいですか。」— 끝을 올려 질문 톤으로 3번. (が는 "바로 저 사람이?"라고 콕 집는 조사 — 5주차에 제대로 다룹니다)

10차시 — 자기소개 완성: 30초 스피치

① 오늘의 대화

일본어 교실에서 돌아가며 자기소개하는 날.

  • A: では、じこしょうかいを どうぞ。 [데와, 지코쇼-카이오 도-조]
    • 그럼, 자기소개 부탁합니다.
  • B: はい。はじめまして。キム・ミンジュンです。 [하이. 하지메마시테. 키무 민준데스]
    • 네. 처음 뵙겠습니다. 김민준입니다.
  • B: かんこくの ソウルから きました。かいしゃいんです。 [캉코쿠노 소우루카라 키마시타. 카이샤인데스]
    • 한국 서울에서 왔습니다. 회사원입니다.
  • B: にほんごは まだまだですが、がんばります。 [니홍고와 마다마다데스가, 감바리마스]
    • 일본어는 아직 멀었지만, 열심히 하겠습니다.
  • B: よろしく おねがいします。 [요로시쿠 오네가이시마스]
    • 잘 부탁드립니다.
  • A: ありがとうございました。 [아리가토- 고자이마시타]
    • 감사합니다. (박수)

③ 오늘의 포인트

1. 자기소개 5문장 템플릿

① はじめまして。② (이름)です。③ (나라·도시)から きました。④ (직업)です。⑤ よろしく おねがいします。 여기에 한 스푼: 「にほんごは まだまだですが、がんばります」(일본어는 아직 멀었지만 열심히 하겠습니다) — 이 한 줄이 들어가면 박수가 나옵니다. 〜が는 "~지만"으로 문장을 잇는 접속 조사입니다.

2. 나라 + ご = 언어

にほん+ご=일본어, かんこく+ご=한국어, ちゅうごく+ご=중국어. 〜じん(사람)과 함께 세트로: 나라 → +じん(사람)/+ご(말). 단, 영어만 예외로 えいご입니다.

④ 소리 내어 연습

  1. 템플릿에 자기 정보를 넣어 자기소개를 종이에 쓰고(가나로!), 안 보고 말할 수 있을 때까지 반복하세요. 목표: 30초, 막힘 3회 이하.
  2. 스마트폰으로 자기소개를 녹음해서 들어 보세요. 장음(〜です는 짧게, きました는 끊어서)이 뭉개지지 않았는지 확인.
  3. 누군가에게(가족이라도) 실제로 이 자기소개를 들려주고 박수를 받아 내세요.

주말 복습 — 2주차: 가타카나 + 자기소개 총정리

통합 대화 — 교류회에서의 첫 만남 풀버전

  • A: こんばんは。はじめまして。さとうです。 [콤방와. 하지메마시테. 사토-데스]
    • 안녕하세요(저녁). 처음 뵙겠습니다. 사토입니다.
  • B: はじめまして。キム・ミンジュンです。よろしく おねがいします。 [하지메마시테. 키무 민준데스. 요로시쿠 오네가이시마스]
    • 처음 뵙겠습니다. 김민준입니다. 잘 부탁드립니다.
  • A: ミンジュンさんは ちゅうごくじん ですか。 [민준상와 츄-고쿠진 데스카]
    • 민준 씨는 중국인이세요?
  • B: いいえ、ちゅうごくじん じゃないです。かんこくじんです。ソウルから きました。 [이-에, 츄-고쿠진 쟈나이데스. 캉코쿠진데스. 소우루카라 키마시타]
    • 아니요, 중국인이 아니에요. 한국인이에요. 서울에서 왔어요.
  • A: そうですか。おしごとは なんですか。 [소-데스카. 오시고토와 난데스카]
    • 그렇군요. 하시는 일이 뭐예요?
  • B: かいしゃいんです。ITの かいしゃです。さとうさんは? [카이샤인데스. 아이티-노 카이샤데스. 사토-상와?]
    • 회사원이에요. IT 회사요. 사토 씨는요?
  • A: わたしは だいがくせいです。にほんごの せんせい じゃないですよ。 [와타시와 다이가쿠세-데스. 니홍고노 센세- 쟈나이데스요]
    • 저는 대학생이에요. 일본어 선생님이 아니고요.
  • B: そうですか。これから よろしく おねがいします。 [소-데스카. 코레카라 요로시쿠 오네가이시마스]
    • 그렇군요. 앞으로 잘 부탁드립니다.

※ 새 표현: 문장 끝 よ는 "~예요(알려 드려요)"라고 정보를 건네는 어감, これから는 "앞으로". 「○○さんは?」처럼 조사 は에서 끊고 끝을 올리면 "○○씨는요?"라는 훌륭한 되묻기가 됩니다.

가타카나 최종 점검

  1. 가타카나 46자를 안 보고 쓰기. 특히 シ/ツ, ソ/ン은 삐침 방향을 말로 설명해 보기.
  2. 읽기 시험: コーヒー / ソウル / プサン / アメリカ / スーパー / ニュース / チェ / パク
  3. 자기 이름 가타카나를 명함 크기 종이에 크게 써서 책상 앞에 붙이세요.

셀프 체크리스트

  • [ ] 가타카나 46자를 읽을 수 있다 (쓰기는 틀린 글자 8개 이하)
  • [ ] シ·ツ·ソ·ン을 구분한다
  • [ ] 장음 기호 ー의 역할을 안다
  • [ ] 자기소개 5문장을 안 보고 30초 안에 말할 수 있다
  • [ ] です/ですか/じゃないです를 자유롭게 바꿔 말할 수 있다
  • [ ] 질문에 いいえ로 답할 때 정답을 덧붙이는 습관이 생겼다
  • [ ] 나라+じん/ご 규칙으로 새 단어(예: ちゅうごくじん)를 스스로 만들 수 있다

자기소개가 아직 버벅이면 주말 내내 그것만 반복해도 됩니다. 3주차부터는 한글 발음 보조가 사라집니다 — 이번 주말이 가나 완성의 마지노선입니다.

다음 주 예고: 3주차에서는 숫자의 세계로 갑니다. 전화번호·시간·요일·날짜, 그리고 물건을 가리키는 これ/それ/あれ와 쇼핑의 필수 질문 「いくらですか(얼마예요?)」까지. 기초 한자도 후리가나와 함께 처음 등장합니다. 何時(なんじ)ですか — 지금 몇 시예요?
