4주차 — 실전 쇼핑과 주문: 편의점·옷가게·식당·카페

이번 주 목표

  1. 편의점·옷가게·식당·카페에서 원하는 것을 주문하고 계산까지 마칠 수 있다.
  2. 주문 3형제 「〜を ください / 〜を おねがいします / 〜に します」를 상황에 맞게 골라 쓴다.
  3. 점원의 정형 멘트(いらっしゃいませ 등)를 알아듣고 당황하지 않는다.

이번 주부터는 "말하기"만큼 "알아듣기"가 중요합니다. 점원 대사는 빠르지만 거의 100% 정해진 문장입니다 — 통째로 귀에 넣어 두면 됩니다.

16차시 — 편의점: 기본 결제 흐름

① 오늘의 대화

편의점 계산대에서. 삼각김밥과 차를 들고.

  • 店員(てんいん): いらっしゃいませ。
    • 어서 오세요.
  • A: (상품을 올려놓는다)
  • 店員: おにぎりと お茶(ちゃ)ですね。袋(ふくろ)は ご利用(りよう)ですか。
    • 삼각김밥과 차 맞으시죠? 봉투 필요하세요?
  • A: あ、大丈夫(だいじょうぶ)です。
    • 아, 괜찮아요. (필요 없어요)
  • 店員: 合計(ごうけい)、320円(さんびゃくにじゅうえん)です。
    • 합계 320엔입니다.
  • A: カードで おねがいします。
    • 카드로 부탁해요.
  • 店員: ありがとうございました。
    • 감사합니다.

일본어읽기뜻
店員てんいん점원
いらっしゃいませいらっしゃいませ어서 오세요 (대답 안 해도 됨)

③ 오늘의 포인트

1. 거절의 만능 표현 — 大丈夫です

"봉투 필요하세요?" "포인트카드 있으세요?" 같은 질문에 필요 없으면 전부 大丈夫です로 통합니다. 한국어 "괜찮아요"와 용법이 같아서 바로 체득됩니다. 필요하면 「はい、おねがいします」.

2. 수단의 で — "~(으)로"

カードで(카드로), 現金(げんきん)で(현금으로), 日本語で(일본어로). 한국어 조사 "~(으)로"의 수단 용법과 정확히 대응합니다. 「カードで おねがいします」는 계산대 필수 문장.

④ 소리 내어 연습

  1. 점원 대사 3줄(いらっしゃいませ/袋は ご利用ですか/合計〜円です)을 빠르게 읽고, 각각에 자기 대답을 붙이세요.
  2. 결제 수단 바꿔 말하기: 現金で / カードで / スマホで + おねがいします。
  3. 「おにぎりと お茶」처럼 と로 물건 2개 묶기: 자기 장바구니 조합 3개 만들어 말하기. (예: パンと コーヒー)

17차시 — 옷가게: 보여 주세요, 입어 봐도 돼요?

① 오늘의 대화

옷가게에서 셔츠를 고르며.

  • A: すみません、あの 黒(くろ)い シャツを 見(み)せてください。
    • 저기요, 저 검은 셔츠를 보여 주세요.
  • 店員: はい、こちらですね。どうぞ。
    • 네, 이거죠? 여기요.
  • A: ちょっと 大(おお)きいですね。もう 少(すこ)し 小(ちい)さいのは ありますか。
    • 좀 크네요. 조금 더 작은 건 있어요?
  • 店員: はい、Mサイズも ありますよ。
    • 네, M사이즈도 있어요.
  • A: じゃ、それを おねがいします。試着(しちゃく)、いいですか。
    • 그럼 그걸로 부탁해요. 입어 봐도 돼요?
  • 店員: はい、どうぞ。こちらへ。
    • 네, 그럼요. 이쪽으로 오세요.

② 새 단어

일본어읽기뜻

見せてくださいみせてください보여 주세요

③ 오늘의 포인트

1. 〜てください 맛보기: 見せてください

「〜てください」는 "~해 주세요"입니다. 동사 활용(て형)은 2권 주제라서, 지금은 자주 쓰는 완성품 3개만 통째로 갖고 갑니다: 見せてください(보여 주세요) / 待(ま)ってください(기다려 주세요) / もう一度(いちど) 言(い)ってください(다시 한번 말해 주세요). 마지막 것은 일본어 학습자의 생명줄입니다.

2. ありますか — "있어요?"의 데뷔

「Mサイズは ありますか」(M사이즈는 있어요?), 「もう少し 小さいのは ありますか」(좀 더 작은 건 있어요?). あります(있습니다)는 5주차의 핵심 문형인데, 가게에서 먼저 만나 둡니다. 대답: はい、ありますよ / すみません、ないです(없어요).

④ 소리 내어 연습

  1. 색 바꿔 요청하기: 黒いの / 白いの / 赤(あか)いの(빨간 것) + を 見せてください。
  2. 「もう少し 大きいのは ありますか」 ↔ 「もう少し 小さいのは ありますか」 번갈아 3번.
  3. 생명줄 문장 암기 확인: "다시 한번 말해 주세요"를 일본어로 즉답. (もう一度 言ってください / 더 정중히는 お願いします를 붙임)

18차시 — 식당 입장과 주문: 〜に します

① 오늘의 대화

라멘집에 두 명이 들어간다.

  • 店員: いらっしゃいませ。何名様(なんめいさま)ですか。
    • 어서 오세요. 몇 분이세요?
  • A: 二人(ふたり)です。
    • 두 명이요.
  • 店員: こちらへ どうぞ。ご注文(ちゅうもん)は?
    • 이쪽으로 오세요. 주문은요?
  • A: えっと、おすすめは 何(なん)ですか。
    • 음, 추천 메뉴는 뭐예요?
  • 店員: みそラーメンが 人気(にんき)ですよ。
    • 미소라멘이 인기예요.
  • A: じゃ、わたしは みそラーメンに します。
    • 그럼 저는 미소라멘으로 할게요.
  • B: わたしは しょうゆラーメンと ぎょうざを おねがいします。
    • 저는 간장라멘하고 교자를 부탁해요.

② 새 단어

일본어읽기뜻
何名様なんめいさま몇 분 (손님 수 묻기)
一人 / 二人 / 三人ひとり / ふたり / さんにん한 명 / 두 명 / 세 명
注文ちゅうもん주문

〜に しますにします~으로 할게요 (선택)

③ 오늘의 포인트

1. 주문 3형제 총정리

표현어감상황
〜を ください~을 주세요직접적. 물건 살 때 무난
〜を おねがいします~을 부탁해요한 톤 부드러움. 어디서나 안전
〜に します~으로 할게요여럿 중 선택했을 때

메뉴판을 보고 고민하다 결정하는 순간이 바로 に します의 자리입니다. 한국어 "~으로 할게요"와 같은 발상.

2. 사람 수 세기 — 예외 2개

一人(ひとり)와 二人(ふたり)만 고유어이고, 3명부터는 숫자+にん(さんにん、よにん※、ごにん…)입니다. ※4명은 よにん(しにん이라고 하면 死人과 소리가 겹쳐서 안 씁니다). 「何名様ですか」에 손가락 들며 「二人です」면 충분합니다.

④ 소리 내어 연습

  1. 「おすすめは 何ですか」→ (점원 답 상상) → 「じゃ、それに します」 흐름을 2번 연기.
  2. 인원수 즉답: 혼자 / 둘 / 셋 / 넷이 들어갔을 때 각각 뭐라고 답할지.
  3. 2인분 주문 통합: 「みそラーメンと ぎょうざを おねがいします。」에 자기가 좋아하는 메뉴를 넣어 변형 2개.

19차시 — 식당에서 부르기·추가·계산

① 오늘의 대화

식사 중반, 그리고 계산까지.

  • A: すみませーん!
    • 저기요~! (점원 부르기)
  • 店員: はい!
    • 네!
  • A: お水(みず)を おねがいします。あと、ぎょうざを もう ひとつ。
    • 물 좀 부탁해요. 그리고 교자 하나 더요.
  • 店員: かしこまりました。
    • 알겠습니다.
  • (식사 후)
  • A: すみません、お会計(かいけい) おねがいします。
    • 저기요, 계산 부탁합니다.
  • 店員: はい、合計(ごうけい)で 2,100円(にせんひゃくえん)です。
    • 네, 합계 2,100엔입니다.
  • A: ごちそうさまでした!
    • 잘 먹었습니다!

② 새 단어

일본어읽기뜻

かしこまりましたかしこまりました알겠습니다 (점원의 정중한 응답)

③ 오늘의 포인트

1. 물건 개수 세기 — ひとつ、ふたつ、みっつ…

물건은 ひとつ(1)、ふたつ(2)、みっつ(3)、よっつ(4)、いつつ(5)、むっつ(6)、ななつ(7)、やっつ(8)、ここのつ(9)、とお(10). 13차시 날짜(ついたち、ふつか…)와 어원이 같아서 세트로 외우면 덜 헷갈립니다. 주문할 땐 「ぎょうざを ふたつ おねがいします」처럼 물건を + 개수 + おねがいします 어순.

2. 식사 인사 한 쌍 + 부르기 요령

먹기 전 いただきます, 먹은 후 ごちそうさまでした. 계산하며 or 나가며 점원에게 ごちそうさまでした라고 하면 아주 자연스럽습니다. 점원을 부를 땐 すみません을 크고 길게(すみませーん). 1주차에 배운 "만능 すみません"의 실전 최다 용법입니다.

④ 소리 내어 연습

  1. 추가 주문 3연속: お水を おねがいします / ビールを もう ひとつ / ごはんを おねがいします。
  2. ひとつ~とお를 손가락 꼽으며 왕복 2번.
  3. 입장부터 퇴장까지 셀프 시뮬레이션: 二人です → 주문 → すみませーん+추가 → お会計 おねがいします → ごちそうさまでした。

20차시 — 카페: 사이즈·포장·마무리

① 오늘의 대화

카페 카운터에서.

  • 店員: いらっしゃいませ。お決(き)まりですか。
    • 어서 오세요. 결정하셨어요?
  • A: アイスコーヒーを ひとつ おねがいします。
    • 아이스커피 하나 부탁해요.
  • 店員: サイズは いかがなさいますか。
    • 사이즈는 어떻게 하시겠어요?
  • A: Mで おねがいします。
    • M으로 부탁해요.
  • 店員: 店内(てんない)で お召(め)し上(あ)がりですか。
    • 매장에서 드시나요?
  • A: いいえ、テイクアウトで。
    • 아니요, 테이크아웃으로요.
  • 店員: 450円(よんひゃくごじゅうえん)です。
    • 450엔입니다.

② 새 단어

일본어읽기뜻
お決まりですかおきまりですか결정하셨어요?
アイスコーヒーあいすこーひー아이스커피
いかがなさいますかいかがなさいますか어떻게 하시겠어요? (매우 정중)

③ 오늘의 포인트

1. 점원의 초정중어는 "듣고 고르기"만 하면 된다

いかがなさいますか, お召し上がりですか 같은 존경어를 내가 만들 필요는 없습니다. 질문의 키워드(サイズ? 店内?)만 잡아서 짧게 답하면 됩니다: 「Mで」「テイクアウトで」. 이때의 で도 16차시 수단의 で — "~(으)로"입니다. 점원 말은 길게, 내 말은 짧게가 카페 회화의 정석.

2. 이번 주 총복습 포인트: で의 3용법 미리 정리

카드で(수단), Mで(선택·지정), 店内で(장소에서 — "매장에서 드세요?"의 で). 장소의 で는 7주차 동사 문장에서 본격 등장하니, 지금은 "다 같은 で구나"만 눈에 담아 두세요.

④ 소리 내어 연습

  1. 주문 변형: ホットコーヒーを ひとつ / カフェラテを ふたつ / アイスティーを ひとつ + おねがいします。
  2. 점원 질문 즉답 훈련 — 스스로 묻고 답하기: 「サイズは?」→「Sで」/「店内で お召し上がりですか」→「はい、店内で」or「テイクアウトで」。
  3. 20차시 대화 전체를 점원·손님 2역으로 통독 2번. 두 번째는 점원 대사를 빠르게.

주말 복습 — 4주차: 주문·계산 총정리

통합 대화 — 쇼핑몰에서의 하루 (옷가게 → 푸드코트)

  • 店員: いらっしゃいませ。
    • 어서 오세요.
  • A: すみません、この 白いシャツ、Lサイズは ありますか。
    • 저기요, 이 하얀 셔츠 L사이즈는 있어요?
  • 店員: はい、ありますよ。こちらです。
    • 네, 있어요. 여기요.
  • A: じゃ、これに します。カードで おねがいします。
    • 그럼 이걸로 할게요. 카드로 부탁해요.
  • 店員: ありがとうございます。2,900円(にせんきゅうひゃくえん)です。
    • 감사합니다. 2,900엔입니다.
  • (푸드코트로 이동)
  • B: 何(なに)に しますか。
    • 뭐로 할래요?
  • A: うーん、わたしは カレーに します。ミンジュンさんは?
    • 음, 저는 카레로 할게요. 민준 씨는요?
  • B: わたしは うどんと おにぎりを おねがいします。あ、お水も ください。
    • 저는 우동하고 삼각김밥 부탁해요. 아, 물도 주세요.
  • (식사 후)
  • A: ごちそうさまでした。おいしかったですね。
    • 잘 먹었습니다. 맛있었죠?
  • B: そうですね!
    • 그러게요!

※ 미리보기: おいしかったです(맛있었어요)는 형용사의 과거형 — 6주차에서 배웁니다. 「何に しますか」는 に します의 질문형으로, 친구끼리 메뉴 정할 때의 단골 문장입니다.

상황별 즉답 훈련 (1초 안에)

  1. 점원: 袋は ご利用ですか → (필요 없다) →
  2. 점원: 何名様ですか → (셋이다) →
  3. 점원: サイズは いかがなさいますか → (L로) →
  4. 점원: 店内で お召し上がりですか → (포장) →
  5. 물을 더 받고 싶다 →
  6. 계산하고 싶다 →

(답 예: ①大丈夫です ②三人です ③Lで おねがいします ④テイクアウトで ⑤お水を おねがいします ⑥お会計 おねがいします)

셀프 체크리스트

  • [ ] 주문 3형제(ください/おねがいします/に します)의 어감 차이를 설명할 수 있다
  • [ ] 수단·지정의 で(カードで、Mで)를 쓸 수 있다
  • [ ] ひとり·ふたり와 ひとつ~とお를 셀 수 있다
  • [ ] 점원 정형 멘트 5개 이상을 듣고 뜻을 안다
  • [ ] いただきます/ごちそうさまでした를 때에 맞게 쓴다
  • [ ] 「もう一度 言ってください」가 반사적으로 나온다
