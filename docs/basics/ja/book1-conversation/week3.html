3주차 — 숫자·시간·날짜 + これ/それ/あれ

이번 주 목표

  1. 0~10,000의 숫자를 읽고, 전화번호·나이·가격을 말할 수 있다.
  2. 시간(〜時〜分)과 요일·날짜를 묻고 답할 수 있다.
  3. これ/それ/あれ로 물건을 가리키고, 「いくらですか」로 가격을 물을 수 있다.

이번 주부터 한글 발음 보조가 없습니다. 대신 기초 한자가 漢字(かんじ) 형식의 후리가나와 함께 등장합니다. 후리가나를 읽는 것이지 한자를 외우는 것이 아닙니다 — 한자 쓰기는 이 책의 범위 밖입니다.

11차시 — 숫자 0〜10,000 / 전화번호와 나이

① 오늘의 대화

어학원에서 연락처를 교환하며.

  • A: ミンジュンさん、電話番号(でんわばんごう)は 何番(なんばん)ですか。
    • 민준 씨, 전화번호가 몇 번이에요?
  • B: 090-1234-5678です。
    • (제로큐-제로 이치니-산욘 고-로쿠나나하치) 090-1234-5678이에요.
  • A: 090-1234-5678ですね。ありがとうございます。
    • 090-1234-5678이죠? 감사합니다.
  • B: さとうさんは おいくつですか。
    • 사토 씨는 몇 살이세요?
  • A: 21(にじゅういっ)さいです。
    • 스물한 살이에요.

② 새 단어

일본어읽기뜻
電話番号でんわばんごう전화번호
何番なんばん몇 번
おいくつおいくつ몇 살 (정중한 물음)
〜さいさい~살, ~세
ゼロ / まるぜろ / まる0 (전화번호에선 ゼロ·まる 둘 다 씀)
〜ですねですね~이죠? (확인)

③ 오늘의 포인트

1. 숫자의 뼈대 — 이것만 외우면 10,000까지

12345678910
いちにさんよん/しごろくなな/しちはちきゅう/くじゅう
  • 조합은 한국어와 같은 원리: 21 = にじゅういち(2·10·1), 345 = さんびゃくよんじゅうご.
  • 100 = ひゃく, 1,000 = せん, 10,000 = まん. 발음 변화 3개만 주의: 300 さんびゃく / 600 ろっぴゃく / 800 はっぴゃく, 3,000 さんぜん / 8,000 はっせん.
  • 4는 보통 よん, 7은 なな, 9는 きゅう를 쓰면 안전합니다(し·しち·く는 듣기용으로만).

2. 확인의 ですね

상대 말을 받아 「090-…ですね」 하고 끝을 내리면 "~이죠?(맞죠?)" 하는 확인이 됩니다. 전화번호·예약·주문 확인에서 계속 나오는 패턴입니다. 1주차의 いい てんきですね(공감)와 어감이 다른 점에 주의 — 공감의 ね는 올리고, 확인의 ね는 살짝 내립니다.

④ 소리 내어 연습

  1. 자기 휴대폰 번호를 일본어로 3번 말하세요. (0은 ゼロ, 하이픈은 の라고 읽어도 됩니다: 090の1234の…)
  2. 가족 나이를 3명 이상 말해 보세요: 「ちちは 55(ごじゅうご)さいです。」(ちち=아버지)
  3. 큰 소리로 읽기: 39 / 74 / 210 / 300 / 680 / 800 / 3,000 / 8,500 / 10,000

12차시 — 지금 몇 시? 何時(なんじ)ですか

① 오늘의 대화

수업 쉬는 시간, 시계가 없는 교실에서.

  • A: すみません、今(いま) 何時(なんじ)ですか。
    • 저기요, 지금 몇 시예요?
  • B: えっと、2時(にじ)半(はん)です。
    • 어… 2시 반이에요.
  • A: 授業(じゅぎょう)は 何時(なんじ)からですか。
    • 수업은 몇 시부터예요?
  • B: 3時(さんじ)からです。4時(よじ)半(はん)までです。
    • 3시부터예요. 4시 반까지고요.
  • A: そうですか。ありがとうございます。
    • 그렇군요. 감사합니다.

② 새 단어

일본어읽기뜻
今いま지금
何時なんじ몇 시
〜時 / 〜分じ / ふん(ぷん)~시 / ~분
半はん반 (30분)
授業じゅぎょう수업
〜から / 〜までから / まで~부터 / ~까지
えっとえっと어…, 음… (생각할 때)

③ 오늘의 포인트

1. 시각 읽기 — 예외 3개만 조심

1時 いちじ, 2時 にじ, 3時 さんじ… 규칙적이지만 4時 よじ / 7時 しちじ / 9時 くじ 세 개만 예외입니다(よんじ·ななじ·きゅうじ라고 하지 않음). 분은 〜ふん이지만 숫자에 따라 ぷん으로 변합니다: 1分 いっぷん, 3分 さんぷん, 5分 ごふん, 10分 じゅっぷん. 처음엔 「〜時半(반)」과 「〜時ちょうど(정각)」만 정확히 해도 충분합니다.

2. から〜まで는 한국어 "부터~까지" 그대로

「3時から 4時半までです。」= "3시부터 4시 반까지입니다." 어순·용법이 한국어와 완전히 같습니다. 시간뿐 아니라 장소에도 그대로 씁니다(ソウルから おおさかまで — 2주차의 「ソウルから きました」가 바로 이 から였습니다).

④ 소리 내어 연습

  1. 지금 실제 시각을 보고 「今、○時○分です」라고 말하세요. 오늘 하루 시계를 볼 때마다 일본어로 중얼거리기.
  2. 자기 일과로 말하기: 「仕事(しごと)は 9時(くじ)から 6時(ろくじ)までです。」
  3. 예외 3개 스피드 퀴즈: 4時 → ? / 7時 → ? / 9時 → ? (よじ・しちじ・くじ가 즉답으로 나올 때까지)

13차시 — 요일과 날짜: 今日(きょう)は 何曜日(なんようび)?

① 오늘의 대화

어학원 접수처에서 시험 일정을 확인하며.

  • A: すみません、テストは いつですか。
    • 저기요, 시험은 언제예요?
  • B: 金曜日(きんようび)です。
    • 금요일이에요.
  • A: 今日(きょう)は 何曜日(なんようび)ですか。
    • 오늘이 무슨 요일이죠?
  • B: 水曜日(すいようび)ですよ。テストは 15日(じゅうごにち)です。
    • 수요일이에요. 시험은 15일이에요.
  • A: あさってですね! わかりました。
    • 모레네요! 알겠습니다.

② 새 단어

일본어읽기뜻
いついつ언제
今日 / 明日 / あさってきょう / あした / あさって오늘 / 내일 / 모레
何曜日なんようび무슨 요일
テストてすと시험, 테스트
わかりましたわかりました알겠습니다
〜日にち~일 (날짜)

③ 오늘의 포인트

1. 요일 — 월화수목금토일이 아니라 "달·불·물·나무·쇠·흙·해"

월화수목금토일
月曜日(げつようび)火曜日(かようび)水曜日(すいようび)木曜日(もくようび)金曜日(きんようび)土曜日(どようび)日曜日(にちようび)

한자 뜻이 한국어 요일명과 같아서(月=월, 火=화…) 한국인에게는 반칙급으로 쉽습니다.

2. 날짜 — 1일~10일은 통째로 암기

11일부터는 숫자+にち(じゅういちにち…)로 규칙적이지만, 1~10일은 고유어입니다: 1日 ついたち, 2日 ふつか, 3日 みっか, 4日 よっか, 5日 いつか, 6日 むいか, 7日 なのか, 8日 ようか, 9日 ここのか, 10日 とおか. 그리고 14日 じゅうよっか, 20日 はつか, 24日 にじゅうよっか만 추가 예외. 오늘은 1~10일만 노래처럼 외우세요.

④ 소리 내어 연습

  1. 오늘 실제 날짜·요일로: 「今日は ○月(がつ)○日、○曜日です。」 (월은 숫자+がつ. 단 4月 しがつ, 7月 しちがつ, 9月 くがつ)
  2. 자기 생일 말하기: 「誕生日(たんじょうび)は ○月○日です。」
  3. ついたち부터 とおか까지 손가락 꼽으며 2번 왕복(1→10, 10→1).

14차시 — これ・それ・あれ / 이건 뭐예요?

① 오늘의 대화

일본인 친구 집에 놀러 가서.

  • A: たなかさん、これは 何(なん)ですか。
    • 다나카 씨, 이건 뭐예요?
  • B: それは こけしです。日本(にほん)の 人形(にんぎょう)ですよ。
    • 그건 고케시예요. 일본 인형이에요.
  • A: へえ、かわいいですね。じゃ、あれは 何ですか。
    • 와, 귀엽네요. 그럼 저건 뭐예요?
  • B: あれは 招き猫(まねきねこ)です。
    • 저건 마네키네코(복고양이)예요.
  • A: 招き猫! 有名(ゆうめい)ですね。
    • 마네키네코! 유명하죠.

② 새 단어

일본어읽기뜻
これ / それ / あれこれ / それ / あれ이것 / 그것 / 저것
どれどれ어느 것
何なん / なに무엇
人形にんぎょう인형
かわいいかわいい귀엽다
有名ゆうめい유명함
へえへえ와, 오~ (감탄 맞장구)

③ 오늘의 포인트

1. これ/それ/あれ = 이것/그것/저것 — 완전 대응

내 근처 これ, 상대 근처 それ, 둘 다에서 먼 것 あれ, 물음 どれ. 한국어와 정확히 같은 3분법입니다. 대화에서 A가 これ(내 쪽)라고 물으면 B는 それ(상대 쪽)로 받는 것에 주목 — 시점이 바뀌면 지시어도 바뀝니다. 명사 앞에 붙이는 형태는 この/その/あの + 명사: このほん(이 책), あのひと(저 사람 — 9차시에서 이미 만났죠).

2. 맞장구 3종: へえ / そうですか / いいですね

일본어 대화는 맞장구(あいづち)가 윤활유입니다. 감탄엔 へえ, 수긍엔 そうですか, 호평엔 いいですね. 이번 주 대화들에서 어디에 쓰였는지 다시 찾아보세요.

④ 소리 내어 연습

  1. 방 안 물건 3개를 가리키며: 「これは ○○です」 (모르는 단어는 「これは 何ですか」라고 묻는 연습으로 대체 — 그게 이 차시의 진짜 기술입니다)
  2. 혼자 2역: 「これは 何ですか」→「それは 〜です」 를 물건 3개로 반복.
  3. この/その/あの 버전: 「このほんは わたしのです。」(이 책은 제 거예요 — 〜の로 끝나면 "~의 것")

15차시 — いくらですか: 가격 묻기

① 오늘의 대화

기념품 가게에서.

  • A: すみません、この 招き猫(まねきねこ)は いくらですか。
    • 저기요, 이 마네키네코는 얼마예요?
  • B: それは 1,500円(せんごひゃくえん)です。
    • 그건 1,500엔입니다.
  • A: じゃ、あの 大(おお)きいのは いくらですか。
    • 그럼 저 큰 건 얼마예요?
  • B: あれは 3,800円(さんぜんはっぴゃくえん)です。
    • 저건 3,800엔입니다.
  • A: うーん、じゃ、これを ください。
    • 음, 그럼 이걸 주세요.
  • B: はい、ありがとうございます。
    • 네, 감사합니다.

② 새 단어

일본어읽기뜻
いくらいくら얼마
〜円えん~엔
大きい / 小さいおおきい / ちいさい크다 / 작다
〜を くださいをください~을 주세요
うーんうーん음… (고민)
〜のの~한 것 (大きいの=큰 것)

③ 오늘의 포인트

1. 쇼핑 2대 문장: いくらですか + 〜を ください

「これは いくらですか」(이건 얼마예요?) → 듣기 → 「これを ください」(이걸 주세요). 이 두 문장이면 일본 어느 가게에서든 물건을 살 수 있습니다. ください 문형은 4주차 내내 확장됩니다. 가격 듣기는 11차시 숫자가 실전 투입되는 순간: 1,500 = せんごひゃく(1,000+500), 3,800 = さんぜんはっぴゃく(발음 변화 2개가 다 들어간 숫자!).

2. 大きいの = "큰 것" — の의 두 번째 얼굴

8차시에서 の는 "~의"였는데, 형용사 뒤에 오면 "~한 것"이 됩니다: 大きいの(큰 것), 小さいの(작은 것), わたしの(내 것). 물건 이름을 몰라도 「あの 大きいのは いくらですか」로 살아남을 수 있습니다.

④ 소리 내어 연습

  1. 읽기: 500円 / 980円 / 1,200円 / 3,600円 / 8,800円 / 10,000円 (즉답이 목표)
  2. 혼자 가게 놀이: 방 안 물건 2개에 마음속 가격을 붙이고, 손님·점원 2역으로 가격 묻기→구매까지.
  3. 「あの 小さいのは いくらですか。」를 실제 가리키는 동작과 함께 3번.

주말 복습 — 3주차: 숫자·시간·날짜·지시어 총정리

통합 대화 — 잡화점에서 시계를 사다

  • A: すみません、あの 時計(とけい)は いくらですか。
    • 저기요, 저 시계는 얼마예요?
  • B: どれですか。
    • 어느 거요?
  • A: あの 小(ちい)さいのです。白(しろ)いの。
    • 저 작은 거요. 하얀 거.
  • B: ああ、あれは 2,800円(にせんはっぴゃくえん)です。
    • 아아, 저건 2,800엔입니다.
  • A: じゃ、あれを ください。
    • 그럼 저걸 주세요.
  • B: はい。ちょうど 3時(さんじ)ですから、コーヒーサービスが ありますよ。
    • 네. 마침 3시라서 커피 서비스가 있어요.
  • A: へえ、いいですね。あ、今日(きょう)は 何日(なんにち)ですか。
    • 와, 좋네요. 아, 오늘 며칠이죠?
  • B: 20日(はつか)、土曜日(どようび)です。
    • 20일, 토요일입니다.
  • A: ありがとうございます。
    • 감사합니다.

※ 미리보기: 「〜が あります(~이 있어요)」는 5주차의 주인공입니다. 여기선 귀에만 담아 두세요. 白いの(하얀 것)는 15차시 「大きいの」와 같은 패턴입니다.

숫자·시간·날짜 스피드 퀴즈 (소리 내어 즉답)

  1. 숫자: 47 / 300 / 608 / 800 / 1,500 / 3,000 / 8,800 / 10,000
  2. 시각: 4時 / 7時 / 9時 / 2時半 / 10時10分
  3. 날짜: 1日 / 4日 / 8日 / 14日 / 20日 / 오늘 날짜
  4. 내 정보: 전화번호 / 나이 / 생일 / 일과(〜から〜まで)

셀프 체크리스트

  • [ ] 1~10,000을 읽을 수 있고 300·600·800·3,000·8,000의 발음 변화를 안다
  • [ ] 4時·7時·9時를 반사적으로 よじ·しちじ·くじ로 읽는다
  • [ ] 요일 7개를 일본어로 말할 수 있다
  • [ ] 1日~10日 고유어 읽기를 외웠다 (+14日·20日)
  • [ ] これ/それ/あれ/どれ와 この/その/あの를 구분해 쓴다
  • [ ] 「いくらですか」→ 가격 알아듣기 → 「〜を ください」 흐름이 된다
  • [ ] から/まで를 시간·장소 양쪽에 쓸 수 있다

숫자는 이번 주에 끝나지 않습니다. 4주차 쇼핑·식당에서 매일 다시 쓰게 되니, 퀴즈에서 버벅인 숫자는 출퇴근길에 반복하세요.

다음 주 예고: 4주차는 드디어 실전 — 편의점, 옷가게, 식당, 카페에 들어갑니다. 「〜を ください」가 「〜を おねがいします」「〜に します」로 진화하고, 점원의 빠른 일본어에서 필요한 것만 골라 듣는 요령도 배웁니다. いらっしゃいませ! 어서 오세요!
