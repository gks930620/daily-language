5주차 — 길 묻기·위치·교통: 〜は どこですか / 〜が あります・います

이번 주 목표

  1. 「〜は どこですか」로 장소를 묻고, 위치 설명(となり·まえ·うしろ·ちかく)을 알아듣는다.
  2. 존재문 「〜が あります(사물) / います(사람·동물)」를 구분해서 쓴다.
  3. 전철·버스·택시를 타고 목적지까지 갈 수 있다(몇 번 승강장·~행·환승·요금).

3주차의 これ/それ/あれ가 이번 주 ここ/そこ/あそこ(장소판)로 확장되고, 4주차 옷가게에서 만난 「ありますか」가 주인공으로 승격합니다.

21차시 — トイレは どこですか: 장소 묻기

① 오늘의 대화

쇼핑몰에서, 지나가는 직원에게.

  • A: すみません、トイレは どこですか。
    • 저기요, 화장실은 어디예요?
  • B: トイレは あそこです。エレベーターの となりですよ。
    • 화장실은 저기예요. 엘리베이터 옆이에요.
  • A: あそこですね。ありがとうございます。あ、コンビニも ありますか。
    • 저기군요. 감사합니다. 아, 편의점도 있어요?
  • B: はい、1階(いっかい)に ありますよ。
    • 네, 1층에 있어요.
  • A: わかりました。ありがとうございます。
    • 알겠습니다. 감사합니다.

② 새 단어

일본어읽기뜻
トイレといれ화장실
どこどこ어디
ここ / そこ / あそこここ / そこ / あそこ여기 / 거기 / 저기
エレベーターえれべーたー엘리베이터
となりとなり옆, 이웃
コンビニこんびに편의점
〜階かい~층 (1階은 いっかい)

③ 오늘의 포인트

1. こそあど 2탄 — 장소판 ここ/そこ/あそこ/どこ

3주차 これ/それ/あれ/どれ(물건)의 장소 버전입니다. 여기 ここ, 거기 そこ, 저기 あそこ, 어디 どこ — 한국어와 1:1 대응. 「トイレは どこですか」는 여행 일본어 사용 빈도 1위 문장이니 자다가도 나오게 만드세요.

2. 위치 말하기 기본형: AのBです / Aに あります

「エレベーターの となりです」= 엘리베이터(의) 옆이에요. 위치 명사는 の로 잇습니다(8차시 の의 재등장). 「1階に あります」= 1층에 있어요 — 장소に + あります가 존재문의 기본 어순입니다. 한국어 "1층에 있어요"와 조사까지 똑같이 대응합니다(に=에).

④ 소리 내어 연습

  1. 바꿔 묻기: 駅(えき, 역) / 銀行(ぎんこう, 은행) / コンビニ + は どこですか。
  2. 지금 있는 방에서: 「トイレは あそこです」처럼 실제 손으로 가리키며 ここ/そこ/あそこ 문장 3개.
  3. 「コンビニは 1階に あります。」→ 2階(にかい)/3階(さんがい)로 바꿔 말하기. (3階은 さんがい로 탁음!)

22차시 — 위치 명사 총출동: まえ·うしろ·ちかく

① 오늘의 대화

역 앞에서 스마트폰 지도를 보며 행인에게.

  • A: すみません、さくら銀行(ぎんこう)は この 近(ちか)くですか。
    • 저기요, 사쿠라 은행이 이 근처예요?
  • B: はい。駅(えき)の 前(まえ)に 郵便局(ゆうびんきょく)が ありますね。
    • 네. 역 앞에 우체국이 있죠?
  • A: はい、あの 白い 建物(たてもの)ですね。
    • 네, 저 하얀 건물이죠.
  • B: 銀行は その 郵便局の うしろに あります。
    • 은행은 그 우체국 뒤에 있어요.
  • A: 前じゃなくて、うしろですね。ありがとうございます!
    • 앞이 아니라 뒤군요. 감사합니다!

② 새 단어

일본어읽기뜻
近くちかく근처
駅えき역
前 / うしろまえ / うしろ앞 / 뒤
中 / 上 / 下なか / うえ / した안 / 위 / 아래
郵便局ゆうびんきょく우체국
銀行ぎんこう은행
建物たてもの건물
〜じゃなくてじゃなくて~이 아니라

③ 오늘의 포인트

1. 위치 명사 세트 — 전부 「AのBに あります」 틀에 끼운다

前(앞)·うしろ(뒤)·となり(옆)·中(안)·上(위)·下(아래)·近く(근처). 문장 틀은 하나: 「(기준물)の (위치)に あります」 — 「駅の 前に あります」(역 앞에 있어요), 「テーブルの 上に あります」(테이블 위에 있어요). 한국어 어순 그대로라 틀만 잡히면 무한 생산이 됩니다.

2. 정정할 때: 〜じゃなくて、〜です

「前じゃなくて、うしろです」= 앞이 아니라 뒤예요. 2주차 じゃないです(부정)의 연결형입니다. 잘못 알아들었을 때 확인·정정하는 필수 패턴: 「3階じゃなくて、2階ですか。」(3층이 아니라 2층이에요?)

④ 소리 내어 연습

  1. 방 안 물건으로 위치 문장 5개: 「かばんは いすの 下に あります」처럼 (かばん=가방, いす=의자).
  2. 정정 연습: 「トイレは 2階じゃなくて、3階に あります。」— 층수를 바꿔 2개 더.
  3. 눈 감고 자기 동네 설명: 「駅の 近くに コンビニが あります。銀行は コンビニの となりに あります。」

23차시 — あります vs います: 있다의 두 얼굴

① 오늘의 대화

친구 집에 도착해서, 거실을 둘러보며.

  • A: あ、ねこが いますね! かわいい!
    • 아, 고양이가 있네요! 귀여워!
  • B: モモです。もう 一匹(いっぴき)、しろい ねこも いますよ。
    • 모모예요. 한 마리 더, 하얀 고양이도 있어요.
  • A: へえ、どこに いますか。
    • 와, 어디에 있어요?
  • B: たぶん ソファーの 下に います。
    • 아마 소파 밑에 있을 거예요.
  • A: あ、いた! ところで、お手洗(てあら)いは どこに ありますか。
    • 아, 있다! 그런데 화장실은 어디에 있어요?
  • B: ろうかの つきあたりに あります。
    • 복도 끝에 있어요.

② 새 단어

일본어읽기뜻
ねこ / いぬねこ / いぬ고양이 / 개
いますいます(사람·동물이) 있습니다
一匹いっぴき(작은 동물) 한 마리
たぶんたぶん아마
ソファーそふぁー소파
ところでところで그런데 (화제 전환)
お手洗いおてあらい화장실 (トイレ보다 정중)
つきあたりつきあたり(복도·길의) 막다른 끝

③ 오늘의 포인트

1. 규칙은 하나 — 스스로 움직이면 います, 아니면 あります

사람·동물처럼 살아 움직이는 것은 います, 사물·식물·건물·행사는 あります. 한국어는 둘 다 "있다"라서 여기가 한국인이 가장 오래 틀리는 지점입니다.

  • ねこが います / コンビニが あります
  • 부정형: いません / ありません(=ないです, 17차시에서 만난 그 ないです)
  • 질문: 「〜が いますか/ありますか」→ 「はい、います(あります) / いいえ、いません(ないです)」

2. が와 は — 새 정보는 が, 이미 아는 화제는 は

처음 등장시킬 때: 「ねこが いますね!」(어, 고양이가 있네 — 새 정보). 이미 화제에 오른 것의 위치를 물을 때: 「お手洗いは どこに ありますか」(화장실은 어디에…). 완벽히 이해 안 돼도 괜찮습니다 — "발견·새 정보는 が, 아는 것 꺼내 오면 は"만 기억하고, 예문을 통째로 외우세요.

④ 소리 내어 연습

  1. 판정 퀴즈(소리 내어 완성): 先生が __ / 銀行が _ / いぬが _ / テストが __ (답: います/あります/います/あります)
  2. 자기 집 버전: 「うちに ねこが います」 또는 「うちに ねこは いません」— 사실대로 3문장.
  3. 「ところで」로 화제 전환 연습: 아무 이야기 뒤에 「ところで、お手洗いは どこですか」를 붙여 2번.

24차시 — 전철 타기: 몇 번 승강장, 어디행?

① 오늘의 대화

시부야에 가려고 역무원에게.

  • A: すみません、渋谷(しぶや)まで 行(い)きたいんですが…。
    • 저기요, 시부야까지 가고 싶은데요….
  • B: 渋谷なら、2番線(にばんせん)の 電車(でんしゃ)ですよ。
    • 시부야라면 2번 승강장 전철이에요.
  • A: 乗(の)り換(か)えは ありますか。
    • 환승은 있나요?
  • B: いいえ、ありません。渋谷行(ゆ)きですから、そのままで 大丈夫です。
    • 아니요, 없어요. 시부야행이니까 그대로 가시면 돼요.
  • A: いくらですか。
    • (요금은) 얼마예요?
  • B: 180円(ひゃくはちじゅうえん)です。
    • 180엔입니다.
  • A: わかりました。ありがとうございます。
    • 알겠습니다. 감사합니다.

② 새 단어

일본어읽기뜻
電車でんしゃ전철
〜番線ばんせん~번 승강장(선)
乗り換えのりかえ환승
〜行きゆき(いき)~행
〜たいんですがたいんですが~하고 싶은데요… (부드러운 운 떼기)
〜ならなら~라면
そのままそのまま그대로
〜からから~니까 (이유. "부터"의 から와 동형)

③ 오늘의 포인트

1. 운 떼기의 기술: 〜たいんですが…

「渋谷まで 行きたいんですが…」= "시부야까지 가고 싶은데요…". 뒤를 흐리면 상대가 알아서 도와주는, 일본어 특유의 부드러운 부탁 화법입니다. 〜たいです(하고 싶다) 문형은 8주차에서 본격적으로 배우니, 이번 주는 이 문장 통째로 하나만 암기: 「(장소)まで 行きたいんですが…」. 3주차 まで(까지)가 장소로 확장된 것도 확인하세요.

2. 이유의 から — "~니까"

「渋谷行きですから、そのままで 大丈夫です」= 시부야행이니까 그대로 괜찮아요. "부터"의 から(3주차)와 모양이 같지만, 문장 뒤에 붙으면 이유입니다. 한국어 "~니까"와 위치까지 같습니다: 「テストは 明日ですから、がんばります。」

④ 소리 내어 연습

  1. 목적지 바꿔 운 떼기: 新宿(しんじゅく)/大阪(おおさか)/空港(くうこう, 공항) + まで 行きたいんですが…。
  2. 역무원 답 알아듣기 훈련 — 스스로 읽고 핵심만 한국어로 요약: 「3番線です。乗り換えは ありません。」
  3. 이유의 から 문장 만들기: 「明日は 日曜日ですから、___。」 뒤를 자유롭게(예: 大丈夫です).

25차시 — 택시와 버스: 〜まで おねがいします

① 오늘의 대화

호텔 앞에서 택시를 타고.

  • 運転手(うんてんしゅ): どちらまで?
    • 어디까지 가세요?
  • A: さくらホテルまで おねがいします。
    • 사쿠라 호텔까지 부탁합니다.
  • 運転手: はい、かしこまりました。
    • 네, 알겠습니다.
  • A: すみません、時間(じかん)は どのぐらい かかりますか。
    • 저기요, 시간은 어느 정도 걸려요?
  • 運転手: そうですね、15分(じゅうごふん)ぐらいですね。
    • 글쎄요, 15분 정도네요.
  • (도착)
  • A: あ、ここで いいです。ありがとうございました。
    • 아, 여기서 (내리면) 돼요. 감사합니다.

② 새 단어

일본어읽기뜻
運転手うんてんしゅ운전기사
どちらどちら어디, 어느 쪽 (どこ의 정중형)
ホテルほてる호텔
時間じかん시간
どのぐらいどのぐらい어느 정도
かかりますかかります(시간·돈이) 걸립니다/듭니다
〜ぐらいぐらい~정도
ここで いいですここでいいです여기서 돼요/괜찮아요

③ 오늘의 포인트

1. 택시 일본어는 딱 두 문장

탈 때: 「(목적지)まで おねがいします」 — 4주차 주문 문형 おねがいします에 장소まで를 얹은 것뿐입니다. 내릴 때: 「ここで いいです」. 이 で는 장소 지정(여기서)이고, いいです는 "됐어요/충분해요". 참고로 점원·기사가 쓰는 どちら는 どこ의 정중형 — 알아듣기만 하면 됩니다.

2. 소요 시간 묻기: どのぐらい かかりますか

「時間は どのぐらい かかりますか」(시간이 얼마나 걸려요?)는 교통 필수 질문. 답은 「〜分ぐらいです」. ぐらい(정도)는 시간·가격·개수 어디에나 붙습니다: 30分ぐらい, 1,000円ぐらい, ふたつぐらい.

④ 소리 내어 연습

  1. 목적지 바꿔 타기: 駅まで / 空港まで / さくら銀行まで + おねがいします。
  2. 「どのぐらい かかりますか」→ 스스로 답: 「10分ぐらいです」「1時間(いちじかん)ぐらいです」。
  3. 하차 3연기: 손을 들며 「あ、ここで いいです。」 — 타이밍 맞춰 자연스럽게 나올 때까지.

주말 복습 — 5주차: 길·위치·교통 총정리

통합 대화 — 역에서 친구 집까지

  • A: もしもし、たなかさん? 今(いま)、駅(えき)に います。
    • 여보세요, 다나카 씨? 지금 역에 있어요.
  • B: あ、ミンジュンさん! 駅の 前に バスていが ありますね。
    • 아, 민준 씨! 역 앞에 버스 정류장이 있죠?
  • A: えっと…、あ、はい。コンビニの となりですね。
    • 어…, 아, 네. 편의점 옆이요.
  • B: そうです。そこから 3番(さんばん)の バスに のってください。
    • 맞아요. 거기서 3번 버스를 타세요.
  • A: 3番ですね。どのぐらい かかりますか。
    • 3번이죠? 어느 정도 걸려요?
  • B: 10分ぐらいです。「さくら公園(こうえん)前」で おりてください。
    • 10분 정도예요. '사쿠라 공원 앞'에서 내리세요.
  • A: 公園の 前ですね。わかりました。
    • 공원 앞이군요. 알겠습니다.
  • B: うちは 公園の うしろに あります。白い 建物ですよ。じゃ、あとで!
    • 우리 집은 공원 뒤에 있어요. 하얀 건물이에요. 그럼 이따 봐요!

※ 새 표현: もしもし(여보세요 — 7주차 전화 차시의 주인공), のってください(타세요)/おりてください(내리세요)는 17차시 〜てください 완성품 목록에 추가. あとで=이따가. バスてい=버스 정류장, 公園=공원.

위치 설명 실전 훈련

  1. 자기 동네 지도를 머릿속에 그리고, 역(또는 버스정류장)→집까지를 일본어 3문장으로: 「駅の 前に 〜が あります。〜の となりに 〜が あります。うちは 〜の 近くに あります。」
  2. あります/います 즉답 10연발: 先生 / 電車 / いぬ / 郵便局 / ねこ / テスト / 運転手 / ホテル / 学生 / お手洗い
  3. 24·25차시 핵심 두 문장 암송: 「〜まで 行きたいんですが…」 / 「〜まで おねがいします」

셀프 체크리스트

  • [ ] 「〜は どこですか」와 ここ/そこ/あそこ/どこ를 자유자재로 쓴다
  • [ ] 위치 명사 7개(前·うしろ·となり·中·上·下·近く)를 「AのBに あります」 틀에 넣을 수 있다
  • [ ] あります/います를 대상에 따라 구분한다 (부정형 ありません/いません 포함)
  • [ ] 새 정보의 が, 화제의 は 감각이 예문 수준에서는 잡혔다
  • [ ] 「〜まで 行きたいんですが…」로 운을 뗄 수 있다
  • [ ] 이유의 から(~니까)와 범위의 から(~부터)를 구분한다
  • [ ] 택시 두 문장(まで おねがいします / ここで いいです)이 자동으로 나온다

이번 주 문형 「〜が あります/います」는 앞으로 모든 주차에 재등장합니다. 판정(있는 것이 살아 있나?)이 0.5초 안에 되면 합격입니다.

다음 주 예고: 6주차는 대화의 꽃, 스몰토크입니다. 「暑(あつ)いですね(덥네요)」로 시작하는 날씨 이야기, 「〜が 好(す)きです(좋아해요)」로 이어지는 취미 이야기, 그리고 일본어 형용사의 두 갈래(い형용사·な형용사)와 과거형 「おいしかったです(맛있었어요)」까지. 어색한 침묵이 사라집니다.
