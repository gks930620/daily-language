버전 4

대화

  • はじめまして — 처음 뵙겠습니다
  • こんにちは — 안녕하세요 (낮)
  • さようなら — 안녕히 가세요
  • 私(わたし)は ○○です — 저는 ○○입니다 → 私は ミナです。
  • ○○から 来(き)ました — ○○에서 왔습니다 → 韓国(かんこく)から 来ました。
  • どうぞ よろしく お願(ねが)いします — 아무쪼록 잘 부탁드립니다

대화 1개:

A: はじめまして。私は ミナです。韓国から 来ました。 B: 田中(たなか)です。どうぞ よろしく お願いします。

문법

  • AはBです — A는 B입니다 → 私(わたし)は 学生(がくせい)です。
  • 〜ですか — 〜입니까? → 学生ですか。
  • 〜じゃありません — 〜이 아닙니다 → 学生じゃありません。
  • は — 은/는 (발음은 わ[와]) → 私は
  • これ / それ / あれ — 이것 / 그것 / 저것 → これは 本(ほん)です。

표현

  • おはようございます — (아침) 안녕하세요
  • ありがとうございます — 감사합니다
  • すみません — 죄송합니다 / 실례합니다
  • いただきます — 잘 먹겠습니다
  • お願(ねが)いします — 부탁합니다
  • はい / いいえ — 네 / 아니요

한자

  • 一 いち(일) · 二 に(이) · 三 さん(삼) · 四 し(사) · 五 ご(오)
  • 六 ろく(육) · 七 しち(칠) · 八 はち(팔) · 九 きゅう(구) · 十 じゅう(십)
  • 日本(にほん) — 일본
  • 人(ひと / じん) — 사람 / 〜인 → 韓国人(かんこくじん) 한국인
  • 日(ひ / にち) — 날 → 日(ひ) 날 · 日曜日(にちようび) 일요일
