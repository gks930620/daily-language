대화에서 뽑은 문법 — 명사문

대화의 뼈대는 전부 「AはBです」다.

대화 속 문장형태설명
私(わたし)は キム・ミナです。긍정 AはBですA(私)는 B(ミナ)입니다
ミナさんは 学生(がくせい)ですか。의문 〜ですか끝에 か만 붙이면 질문 (어순 그대로)
学生じゃありません。부정 〜じゃありません「です」의 부정 = 〜이 아닙니다
  • 조사 は: 「私は」「ミナさんは」의 は는 화제를 얹는 "은/는". 글자는 は지만 발음은 わ[와].
  • 지시어 これ: 「これは 私の 名刺です」의 これ = 이것. 짝을 채우면:
これそれあれ
이것(나에게 가까움)그것(상대에게 가까움)저것(둘 다 멂)
  • 연습: それは 何(なん)ですか。(그것은 무엇입니까?) / あれは 私の かばんです。(저것은 제 가방입니다.)
