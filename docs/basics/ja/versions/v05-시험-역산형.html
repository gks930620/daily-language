v05 — 시험 역산형 (JLPT)

일본어 기초 4권 세트(①대화 입문 ②기본 문법 ③필수 표현 ④필수 한자, 6개월)의 대안 구성 버전. 비교 기준선: 정통 구성(정중체 대화 → 체계 문법 → 상황별 표현 → 급수순 한자).

1. 콘셉트

  • 커리큘럼을 "가르치고 싶은 순서"가 아니라 JLPT N5→N4 실제 출제 유형에서 역산해 만든다. 모든 단원의 제목이 "〜을 배운다"가 아니라 "이 문제를 풀 수 있게 된다"로 정의된다 — 단원 목표가 곧 문제 하나다.
  • 4권의 분담이 JLPT 과목 구조와 1:1로 포개진다: 1권=청해·회화, 2권=문법, 3권=어휘, 4권=문자(한자). 정통 구성의 "대화 입문/기본 문법/필수 표현/필수 한자"가 시험 과목명으로 재정의된 셈이다.
  • 매 단원 끝에 JLPT식 미니 문제 3~5문항이 붙는다. 풀면 통과, 틀리면 그 단원을 다시 — 진도의 단위가 "페이지"가 아니라 "정답률"이다.
  • 시험용 버전이지만 회화를 버리지 않는다: 1권이 청해 지문을 말하기 대본으로 겸용한다(들리는 문장 = 말할 수 있는 문장이라는 설계).

2. 누구에게 맞나 / 안 맞나

맞는 사람

  • 6개월 뒤 JLPT(N4 목표, 최소 N5 확보)라는 측정 가능한 목표가 필요한 사람. 유학·취업 요건으로 급수가 필요한 경우 사실상 유일한 선택.
  • "내가 지금 몇 점짜리인가"를 계속 확인해야 안심되는 사람 — 매 단원 정답률, 월 1회 유형별 점검, 모의고사 3회가 내장돼 있다.

안 맞는 사람

  • 시험 볼 계획이 전혀 없는 사람 — 출제되지 않는 것(생활 잡담력, 구어 축약)의 우선순위가 낮아, 여행·생활 목적이면 v04가 낫다.
  • 문제 풀이 형식 자체에 스트레스를 받는 사람. 이 버전은 매일이 4지선다다.

3. 4권 구성 개요

권이 버전에서의 정체정통 구성과 다른 점
1권청해·회화 — 청해 유형(即時応答·発話表現·課題理解·ポイント理解)별 훈련 + 대본 낭독상황별 대화가 아니라 출제 유형별 대화. 모든 대화가 "문제 스크립트" 겸 "말하기 대본"
2권문법 역산 — 文法形式の判断·文の組(く)み立(た)て(★별표 문제)·文章の文法 대비문법 배열이 교수 순서가 아니라 N5→N4 출제 포인트 순. 단원명이 전부 목표 문제
3권어휘 역산 — 文脈規定·言い換え類義(N4부터 用法 추가) 대비, N5 약 800어 → N4 약 1,500어주제별 단어장이 아니라 문제 유형이 요구하는 방식(빈칸·바꿔 말하기·용법)으로 어휘를 훈련
4권문자 역산 — 漢字読み·表記 대비, N5 한자 약 100자 → N4 약 300자획수·부수순이 아니라 출제 빈도순 + 유사자 함정(日/曰, 木/本) 훈련 내장

4권 — 문자 역산 (24주)

시즌 2 (N4)+200자(주 17자 내외). 음독 다중 읽기(行=こう·ぎょう 등)와 送(おく)り仮名(がな) 함정 추가

4. 1단원 샘플

1권 청해·회화 — 1주차 「即時応答(そくじおうとう): 인사에 0.5초 안에 반응한다」

단원 목표: 첫인사·이름·출신을 묻는 한 마디에 즉시 올바른 응답을 고를(그리고 말할) 수 있다.

먼저, 대본으로 익힌다 (소리 내어 3회 — 이 대본이 그대로 아래 문제의 스크립트다)

발화해석
はじめまして。どうぞ よろしく。처음 뵙겠습니다. 잘 부탁해요.
— こちらこそ、よろしく お願(ねが)いします。저야말로 잘 부탁드립니다.
お名前(なまえ)は？성함은?
— キム・ミナです。김민아입니다.
お国(くに)は どちらですか。어느 나라 분이세요?
— 韓国(かんこく)です。한국입니다.

JLPT식 미니 문제 (음성 대신 스크립트를 가리고 듣는 역할을 파트너/음성앱이 해 준다)

  • 問1 「はじめまして。どうぞ よろしく。」
    1. こちらこそ、よろしく。 2. さようなら。 3. ごちそうさまでした。
  • 問2 「お名前(なまえ)は？」
    1. 二十歳(はたち)です。 2. キム・ミナです。 3. 韓国(かんこく)です。
  • 問3 「お国(くに)は どちらですか。」
    1. 韓国(かんこく)です。 2. 会社員(かいしゃいん)です。 3. 三人(さんにん)です。

정답: 1 / 2 / 1 해설 — 問2의 함정: お名前(이름)를 묻는데 나이(はたち)나 나라(韓国)로 답하는 오답이 반드시 섞인다. 즉시응답 유형은 "질문의 의문사를 잡는 귀"의 시험이다. 問3의 どちら는 どこ의 정중형 — 시험은 정중형으로 묻는다.

2권 문법 역산 — 1단원 「わたし＿＿がくせいです의 빈칸을 0.5초에 고른다」

단원 목표: 조사 は를 다른 조사(を·に·で)와 구별해 문법형식판단 문제를 푼다. ★별표 문제 첫 경험.

최소한의 정리 (문제를 풀기 위한 만큼만)

  • は = 은/는(주제), 발음 [わ] / を = 을/를(대상) / に = 에(방향·시간) / で = 에서(동작 장소)
  • 명사문 골격: 「AはBです」 부정은 「Bでは ありません」(=じゃ ありません).

JLPT식 미니 문제

  • 問1 わたし（　　）がくせいです。
    1. は 2. を 3. に 4. で
  • 問2 A「田中(たなか)さんは 先生(せんせい)ですか。」 B「いいえ、先生（　　）ありません。」
    1. では 2. には 3. とは 4. へは
  • 問3 ★문장 조립 きむらさんは ＿＿ ＿＿ ＿★＿ ＿＿。
    1. ゆうめいな 2. だいがく 3. の 4. せんせいです

(완성문: きむらさんは だいがくの ゆうめいな せんせいです。— 기무라 씨는 대학의 유명한 선생님입니다.)

정답: 1 / 1 / ★에 들어가는 것은 1(ゆうめいな) — 배열은 2→3→1→4. 해설 — ★문제는 "다 아는 단어인데 순서를 못 잡으면" 틀리는 유형. 수식(だいがくの·ゆうめいな)이 명사(せんせい) 앞에 쌓이는 어순은 한국어와 같으므로, 한국어로 먼저 조립하고 옮기면 빠르다.

3권 어휘 역산 — 1단원 「인사·정형구: 文脈規定(ぶんみゃくきてい) 첫 6문항 유형」

단원 목표: 아침/식사/첫만남 인사를 문맥 빈칸 문제로 판별한다.

JLPT식 미니 문제

  • 問1 あさ、せんせいに 「（　　）」と いいます。
    1. おはようございます 2. おやすみなさい 3. ごちそうさまでした 4. いただきます
  • 問2 はじめて あった 人(ひと)に 「（　　）」と いいます。
    1. おかえりなさい 2. はじめまして 3. いってきます 4. ただいま
  • 問3 言い換え類義 「あの 人は きょうしです。」와 뜻이 가장 가까운 것은?
    1. せんせい 2. がくせい 3. いしゃ 4. かいしゃいん

정답: 1 / 2 / 1 해설 — 問3이 言い換え類義(바꿔 말하기) 유형: きょうし(教師)≒せんせい(先生). 시험은 같은 뜻의 한자어↔일반어 짝(教師/先生, 品物/もの 등)을 반복해서 묻는다. 이 책의 단어장은 처음부터 유의어 짝 단위로 실려 있다.

4권 문자 역산 — 1단원 「최빈출 8자: 漢字読み(かんじよみ)와 表記(ひょうき)의 첫 함정」

단원 목표: 최빈출 한자 8자를 읽고, 유사자 함정(本/木, 日/曰)을 피한다.

JLPT식 미니 문제

  • 問1 漢字読み あの 人は がくせいです。
    1. ひと 2. じん 3. にん 4. ほん
  • 問2 漢字読み 三時に あいましょう。
    1. さんじ 2. みじ 3. さんとき 4. みっじ
  • 問3 表記 にほんの りょうりが すきです。
    1. 日本 2. 日木 3. 目本 4. 曰本

정답: 1 / 1 / 1 해설 — 問1: 人은 혼자 오면 ひと, 単語 속에서는 じん·にん(韓国人/三人). "단독=훈독, 조합=음독"이 첫 감각. 問3이 表記 유형의 전형: 오답이 전부 비슷하게 생긴 글자(木/本, 目/日, 曰/日)다. 이 책의 유사자 함정 코너가 매주 이런 짝을 하나씩 무력화한다.

5. 6개월 주행 계획

주간 리듬(고정): 월(2권 문법 2단원) → 화(3권 어휘 + 미니 문제) → 수(1권 청해 유형 훈련) → 목(4권 한자 + 表記 함정) → 금(주간 통합 미니모의 20문항) → 토(오답만 재풀이 + 1권 대본 섀도잉) → 일(휴식). 모든 단원은 미니 문제 정답률 80% 이상일 때만 통과.
