문법

명사문 「AはBです」3형

형태문형예문해석
긍정AはBです私(わたし)は 学生(がくせい)です。저는 학생입니다.
의문AはBですか田中(たなか)さんは 学生ですか。다나카 씨는 학생입니까?
부정AはBじゃありません私は 学生じゃありません。저는 학생이 아닙니다.
