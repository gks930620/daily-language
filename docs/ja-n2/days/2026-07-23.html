사회 — 지방 이주와 빈집 재생, 달라지는 일하는 방식(칼럼)

都市部での生活に疲れ、地方への移住を選ぶ若い世代が、ここ数年で目に見えて増えている。背景には、リモートワークの普及によって、働く場所に縛られない生き方が現実味を帯びてきたことがある。各地の自治体も、空き家の改修費用を補助するなど、移住者を呼び込もうとあの手この手を尽くしている。とはいえ、住み慣れない土地に溶け込むのは容易ではなく、地域との関係づくりに戸惑う人も少なくない。それでも、自分らしい暮らしを求めて一歩を踏み出す人の姿は、これからの働き方を考える上で示唆に富んでいる。

전문 읽기

としぶでのせいかつにつかれ、ちほうへのいじゅうをえらぶわかいせだいが、ここすうねんでめにみえてふえている。はいけいには、リモートワークのふきゅうによって、はたらくばしょにしばられないいきかたがげんじつみをおびてきたことがある。かくちのじちたいも、あきやのかいしゅうひようをほじょするなど、いじゅうしゃをよびこもうとあのてこのてをつくしている。とはいえ、すみなれないとちにとけこむのはよういではなく、ちいきとのかんけいづくりにとまどうひともすくなくない。それでも、じぶんらしいくらしをもとめていっぽをふみだすひとのすがたは、これからのはたらきかたをかんがえるうえでしさにとんでいる。

  1. 都市部での生活に疲れ、地方への移住を選ぶ若い世代が、ここ数年で目に見えて増えている。

    해석 · 구문분석

    としぶでのせいかつにつかれ、ちほうへのいじゅうをえらぶわかいせだいが、ここすうねんでめにみえてふえている。

    도시에서의 생활에 지쳐, 지방으로의 이주를 택하는 젊은 세대가 최근 몇 년 사이에 눈에 띄게 늘고 있다.

    都市部での生活に / 疲れ、/ 地方への移住を選ぶ / 若い世代が、/ ここ数年で / 目に見えて / 増えている。 · 疲れ : 疲れる(지치다)의 연용중지형(~하여). 문장을 문어적으로 끊어 잇는다. · 地方への : ~への는 방향·대상 '~로의'. 뒤의 移住를 수식. · 目に見えて : 관용구 '눈에 띄게, 뚜렷이'. 변화의 정도를 강조하는 부사구. · 増えている : 増える(늘다)의 진행형. 지속되는 추세를 나타낸다.

    • 移住 — 이주
    • 目に見えて — 눈에 띄게, 뚜렷이

  2. 背景には、リモートワークの普及によって、働く場所に縛られない生き方が現実味を帯びてきたことがある。

    해석 · 구문분석

    はいけいには、リモートワークのふきゅうによって、はたらくばしょにしばられないいきかたがげんじつみをおびてきたことがある。

    그 배경에는, 재택근무의 보급으로 인해 일하는 장소에 얽매이지 않는 삶의 방식이 현실성을 띠게 되었다는 사정이 있다.

    背景には、/ リモートワークの普及によって、/ 働く場所に縛られない / 生き方が / 現実味を帯びてきた / ことがある。 · 背景には : '그 배경에는'. 논설·기사체에서 원인을 제시하는 도입 표현. · ~によって : 원인·수단 '~에 의해'. 普及를 원인으로 든다. · 縛られない : 縛る(얽매다)의 수동+부정. 場所に縛られない=장소에 얽매이지 않는. · 現実味を帯びる : 관용 '현실성을 띠다'. ~てきた로 점차 그렇게 되어 옴을 나타낸다. · ことがある : 여기 こと는 '(한) 적'이 아니라 '사정·요인'. 「背景には〜ことがある」=배경에는 〜라는 사정이 있다.

    • 普及 — 보급
    • 縛られる — 얽매이다(縛る의 수동)
    • 帯びる — (성질·기운을) 띠다

  3. 各地の自治体も、空き家の改修費用を補助するなど、移住者を呼び込もうとあの手この手を尽くしている。

    해석 · 구문분석

    かくちのじちたいも、あきやのかいしゅうひようをほじょするなど、いじゅうしゃをよびこもうとあのてこのてをつくしている。

    각지의 지자체도 빈집의 개보수 비용을 보조하는 등, 이주자를 끌어들이려고 온갖 수단을 다 쓰고 있다.

    各地の自治体も、/ 空き家の改修費用を補助するなど、/ 移住者を呼び込もうと / あの手この手を尽くしている。 · ~も : '~도'. 앞의 흐름(개인)에 더해 행정까지 나섬을 표시. · ~など : 예시 '~등'. 여러 시책 가운데 하나로 補助를 든다. · 呼び込もうと : 呼び込む(끌어들이다)의 의지형+と='유치하려고'. 목적·의도. · あの手この手 : 관용 '이런저런 온갖 수단'. 尽くす와 붙어 '갖은 수를 다 쓰다'.

    • 自治体 — 지자체
    • 改修 — 개수, 보수
    • 尽くす — (수단을) 다하다

  4. とはいえ、住み慣れない土地に溶け込むのは容易ではなく、地域との関係づくりに戸惑う人も少なくない。

    해석 · 구문분석

    とはいえ、すみなれないとちにとけこむのはよういではなく、ちいきとのかんけいづくりにとまどうひともすくなくない。

    그렇다고는 해도, 낯선 땅에 녹아드는 것은 쉽지 않아, 지역과의 관계 형성에 당혹스러워하는 사람도 적지 않다.

    とはいえ、/ 住み慣れない土地に / 溶け込むのは / 容易ではなく、/ 地域との関係づくりに / 戸惑う人も少なくない。 · とはいえ : '그렇다고는 해도'. 앞 내용을 일부 인정하며 반전하는 접속. · 住み慣れない : 住む+慣れる+ない='살아 익숙해지지 않은', 곧 낯선. · 容易ではなく : 容易だ의 부정 연용중지(~하지 않고). 문어체 끊기. · ~づくり : 명사+づくり로 '~만들기'. 関係づくり=관계 형성. · ~も少なくない : 이중부정 '적지 않다', 곧 꽤 많다는 완곡 표현.

    • 溶け込む — 녹아들다, 융화되다
    • 戸惑う — 당혹해하다, 갈팡질팡하다
    • ~も少なくない — 적지 않다(꽤 많다)

  5. それでも、自分らしい暮らしを求めて一歩を踏み出す人の姿は、これからの働き方を考える上で示唆に富んでいる。

    해석 · 구문분석

    それでも、じぶんらしいくらしをもとめていっぽをふみだすひとのすがたは、これからのはたらきかたをかんがえるうえでしさにとんでいる。

    그래도, 자기다운 삶을 찾아 한 발을 내딛는 사람들의 모습은, 앞으로의 일하는 방식을 생각하는 데 있어 시사하는 바가 크다.

    それでも、/ 自分らしい暮らしを求めて / 一歩を踏み出す人の姿は、/ これからの働き方を考える上で / 示唆に富んでいる。 · ~らしい : '~다운'. 自分らしい=자기다운. · ~を求めて : '~을 찾아/구하여'. 목적을 향한 행동을 이끈다. · ~上で : 여기선 '~하는 데 있어서'. 판단·검토의 국면을 가리킨다. · 示唆に富む : 관용 '시사하는 바가 크다'. 富む=풍부하다.

    • 踏み出す — (발을) 내딛다, 착수하다
    • 示唆 — 시사, 넌지시 암시함
