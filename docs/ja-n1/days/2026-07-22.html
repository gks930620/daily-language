문화 — '기다림'이 사라진 즉시성의 시대를 돌아보는 칼럼

スマートフォンの画面をなぞれば、欲しい情報も商品も、待つ間もなく手元に届く時代になった。かつては、返事を待つ数日間の焦れったさそのものが、人と人との関係に独特の情趣を添えていた。ところが今や、既読の二文字がつかないだけで不安に苛まれ、相手の沈黙を勝手に非難と読み替えてしまう。便利さと引き換えに、私たちは「待つ」という時間に宿っていた想像力を、知らず知らずのうちに手放しつつあるのではないか。たまには通知を切り、返事の来ない静けさの中で、相手を思う時間の豊かさを、もう一度取り戻してみたい。

전문 읽기

スマートフォンのがめんをなぞれば、ほしいじょうほうもしょうひんも、まつまもなくてもとにとどくじだいになった。かつては、へんじをまつすうじつかんのじれったさそのものが、ひととひととのかんけいにどくとくのじょうしゅをそえていた。ところがいまや、きどくのにもじがつかないだけでふあんにさいなまれ、あいてのちんもくをかってにひなんとよみかえてしまう。べんりさとひきかえに、わたしたちは「まつ」というじかんにやどっていたそうぞうりょくを、しらずしらずのうちにてばなしつつあるのではないか。たまにはつうちをきり、へんじのこないしずけさのなかで、あいてをおもうじかんのゆたかさを、もういちどとりもどしてみたい。

  1. スマートフォンの画面をなぞれば、欲しい情報も商品も、待つ間もなく手元に届く時代になった。

    해석 · 구문분석

    スマートフォンのがめんをなぞれば、ほしいじょうほうもしょうひんも、まつまもなくてもとにとどくじだいになった。

    스마트폰 화면을 훑기만 하면, 원하는 정보도 상품도 기다릴 새도 없이 손안에 도착하는 시대가 되었다.

    スマートフォンの画面をなぞれば、/ 欲しい情報も商品も、/ 待つ間もなく / 手元に届く / 時代になった。 · なぞれば : なぞる(덧그리다·훑다)의 가정형. 여기선 화면을 '쓸어넘기다'. · ~間もなく : '~할 새도 없이'. 待つ間もなく = 기다릴 틈도 없이. · 手元に届く : 직역 '손안에 도착하다' → 바로 손에 들어오다.

    • なぞる — 덧그리다, (화면을) 훑다
    • ~間もなく — ~할 새도 없이

  2. かつては、返事を待つ数日間の焦れったさそのものが、人と人との関係に独特の情趣を添えていた。

    해석 · 구문분석

    かつては、へんじをまつすうじつかんのじれったさそのものが、ひととひととのかんけいにどくとくのじょうしゅをそえていた。

    예전에는 답장을 기다리는 며칠간의 애타는 마음 그 자체가, 사람과 사람 사이의 관계에 독특한 정취를 더해 주었다.

    かつては、/ 返事を待つ数日間の / 焦れったさそのものが、/ 人と人との関係に / 独特の情趣を / 添えていた。 · かつては : '예전에는'. 문어적 회고의 도입. · 焦れったさそのもの : 焦れったい의 명사형 + そのもの('~그 자체'). 안달나는 그 자체가 주어. · 添える : '곁들이다·더하다'. 情趣を添える = 정취를 더하다.

    • 添える — 곁들이다, 더하다
    • 焦れったい — 애타다, 안달나다

  3. ところが今や、既読の二文字がつかないだけで不安に苛まれ、相手の沈黙を勝手に非難と読み替えてしまう。

    해석 · 구문분석

    ところがいまや、きどくのにもじがつかないだけでふあんにさいなまれ、あいてのちんもくをかってにひなんとよみかえてしまう。

    그런데 지금은, '읽음' 두 글자가 뜨지 않는 것만으로 불안에 시달리고, 상대의 침묵을 멋대로 비난으로 바꿔 읽어 버린다.

    ところが今や、/ 既読の二文字がつかないだけで / 不安に苛まれ、/ 相手の沈黙を / 勝手に非難と / 読み替えてしまう。 · ところが今や : 앞 문장과 대비되는 전환 '그런데 이제는'. · ~だけで : '~만으로'. 既読가 안 붙는 것만으로. · 苛まれ : 苛む(괴롭히다)의 수동 中止形. 不安に苛まれる = 불안에 시달리다. · 読み替える : (원뜻과 다르게) 바꿔 해석하다. 沈黙을 非難으로 오독.

    • 読み替える — 바꿔 해석하다, 다르게 읽다
    • ~だけで — ~만으로

  4. 便利さと引き換えに、私たちは「待つ」という時間に宿っていた想像力を、知らず知らずのうちに手放しつつあるのではないか。

    해석 · 구문분석

    べんりさとひきかえに、わたしたちは「まつ」というじかんにやどっていたそうぞうりょくを、しらずしらずのうちにてばなしつつあるのではないか。

    편리함과 맞바꾸어, 우리는 '기다린다'는 시간에 깃들어 있던 상상력을, 자기도 모르는 사이에 놓아 가고 있는 것은 아닐까.

    便利さと引き換えに、/ 私たちは / 「待つ」という時間に宿っていた想像力を、/ 知らず知らずのうちに / 手放しつつある / のではないか。 · ~と引き換えに : '~와 맞바꾸어·~을 대가로'. · 宿る : (무언가가) 깃들다·서리다. 時間に宿る想像力 = 시간에 깃든 상상력. · 知らず知らずのうちに : 관용구 '자기도 모르는 사이에'. · ~つつある : '~해 가고 있다'(진행). ~のではないか = 완곡한 문제 제기.

    • 知らず知らずのうちに — 자기도 모르는 사이에
    • ~つつある — ~해 가고 있다(진행)

  5. たまには通知を切り、返事の来ない静けさの中で、相手を思う時間の豊かさを、もう一度取り戻してみたい。

    해석 · 구문분석

    たまにはつうちをきり、へんじのこないしずけさのなかで、あいてをおもうじかんのゆたかさを、もういちどとりもどしてみたい。

    가끔은 알림을 끄고, 답장이 오지 않는 고요함 속에서, 상대를 생각하는 시간의 풍요로움을 다시 한 번 되찾아 보고 싶다.

    たまには通知を切り、/ 返事の来ない静けさの中で、/ 相手を思う / 時間の豊かさを、/ もう一度取り戻してみたい。 · たまには : '가끔은'. · 通知を切る : 알림을 끄다. · 静けさ : 静か의 명사형 '고요함·정적'. · 取り戻す : 되찾다. ~てみたい = '~해 보고 싶다'.

    • 静けさ — 고요함, 정적
    • 通知を切る — 알림을 끄다
